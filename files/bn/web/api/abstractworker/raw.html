সারসংক্ষেপ

AbstractWorker ইন্টারফেসটি মূলত বিভিন্ন প্রোপ্রার্টি এবং মেথড কে বিবৃত করে থাকে যা সাধারনভাবে প্রায় সব কর্মীরা ব্যবহার করে থাকেন, {{domxref("Worker")}} অথবা  {{domxref("SharedWorker")}} হিসেবে।

Properties

The AbstractWorker interface doesn't inherit any property.

{{domxref("AbstractWorker.onerror")}}
Is an {{ domxref("EventListener") }} that is called whenever an {{domxref("ErrorEvent")}} of type error bubbles through the worker.

Methods

The AbstractWorker interface doesn't implement or inherit any method.
