সারাংশ

{{ domxref("CameraManager") }}  এর একটি অবজেক্ট রিটার্ন করবে যাতে আপনি ব্যবহারকারীর ডিভাইসের ক্যামেরার (যদি থেকে থাকে) ফাংশন ব্যবহার করতে পারবেন 

কোড

var camera = window.navigator.mozCamera;

Value

navigator.mozCamera is a {{domxref("CameraManager")}} object you can use to access the cameras available on the device.

Specification

Not part of any specification as yet; this will probably be replaced by WebRTC when it gains more widespread support on mobile devices.

অনুমতি

ফায়ারফক্স ওএস ১.৪ পর্যন্ত, ক্যামেরা এপিআই স্বীকৃত একটি এপিআই ছিল, তবে থার্ড-পার্টি কোন অ্যাপে কাজ করবে না।ফায়ারফক্স ওএস ২.০ থেকে এটির অনুমতির ধাপ কমিয়ে privileged করা হয়েছে, তাই এটি এখন ডেভেলপারগণ তাদের অ্যাপে এটি ব্যবহার করতে পারেন

"permissions": {
  "camera": {
    "description": "Required for accessing cameras on the device."
  }
}
