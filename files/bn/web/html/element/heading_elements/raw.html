উপাদান নথিটি  ছয়টি  শিরোনামে মাত্রা বাস্তবায়ন শীর্ষক <h1> সবচেয়ে গুরুত্বপূর্ণ এবং <H6> কমপক্ষে হয়। একটি শিরোনামটির  উপাদান সংক্ষিপ্তভাবে  প্রবর্তন ধারার বিষয়ে আলোচনা করা হয়েছে। তথ্য শিরোনাম ব্যবহারকারী এজেন্ট দ্বারাও ব্যবহার করা যেতে পারে। উদাহরণস্বরূপ, স্বয়ংক্রিয়ভাবে একটি নথি জন্য একটি সূচীপত্র গঠন করা হল ।

Usage note:

  • Do not use lower levels to decrease heading font size: use the CSS {{cssxref("font-size")}} property instead.
  • Avoid skipping heading levels: always start from <h1>, next use <h2> and so on. Also try to have only one first level heading on a page.
