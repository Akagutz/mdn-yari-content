জাভাস্ক্রিপ্ট ১.৪ এ নতুন ফিচারগুলো :

  • in and instanceof অপেরাটের যোগ করা হয়েছে

  • arguments ফাংশন এর প্রপার্টি নয়
  • Deprecated Function.arity in favor of Function.length
