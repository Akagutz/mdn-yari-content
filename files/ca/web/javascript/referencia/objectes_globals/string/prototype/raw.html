La propietat String.prototype representa l'objecte prototip {{jsxref("String")}}.

{{js_property_attributes(0, 0, 0)}}

Descripció

Totes les instàncies {{jsxref("String")}} hereten de String.prototype. Canvis en l'objecte prototip  String són propagats a totes les instàncies {{jsxref("String")}}.

Propietats

String.prototype.constructor
Especifica la funció que crea el prototip d'un objecte.
{{jsxref("String.prototype.length")}}
Reflecteix la llargària de la cadena.
N
S'utilitza per accedir al caràcter que es troba en la Nth posició on N és un número sencer positiu entre 0 i un menys que el valor de {{jsxref("String.length", "length")}}. Aquestes propietas només són llegibles.
