In der Programmierung ist Boolean ein logischer {{Glossary("Type","Datentyp")}}, der nur einen der zwei {{Glossary("Value","Werte")}} true oder false annehmen kann. Ein Boolean ist die Umsetzung von wahr oder falsch in der Programmierung. Ohne die Möglichkeit Dinge in Booleans auszudrücken würden viele Dinge in einer Programmiersprache nicht mehr funktionieren. Zum Beispiel muss sich in JavaScript eine Bedingung eines if-Statements zu einem Boolean auflösen können.  Eine for-Schleife könnte ohne Bedingung nicht feststellen, ob sie den Code ausführen soll oder nicht.

***JavaScript if-Statement***
if(Bedingung) {
   //Code
}

if(true) {
  console.log("Bedingung ist true");
} else {
    console.log("Bedingung ist false");
  }



***JavaScript for-Schleife***
for(Kontrollvariable; Bedingung; Zähler) {
  //Code
}

for(var i=0; i<4; i++) {
  console.log("Ich werde nur ausgegeben wenn die Bedingung true ist");
}
