Gibt den URI der Seite aus, von der die aktuelle Seite aufgerufen wurde.

Syntax

string = document.referrer;

Anmerkungen

Der Wert ist eine leere Zeichenkette, wenn der Benutzer direkt auf die Seite navigiert ist (nicht über einen Link, sondern z.B. über ein Lesezeichen). Da es sich bei dieser Eigenschaft lediglich um einen String handelt, ist der Zugriff auf den DOM der referenzierenden Seite nicht möglich.

Spezifikation

  • DOM Level 2: referrer
