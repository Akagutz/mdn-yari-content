Übersicht

Die caption-side CSS Eigenschaft positioniert den Inhalt einer {{ HTMLElement("caption") }} einer Tabelle auf der angegebenen Seite.

{{cssinfo}}

Syntax

Formale Syntax: {{csssyntax("caption-side")}}

caption-side: top
caption-side: bottom
caption-side: left     /* Warning: non-standard value */
caption-side: right    /* Warning: non-standard value */
 
caption-side: inherit

Werte

top
Ist ein Schlüsselwort, das angibt, dass die Überschriftbox oberhalb der Tabelle positioniert werden soll.
bottom
Ist ein Schlüsselwort, das angibt, dass die Überschriftbox unterhalb der Tabelle positioniert werden soll.
left {{ non-standard_inline() }}
Ist ein Schlüsselwort, das angibt, dass die Überschriftbox links von der Tabelle positioniert werden soll.
{{ Note("Dieser Wert wurde für CSS 2 vorgeschlagen, jedoch aus der finalen Spezifikation für CSS 2.1 entfernt. Er ist daher nicht standardisiert.") }}
right {{ non-standard_inline() }}
Ist ein Schlüsselwort, das angibt, dass die Überschriftbox rechts von der Tabelle positioniert werden soll.
{{ Note("Dieser Wert wurde für CSS 2 vorgeschlagen, jedoch aus der finalen Spezifikation für CSS 2.1 entfernt. Er ist daher nicht standardisiert.") }}
inherit
Ist ein Schlüsselwort, das angibt, dass der caption-side Wert des Elternelements verwendet werden soll.

Beispiele

.contentbox table-caption {
  caption-side: bottom;
}
