Übersicht

 

Das HTML <span> Element ist ein generischer Inline-Container zur Formulierung von Inhalten, der von Natur aus nichts repräsentiert. Es kann verwendet werden, um Elemente für Stylingzwecke zu gruppieren (unter Verwendung der Attribute class oder id), oder weil sie Attributwerte wie lang teilen. Es sollte nur verwendet werden, wenn kein anderes semantisches Element geeignet ist. <span> ist sehr ähnlich wie ein {{HTMLElement("div")}} Element, aber {{HTMLElement("div")}} ist ein Element auf Blockebene, während ein <span> ein Inline-Element ist.

Content categories Flow content, phrasing content.

Attribute

Dieses Element beinhaltet nur die globalen Attribute.

<p><span>Some text</span></p>
