Der Age Header beinhaltet die Zeit in Sekunden die ein Objekt in einem proxy cache war. 

Normalerweise ist der Age Header nahe bei Null. Wenn Age: 0 ist, dann wurde es vermutlich vom Ausgangsserver geholt. Ansonsten ist es normalerweise als Differenz zwischen dem aktuellen date des proxy servers und dem {{HTTPHeader("Date")}} Feld des Standard Header aus der HTTP Antwort.

Header type {{Glossary("Response header")}}
{{Glossary("Forbidden header name")}} Nein

Syntax

Age: <delta-seconds>

Richtlinien

<delta-seconds>

Ein nicht-negativer integer repräsentiert die Zeit in Sekunden die ein Objekt in einem proxy cache war.

Beispiele

Age: 24
