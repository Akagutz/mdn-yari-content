Die getDate() Methode gibt den Tag des Monats eines Datums gemäß der Ortszeit zurück

{{EmbedInteractiveExample("pages/js/date-getdate.html")}}

The source for this interactive example is stored in a GitHub repository. If you'd like to contribute to the interactive examples project, please clone https://github.com/mdn/interactive-examples and send us a pull request.

Syntax

dateObj.getDate()

Rückgabewert

Eine ganze Zahl zwischen 1 and 31, die den Tag des Monats für das gegebenen Datum nach der lokalen Zeit angibt.

Beispiele

Einsatz von getDate()

Im Statement wird die Variable day mit dem Wert 25 initialisiert. Diese passiert mit dem Wert des {{jsxref("Date")}} Objektes Xmas95.

var Xmas95 = new Date('December 25, 1995 23:15:30');
var day = Xmas95.getDate();

console.log(day); // 25
