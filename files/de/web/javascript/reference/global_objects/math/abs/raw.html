Die Funktion Math.abs(x) gibt den Betrag einer Zahl zurück.

Math.abs(x)=|x|={xifx>00ifx=0xifx<0{\mathtt{\operatorname{Math.abs}(x)}} = {|x|} = \begin{cases} x & \text{if} \quad x \geq 0 \\ -x & \text{if} \quad x < 0 \end{cases}

Syntax

Math.abs(x);

Parameter

x
Eine Nummer.

Rückgabewert

Der absolute Wert der übergebenen Zahl.

Beschreibung

Weil abs eine statische Funktion von Math ist, ist sie immer über Math.abs() benutzbar.

Beispiele

Das Verhalten von Math.abs()

Wenn der Methode ein leeres Objekt, ein Array mit mehr als einem Element, ein nicht nummerischer String oder {{jsxref("undefined")}} (kein Parameter) übergeben wird, ist {{jsxref("NaN")}} das Resultat. Für {{jsxref("null")}}, einen Leeren String oder ein leeres Array ist das Ergebnis 0.

Math.abs('-1');     // 1
Math.abs(-2);       // 2
Math.abs(null);     // 0
Math.abs('');       // 0
Math.abs([]);       // 0
Math.abs([2]);      // 2
Math.abs([1,2]);    // NaN
Math.abs({});       // NaN
Math.abs('string'); // NaN
Math.abs();         // NaN
