A browsing context is the environment in which a {{glossary("browser")}} displays a {{domxref("Document")}} (normally a tab nowadays, but possibly also a window or a frame within a page).

Each browsing context has a specific {{glossary("origin")}}, the origin of the active document, and a history that lists all the displayed documents in order.

Communication between browsing contexts is severely restricted. Between browsing context of the same origin, a {{domxref("BroadcastChannel")}} can be opened and used.
