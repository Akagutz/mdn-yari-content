A favicon (favorite icon) is a tiny icon included along with a website, which is displayed in places like the browser's address bar and bookmarks menu.

Usually, favicons are 16 x 16 pixels in size and stored in the {{Glossary("GIF")}}, {{Glossary("PNG")}}, or ICO file format.

They are used to improve user experience and enforce brand consistency. When a familiar icon is seen in the browser's address bar, for example, it helps users know they are in the right place.
