In {{Glossary("HTML")}} a tag is used for creating an {{Glossary("element")}}.  The name of an HTML element is the name used in angle brackets such as <p> for paragraph.  Note that the end tag's name is preceded by a slash character, "</p>", and that in empty elements the end tag is neither required nor allowed. If {{Glossary("Attribute", "attributes")}} are not mentioned, default values are used in each case.
