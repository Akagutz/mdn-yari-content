Type  (or data type) is a characteristic of a {{glossary("value")}} affecting what kind of data it can store — for example in JavaScript a {{domxref("Boolean")}} only holds true/false values, whereas a {{domxref("String")}} holds text strings, a {{domxref("Number")}} holds numbers of any kind, and so on.

A value's data type also affects which operations are valid on that value. For example, an integer can be multiplied by an integer, but not by a string.
