Notes

Tagging standard

  • Grouping tag is:
    • "HTML DOM" represents HTML* interfaces and Document.* added by the HTML Spec. This includes the {{domxref("TimeRanges")}}, {{domxref("Window")}}, {{domxref("WindowProxy")}}, {{domxref("GlobalEventHandlers")}}, {{domxref("WindowEventHandlers")}} and {{domxref("Navigator")}} interfaces.
    • "Text Track API" for all VTT related (currently none); {{domxref("HTMLTrackElement")}} has both "HTML DOM" and "Text Track API"
    • "Text Field Selection API" for {{domxref("HTMLInputElement.setSelectionRange")}}, {{domxref("HTMLInputElement.setRangeText")}}, {{domxref("HTMLInputElement.selectionStart")}}, {{domxref("HTMLInputElement.selectionEnd")}}, {{domxref("HTMLInputElement.selectionDirection")}}, {{domxref("HTMLTextAreaElement.setSelectionRange")}}, {{domxref("HTMLTextAreaElement.setRangeText")}}, {{domxref("HTMLTextAreaElement.selectionStart")}}, {{domxref("HTMLTextAreaElement.selectionEnd")}}, {{domxref("HTMLTextAreaElement.selectionDirection")}}; these are all part of "HTML DOM" too.
    • "Constraint Validation API":
      • {{domxref("ValidityState")}} (only this tag)
      • {{domxref("HTMLFormElement.checkValidity()")}}, {{domxref("HTMLFormElement.reportValidity()")}} (All these have both "HTML DOM" and "Constraint Validation API".)
      • {{domxref("HTMLInputElement.willValidate")}}, {{domxref("HTMLInputElement.validity")}}, {{domxref("HTMLInputElement.validationMessage")}}, {{domxref("HTMLInputElement.checkValidity()")}}, {{domxref("HTMLInputElement.reportValidity()")}}, {{domxref("HTMLInputElement.setCustomValidity()")}} (All these have both "HTML DOM" and "Constraint Validation API".)
      • {{domxref("HTMLButtonElement.willValidate")}}, {{domxref("HTMLButtonElement.validity")}}, {{domxref("HTMLButtonElement.validationMessage")}}, {{domxref("HTMLButtonElement.checkValidity()")}}, {{domxref("HTMLButtonElement.reportValidity()")}}, {{domxref("HTMLButtonElement.setCustomValidity()")}} (All these have both "HTML DOM" and "Constraint Validation API".)
      • {{domxref("HTMLSelectElement.willValidate")}}, {{domxref("HTMLSelectElement.validity")}}, {{domxref("HTMLSelectElement.validationMessage")}}, {{domxref("HTMLSelectElement.checkValidity()")}}, {{domxref("HTMLSelectElement.reportValidity()")}}, {{domxref("HTMLSelectElement.setCustomValidity()")}} (All these have both "HTML DOM" and "Constraint Validation API".)
      • {{domxref("HTMLTextAreaElement.willValidate")}}, {{domxref("HTMLTextAreaElement.validity")}}, {{domxref("HTMLTextAreaElement.validationMessage")}}, {{domxref("HTMLTextAreaElement.checkValidity()")}}, {{domxref("HTMLTextAreaElement.reportValidity()")}}, {{domxref("HTMLTextAreaElement.setCustomValidity()")}} (All these have both "HTML DOM" and "Constraint Validation API".)
      • {{domxref("HTMLKeygenElement.willValidate")}}, {{domxref("HTMLKeygenElement.validity")}}, {{domxref("HTMLKeygenElement.validationMessage")}}, {{domxref("HTMLKeygenElement.checkValidity()")}}, {{domxref("HTMLKeygenElement.reportValidity()")}}, {{domxref("HTMLKeygenElement.setCustomValidity()")}} (All these have both "HTML DOM" and "Constraint Validation API".)
      • {{domxref("HTMLOutputElement.willValidate")}}, {{domxref("HTMLOutputElement.validity")}}, {{domxref("HTMLOutputElement.validationMessage")}}, {{domxref("HTMLOutputElement.checkValidity()")}}, {{domxref("HTMLOutputElement.reportValidity()")}}, {{domxref("HTMLOutputElement.setCustomValidity()")}} (All these have both "HTML DOM" and "Constraint Validation API".)
      • {{domxref("HTMLFieldSetElement.willValidate")}}, {{domxref("HTMLFieldSetElement.validity")}}, {{domxref("HTMLFieldSetElement.validationMessage")}}, {{domxref("HTMLFieldSetElement.checkValidity()")}}, {{domxref("HTMLFieldSetElement.reportValidity()")}}, {{domxref("HTMLFieldSetElement.setCustomValidity()")}} (All these have both "HTML DOM" and "Constraint Validation API".)
      • {{domxref("HTMLObjectElement.willValidate")}}, {{domxref("HTMLObjectElement.validity")}}, {{domxref("HTMLObjectElement.validationMessage")}}, {{domxref("HTMLObjectElement.checkValidity()")}}, {{domxref("HTMLObjectElement.reportValidity()")}}, {{domxref("HTMLObjectElement.setCustomValidity()")}} (All these have both "HTML DOM" and "Constraint Validation API".)
    • "Command API": This one is a bit complex, as old properties have been added new semantics. The Command API tag is set in addition to the regular tags
      • {{domxref("Document.id")}}, {{domxref("Document.title")}}, and {{domxref("Document.accessKeyLabel")}}
      • {{domxref("Node.textContent")}} [It has an added semantic meaning (label) for HTMLAnchorElement, HTMLButtonElement, _some_ HTMLTypeElement (depents of the type parameter!), HTMLOptionElement (but only if it has no label attribute), HTMLLabelElement, HTMLLegendElement. But not for the 90 other type of nodes…]
      • {{domxref("HTMLImageElement.src")}} [It has an added semantic meaning (icon) on the first element descendant of a HTMLAnchorElement or HTMLButtonElement in tree order, relative to that element]
      • {{domxref("HTMLInputElement.src")}} [It has an added semantic meaning (icon) if the input is of type is image]
      • {{domxref("HTMLMenuItemElement.icon")}}
      • {{domxref("HTMLElement.hidden")}} [Added semantic in some cases]
      • {{domxref("HTMLInputElement.type")}} [It has an added semantic meaning]
      • {{domxref("HTMLInputElement.value")}}
      • {{domxref("HTMLOptionElement.label")}} [It has a special semantiv meaning in some cases]
      • See §4.11.6 of HTML WHATWG for all the gory details.
    • "Canvas API"
      • {{domxref("HTMLCanvasElement")}} (has both "HTML DOM" and "Canvas API")
      • {{domxref("RenderingContext")}}, {{domxref("CanvasImageSource")}} (only "Canvas API")
      • {{domxref("CanvasProxy")}} {{domxref("CanvasRenderingContext2D")}},  {{domxref("CanvasDrawingStyles")}}, {{domxref("CanvasPathMethods")}}, {{domxref("CanvasGradient")}}, {{domxref("CanvasPattern")}}, {{domxref("TextMetrics")}}, {{domxref("ImageData")}}, {{domxref("DrawingStyle")}}, {{domxref("Path")}} (have both "Canvas API" and "Worker-exposed")
      • {{domxref("WebGLRenderingContext")}} (has both "WebGL" and "Canvas API")
    • "Hit Regions API" (subset of "Canvas API", so all follows the rules for "Canvas API" too)
      • {{domxref("CanvasRenderingContext2D.addHitRegion")}}, {{domxref("CanvasRenderingContext2D.removeHitRegion")}}
      • {{domxref("MouseEvent.region")}}, {{domxref("Touch.region")}}
    • "Path API" (subset of "Canvas API", so all follows the rules for "Canvas API" too)
      • {{domxref("CanvasPathMethods")}}, {{domxref("Path")}}
    • "Microdata DOM API"
      • {{domxref("Document.getItems")}}, {{domxref("HTMLElement.itemScope")}}, {{domxref("HTMLElement.itemType")}}, {{domxref("HTMLElement.itemId")}}, {{domxref("HTMLElement.itemRef")}}, {{domxref("HTMLElement.itemProp")}}, {{domxref("HTMLElement.properties")}} and {{domxref("HTMLElement.itemValue")}}
