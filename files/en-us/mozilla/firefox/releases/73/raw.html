This article provides information about the changes in Firefox 73 that will affect developers. Firefox 73 is the current Beta version of Firefox, and will ship on February 11, 2020.

Changes for web developers

Developer tools

  • CORS errors now appear as errors in the console (and no longer as warnings) giving them the appropriate visibility ({{bug(1602093)}}).

Removals

HTML

No changes.

 Removals

CSS

  • Adds {{cssxref("overscroll-behavior-block")}} and {{cssxref("overscroll-behavior-inline")}} the logical mappings for {{cssxref("overscroll-behavior-x")}} and {{cssxref("overscroll-behavior-y")}} ({{bug(833953)}})

Removals

SVG

  • The {{SVGAttr("letter-spacing")}} and {{SVGAttr("word-spacing")}} properties now work in SVG ({{bug(371787)}}). 

MathML

  • The deprecated {{MathMLElement("mfenced")}} element has been removed ({{bug(1603773)}}). Use the {{MathMLElement("mrow")}} and {{MathMLElement("mo")}} elements instead.
