The BluetoothDevice.uuids read-only property lists the UUIDs of GATT services provided by the device, that the current origin is allowed to access.

This page describes the W3C Community Group BluetoothDevice.name property. For the Firefox OS propery of the same name, see {{DOMxRef("BluetoothDevice_%28Firefox_OS%29/uuids", "BluetoothDevice.uuids (Firefox OS)")}}.
