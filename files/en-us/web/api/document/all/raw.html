The {{DOMxRef("Document")}} interface's read-only all property returns an {{DOMxRef("HTMLAllCollection")}} rooted at the document node. In other words, it returns the entire contents of the page.

Syntax

var htmlAllCollection = document.all;

Value

An {{DOMxRef("HTMLAllCollection")}} which contains every node in the document.
