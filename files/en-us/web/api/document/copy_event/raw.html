The copy event fires when the user initiates a copy action through the browser's user interface.

Bubbles Yes
Cancelable Yes
Interface {{domxref("ClipboardEvent")}}
Event handler property oncopy

The original target for this event is the Element that was the intended target of the paste action. You can listen for this event on the Document interface to handle it in the capture or bubbling phases. For full details on this event please see the page on the Element: copy event.

Examples

document.addEventListener('copy', (event) => {
    console.log('copy action initiated')
});
