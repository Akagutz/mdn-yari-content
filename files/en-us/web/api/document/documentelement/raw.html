{{ApiRef("DOM")}}

Document.documentElement returns the {{domxref("Element")}} that is the root element of the {{domxref("document")}} (for example, the {{HTMLElement("html")}} element for HTML documents).

Syntax

var element = document.documentElement;

Example

var rootElement = document.documentElement;
var firstTier = rootElement.childNodes;

// firstTier is the NodeList of the direct children of the root element
for (var i = 0; i < firstTier.length; i++) {
   // do something with each direct kid of the root element
   // as firstTier[i]
}

Notes

For any non-empty HTML document, document.documentElement will always be an {{HTMLElement("html")}} element. For any non-empty XML document, document.documentElement will always be whatever element is the root element of the document.
