The documentURI property of the {{domxref("Document")}} interface returns the document location as a string.

As originally defined in DOM3 this attribute read/write. In the modern DOM Standard it is read only.

Syntax

var string = document.documentURI;

Notes

Documents also have a {{domxref("document.URL")}} property which returns the same value.
