{{APIRef("DOM")}}

DOMString is a UTF-16 String. As JavaScript already uses such strings, DOMString is mapped directly to a {{jsxref("String")}}.

Passing {{jsxref("null")}} to a method or parameter accepting a DOMString typically stringifies to "null".
