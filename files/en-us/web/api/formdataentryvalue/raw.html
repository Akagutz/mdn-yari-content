A string which represents a single value from a set of {{domxref("FormData")}} key-value pairs.

This type is returned by the {{domxref("FormData.get()")}} and {{domxref("FormData.getAll()")}} methods. The {{domxref("FormData.get()")}} method returns a single value while {{domxref("FormData.getAll()")}} returns an array of FormDataEntryValues.
