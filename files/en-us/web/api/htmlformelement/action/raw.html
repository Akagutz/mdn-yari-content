The HTMLFormElement.action property represents the action of the {{HTMLElement("form")}} element.

The action of a form is the program that is executed on the server when the form is submitted. This property can be retrieved or set.

Syntax

var string = form.action;
form.action = string;

Example

form.action = '/cgi-bin/publish';

Specification

HTML5, Section 4.10.21 Form submission

DOM Level 2 HTML: action
