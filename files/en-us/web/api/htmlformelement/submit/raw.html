The HTMLFormElement.submit() method submits a given {{HtmlElement("form")}}.

This method is similar, but not identical to, activating a form's submit {{HtmlElement("button")}}. When invoking this method directly, however:

  • No {{event("submit")}} event is raised. In particular, the form's {{domxref("GlobalEventHandlers.onsubmit", "onsubmit")}} event handler is not run.
  • Constraint validation is not triggered.

If a form control (such as a submit button) has a name or id of submit, this method will mask the form's submit method.

Syntax

HTMLFormElement.submit()

Example

document.forms["myform"].submit();

Specification

HTML Living standard: The form element
