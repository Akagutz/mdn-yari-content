{{APIRef("HTML DOM")}}

The target property of the {{domxref("HTMLFormElement")}} interface represents the target of the form's action (i.e., the frame in which to render its output).

Syntax

string = HTMLFormElement.target
HTMLFormElement.target = string

Example

myForm.target = document.frames[1].name;

Specification

HTML 5, Section 4.10.19.6, Form submission

DOM Level 2 HTML: target
