The nonce property of the {{domxref("HTMLElement")}} interface returns the cryptographic number used once that is used by Content Security Policy to determine whether a given fetch will be allowed to proceed.

In later implementations, elements that have a nonce attribute only expose it to scripts (and not to side-channels like CSS attribute selectors).Â 

Syntax

var nonce = HTMLElement.nonce
HTMLElement.nonce = nonce

Value

A cryptographic nonce.
