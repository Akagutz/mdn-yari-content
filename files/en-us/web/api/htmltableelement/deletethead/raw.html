The HTMLTableElement.deleteTHead() removes the {{HTMLElement("thead")}} element from a given {{HtmlElement("table")}}.

Syntax

HTMLTableElement.deleteTHead();

Example

This example uses JavaScript to delete a table's header.

HTML

<table>
  <thead><th>Name</th><th>Occupation</th></thead>
  <tr><td>Bob</td><td>Plumber</td></tr>
  <tr><td>Jim</td><td>Roofer</td></tr>
</table>

JavaScript

let table = document.querySelector('table');
table.deleteTHead();
