The battery read-only property returns a {{domxref("BatteryManager")}} provides information about the system's battery charge level; provides also some new events you can handle to monitor the battery status. This implements the Battery Status API; see that documentation for additional details, a guide to using the API, and sample code.

The battery property has been removed in favor of the standard {{domxref("Navigator.getBattery()")}} method, which returns a battery {{domxref("Promise")}}. Its support is still partial though.

Syntax

var battery = navigator.battery;
