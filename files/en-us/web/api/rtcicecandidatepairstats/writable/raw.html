The obsolete {{domxref("RTCIceCandidatePairStats")}} property writable reports whether or not the connection described by the candidate pair is writable.

Syntax

isWritable = rtcIceCandidatePairStats.writable;

Value

A Boolean value which is true if the connection described by this candidate pair has received acknowledgement of receipt (ACK) for at least one ICE request and that {{Glossary("STUN")}} consent hasn't expired.

Note: This property was removed from the specification in early 2017 because you can determine whether or not an incoming ICE request is available to read by checking to see if {{domxref("RTCIceCandidatePairStats.responsesReceived", "responsesReceived")}} is greater than 0 and that the time specified by {{domxref("RTCIceCandidatePairStats.consentExpiredTimestamp", "consentExpiredTimestamp")}} has not passed:

if (icpStats.responsesReceived > 0
    && icpStats.consentExpiredTimestamp < performance.now()) {
  /* at least one ICE response has been received */
}
