The hash property of the {{domxref("URL")}} interface returns a {{domxref("USVString")}} containing a '#' followed by the fragment identifier of the URL.

The fragment is not percent-decoded. If the URL does not have a fragment identifier, this property contains an empty string â€” "".

{{AvailableInWorkers}}

Syntax

string = object.hash;
object.hash = string;

Value

A {{domxref("USVString")}}.

Examples

var url = new URL('https://developer.mozilla.org/en-US/docs/Web/API/URL/href#Examples');
url.hash // Returns '#Examples'
