{{ApiRef("URL API")}}

The href property of the {{domxref("URL")}} interface is a stringifier that returns a {{domxref("USVString")}} containing the whole URL.

{{AvailableInWorkers}}

Syntax

string = object.href;
object.href = string;

Value

A {{domxref("USVString")}}.

Examples

var url = new URL('https://developer.mozilla.org/en-US/docs/Web/API/URL/href');
var result = url.href; // Returns: 'https://developer.mozilla.org/en-US/docs/Web/API/URL/href'
