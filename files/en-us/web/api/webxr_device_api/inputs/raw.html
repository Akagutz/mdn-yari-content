In this guide, we'll take a look at how to use WebXR's input device management features to select appropriate input devices and receive user input from them.

Actions

Primary actions

Each input source should define a primary action. A primary action is a platform-specific action which responds to the user manipulating it by delivering, in order, the events {{domxref("XRSession.selectstart_event", "selectstart")}}, {{domxref("XRSession.select_event", "select")}}, and {{domxref("XRSession.selectend_event", "selectend")}}. Each of these events is of type {{domxref("XRInputSourceEvent")}}.
