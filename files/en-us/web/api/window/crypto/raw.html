{{APIRef}}

The {{domxref("Window.crypto")}} property returns the {{domxref("Crypto")}} object associated to the global object. This object allows web pages access to certain cryptographic related services. Although the {{domxref("Window.crypto")}} property itself is read-only, all of its methods (and the methods of its child object, {{domxref("SubtleCrypto")}}) are not read-only, and therefore vulnerable to attack by polyfill.

Syntax

var cryptoObj = window.crypto || window.msCrypto; // for IE 11

Example

Using the {{domxref("Window.crypto")}} property to access the getRandomValues() method.

JavaScript

genRandomNumbers = function getRandomNumbers() {
  var array = new Uint32Array(10);
  window.crypto.getRandomValues(array);
 
  var randText = document.getElementById("myRandText");
  randText.innerHTML = "The random numbers are: "
  for (var i = 0; i < array.length; i++) {
    randText.innerHTML += array[i] + " ";
  }
}

HTML

<p id="myRandText">The random numbers are: </p>
<button type="button" onClick='genRandomNumbers()'>Generate 10 random numbers</button>
