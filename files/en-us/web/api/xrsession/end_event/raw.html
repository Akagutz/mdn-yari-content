A end event is fired on {{DOMxRef("XRSession")}} interface when the XR session is ended, either by application or User Agent.

It's a {{DOMxRef("XRSessionEvent")}} so it has an additional property {{DOMxRef("XRSessionEvent.session", "session")}} that referencess the {{DOMxRef("XRSession")}} that has ended.

Bubbles No
Cancelable No
Interface {{DOMxRef("XRSessionEvent")}}
Event handler {{DOMxRef("XRSession.onend")}}

Example

You can use the end event in an addEventListener method:

XRSession.addEventListener("end", function(event) {
  console.log("The XR session has ended")
});

Or use the {{DOMxRef("XRSession.onend")}} event handler property:

XRSession.onend = function(event) {
  console.log("The XR session has ended")
}
