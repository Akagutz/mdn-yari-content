The read-only visibilityState property of the {{DOMxRef("XRSession")}} interface is a string indicating whether the WebXR content is currently visible to the user, and if it is, whether it's the primary focus.

Every time the visibility state changes, a {{DOMxRef("XRSession.visibilitychange_event","visibilitychange")}} event is fired on the {{DOMxRef("XRSession")}} object.

Syntax

visibilityState = xrSession.visibilityState;

Value

A {{domxref("DOMString")}} containing one of the values defined in the enumerated type {{domxref("XRVisibilityState")}}; this string indicates whether or not the XR content is visible to the user and if it is, whether or not it's currently the primary focus.
