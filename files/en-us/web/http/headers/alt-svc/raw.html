The Alt-Svc header is used to list alternate ways to reach this website.

Syntax

Alt-Svc: clear
Alt-Svc: <service-list>; ma=<max-age>
Alt-Svc: <service-list>; ma=<max-age>; persist=1
<service-list>
Comma separated list of service definitions in the format of <service-name>=​"<host-name>:<port-number>", where <service-name> is a valid {{Glossary("ALPN")}} identifier.
<max-age>
The time in seconds for which the alternative service is considered valid for establishing new connections. Defaults to 24 hours.
persist{{Optional_Inline}}
Optional parameter specifying that the alternative service can be remembered between sessions and network changes.
