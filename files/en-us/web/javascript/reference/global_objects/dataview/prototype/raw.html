The DataView.prototype property represents the prototype for the {{jsxref("DataView")}} object.

{{js_property_attributes(0,0,0)}}

Description

DataView instances inherit from DataView.prototype. As with all constructors, you can change the constructor's prototype object to make changes to all DataView instances.

Properties

DataView.prototype.constructor
Specifies the function that creates an object's prototype. The initial value is the standard built-in DataView constructor.
{{jsxref("DataView.prototype.buffer")}} {{readonlyInline}}
The {{jsxref("ArrayBuffer")}} referenced by this view. Fixed at construction time and thus read only.
{{jsxref("DataView.prototype.byteLength")}} {{readonlyInline}}
The length (in bytes) of this view from the start of its {{jsxref("ArrayBuffer")}}. Fixed at construction time and thus read only.
{{jsxref("DataView.prototype.byteOffset")}} {{readonlyInline}}
The offset (in bytes) of this view from the start of its {{jsxref("ArrayBuffer")}}. Fixed at construction time and thus read only.
