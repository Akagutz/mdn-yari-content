The EvalError.prototype property represents the prototype of the {{jsxref("EvalError")}} constructor.

{{js_property_attributes(0, 0, 0)}}

Description

All {{jsxref("EvalError")}} instances inherit from EvalError.prototype. You can use the prototype to add properties or methods to all instances.

Properties

Specifies the function that created an instance's prototype.

Methods

Although the {{jsxref("EvalError")}} prototype object does not contain any methods of its own, {{jsxref("EvalError")}} instances do inherit some methods through the prototype chain.
