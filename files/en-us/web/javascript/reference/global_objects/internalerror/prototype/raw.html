The InternalError.prototype property represents the prototype of the {{jsxref("InternalError")}} constructor.

{{js_property_attributes(0, 0, 0)}}

Description

All {{jsxref("InternalError")}} instances inherit from InternalError.prototype. You can use the prototype to add properties or methods to all instances.
