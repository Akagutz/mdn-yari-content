The Iterator.prototype property represents the {{JSxRef("Iterator")}} prototype object.

{{js_property_attributes(0, 0, 0)}}

Description

All {{JSxRef("Iterator")}} instances inherit from Iterator.prototype. Changes to the Iterator prototype object are propagated to all {{JSxRef("Iterator")}} instances.
