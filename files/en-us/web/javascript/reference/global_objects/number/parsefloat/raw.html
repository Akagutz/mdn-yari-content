{{page("en-US/docs/Web/JavaScript/Reference/Global_Objects/parseFloat", "Parameters")}}

{{page("en-US/docs/Web/JavaScript/Reference/Global_Objects/parseFloat", "Return value")}}

Description

This method has the same functionality as the global {{jsxref("parseFloat", "parseFloat()")}} function:

Number.parseFloat === parseFloat; // true

and is part of ECMAScript 2015 (its purpose is modularization of globals). Please see {{jsxref("parseFloat", "parseFloat()")}} for more detail and examples.

Polyfill

if (Number.parseFloat === undefined) {
    Number.parseFloat = parseFloat;
}
