The Number.prototype property represents the prototype for the {{jsxref("Number")}} constructor.

{{js_property_attributes(0, 0, 0)}}

Description

All {{jsxref("Number")}} instances inherit from Number.prototype. The prototype object of the {{jsxref("Number")}} constructor can be modified to affect all {{jsxref( "Number")}} instances.

Properties

Number.prototype.constructor
Returns the function that created this object's instance. By default this is the {{jsxref("Number")}} object.
