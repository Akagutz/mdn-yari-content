The String.prototype property represents the {{jsxref("String")}} prototype object.

{{js_property_attributes(0, 0, 0)}}

Description

All {{jsxref("String")}} instances inherit from String.prototype. Changes to the String prototype object are propagated to all {{jsxref("String")}} instances.

Properties

String.prototype.constructor {{ReadOnlyInline}}
Specifies the function that creates an object's prototype.
{{jsxref("String.prototype.length")}} {{ReadOnlyInline}}
Reflects the length of the string.
N {{ReadOnlyInline}}
Used to access the character in the Nth position where N is an integer between 0 and one less than the value of {{jsxref("String.length", "length")}}.
