Atención: la aplicación principal del sistema en Gaia, que carga todas las otras aplicaciones, no funciona en Firefox ahora mismo, así que para abrir una aplicación tienes que escribir su subdominio (por ejemplo: http://browser.gaiamobile.org:8080 para abrir el navegador). Más información en el "bug" 796768

Si quieres probar B2G en Firefox 15 o más reciente, puedes hacerlo bajándote la interfaz de usuario Gaia y usándola en tu navegador. Esta página te explica cómo.

Es importante tener en cuenta que esto no es lo mismo que usar Boot to Gecko en un teléfono móvil. Hay varias diferencias, entre ellas:

  • Muchas de las funciones de las interfaces de hardware no funcionan.
  • Cada aplicación se abre en una nueva pestaña, en vez de todas en la misma.
  • Si usas 'Nightly', las versiones diarias pueden ser inestables.

Sin embargo, utilizar B2G en tu navegador te permitirá usar las herramientas para programadores de Firefox para experimentar y evaluar composición y otros elementos.

Prerrequisitos

Para utilizar Gaia en Firefox, necesitas lo siguiente:

  • Firefox 15 o más reciente
  • git (que puedes instalar con tu gestor de paquetes favorito. Para Mac, viene incluido en Xcode 4)
  • Un sistema de compilación. Para más información, visita Prerrequisitos para compilar Firefox OS
  • Una consola de comandos

Bajar Gaia

Atención: Si ya has hecho "pull" y te has bajado Gaia para compilar B2G, puedes usar la copia que tienes. No necesitas hacer "pull" de nuevo.

Si quieres bajarte Gaia para usarla con Firefox, puedes obtener sólo el código fuente de la interfaz y sus aplicaciones (ignorando el código específico para dispositivos móviles) haciendo lo siguiente:

git clone https://github.com/mozilla-b2g/gaia
cd gaia
DEBUG=1 make

Estos comandos obtendrán el código fuente y configurarán Gaia correctamente.

Abrir Gaia en Firefox

Hay dos formas de hacerlo. La primera es más fácil, pero su funcionalidad es más limitada. La segunda es un poco más complicada, pero simula mejor la experiencia de Boot to Gecko.

La forma fácil

Lanza la pantalla de inicio con Firefox ejecutando el siguiente comando en tu consola. Tienes que especificar la ruta correcta al Firefox que vas a utilizar, por supuesto.

/path/to/firefox index.html

También puedes simplemente arrastrar el archivo gaia/index.html a una pestaña de Firefox.

La mejor forma

Para obtener una mejor simulación de la experiencia de Gaia (y también un funcionamiento algo mejor de los APIs), utiliza el perfil especial que viene con Gaia. La URL en el comando siguiente es interceptado y enviado a un servidor web local que arranca al utilizar este perfil (una extensión especial para Firefox controla este servidor local mientras el perfil esta en uso).

/path/to/firefox -profile /Volumes/Mozilla/b2g/B2G/gaia/profile http://system.gaiamobile.org:8080

En Mac OS X, puedes ejecutar:

/Applications/Minefield.app/Contents/MacOS/firefox -profile /path/to/B2G/gaia/profile http://system.gaiamobile.org:8080
