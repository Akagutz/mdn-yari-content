Resumen

JavaScript (JS) es un lenguaje de programación principalmente utilizado en {{Glossary("Browser", "navegadores web")}} para escribir páginas web dinámicamente. Pero a menudo también del lado del {{Glossary("Server","servidor")}}.

No debes confundir JavaScipt con el lenguaje de programación Java, no están relacionados, pero aquí (art. en inglés) puedes ver una comparación.

A fondo

Un poco de historia

Concebido como un lenguaje de servidor por Brendan Eich (cuando trabajaba para Netscape Corp). JavaScript pronto se intregó a Netscape Navigator 2.0 en Septiembre de 1995. JavaScript gozó de un éxito inmediato, por lo que Microsoft introdujo en {{Glossary("MS Internet Explorer","Internet Explorer 3.0")}} un soporte de JavaScript bajo el nombre de JScript en Agosto de 1996.

Hecho conocido: Tiene JavaScript algo que ver con Java? Técnicamente hablando, no, para nada. Java y JavaScript son dos lenguajes de programación diferentes, con casi nada en común, pero aquí (art. en inglés) puedes ver una comparación. Sin embargo, en los primeros días de JavaScript, cuando Netscape decidió lanzarlo, también incluyó soporte para los Java applets dentro de su {{Glossary("browser")}}. Así, por razones de marketing, lo que antes se llamaba "LiveScript" fue rebautizado como "JavaScript".

En Noviembre de 1996, Netscape empieza a trabajar junto con ECMA International para convertir Javascript en un estándar dentro de la industria. Desde entonces, la versión estandarizada de JavaScript es conocida bajo el nombre de ECMAScript y la especificación es conocida como ECMA-262. La más conocida ( y más implementada) versión de ECMAScript es ECMA-262 3ra edición. La versión actual, disponible en todos los  {{Glossary("Browser", "navegadores web")}}  modernos es ECMA-262 5ta edición. Actualmente, se está trabajando en una 6ta edición de ECMAScript. 

Uso

JavaScript es principalmente utilizado en los {{Glossary("Browser", "navegadores web")}}. Permitiendo a los desarrolladores hacer muchísimas cosas: Manipular el contenido de las páginas a través del {{Glossary("DOM")}}, manipular datos con {{Glossary("AJAX")}} e {{Glossary("IndexedDB")}}, crear gráficos con {{Glossary("Canvas")}}, interactuar con el dispositivo que ejecuta el navegador utilizando varias {{Glossary("API","APIs")}}, etc.

El reciente aumento de las {{Glossary("API","APIs")}} disponibles en los navegadores, asi como una gran mejora en el rendimiento, lo convierte en uno de los lenguajes de programacion más usados en el mundo.

Recientemente, JavaScript ha vuelto al servidor con el éxito de Node JS. Esta plataforma provee un entorno completo de ejecución JavaScript fuera del navegador, capaz de ser usado en cualquier plataforma (Linux, MacOS y Windows). No es el único, pero el más usado recientemente para correr JavaScript fuera de los navegadores. Esto permite el uso de JavaScript como un lenguaje de scripting para automatizar ciertos procesos en la PC, así como construir {{Glossary("HTTP")}} completamente funcional y servidores {{Glossary("Web Sockets")}}.
