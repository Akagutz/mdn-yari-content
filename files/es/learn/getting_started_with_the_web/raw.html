Primeros pasos en la Web, es una serie concisa que te introduce en los aspectos prácticos del desarrollo web. Vas a configurar las herramientas que necesitas para construir una página web sencilla y publicar tu propio código.

La historia de tu primer sitio web

Es un montón de trabajo crear un sitio web profesional, así que si eres nuevo en el desarrollo web te animamos a empezar poco a poco. No vas a construir otro Facebook de inmediato, pero no es difícil tener tu propio sitio web sencillo en línea, así que vamos a empezar por ahí.

Trabajando a través de los siguientes artículos, pasarás de la nada a conseguir tu primera página web en la red. ¡Vamos!

Instalación del software básico

Cuando se trata de herramientas para la construcción de un sitio web, hay mucho para elegir. Si apenas estás comenzando, puede que te confundas por la variedad de editores de código, frameworks, y herramientas de testing que hay por ahí. En Instalación de software básico te mostramos paso a paso cómo instalar sólo el software que necesitas para comenzar con el desarrollo web básico.

¿Cuál será la apariencia de tu sitio web?

Antes de empezar a escribir el código de tu sitio web, debes planearlo primero. ¿Qué información mostrarás? ¿Qué tipo de letras y colores usarás? ¿Cuál será la apariencia de tu sitio web? Nosotros te proporcionamos un método sencillo que puedes seguir para planificar el contenido y diseño de tu sitio.

Manejando los archivos

Un sitio web consta de varios archivos: contenido de texto, código, hojas de estilo, contenido audiovisual, etc. Cuando estás construyendo un sitio web, necesitas ensamblar estos archivos en una estructura razonable y asegurarte de que pueden comunicarse entre sí. Manejando los archivos explica cómo configurar una estructura de archivos razonable y qué temas tener en cuenta.

Conceptos básicos de HTML

Hypertext Markup Language (HTML) es el código que se utiliza para estructurar el contenido de tu web, y darle sentido y propósito. Por ejemplo, ¿el contenido de mi pagina web es un conjunto de párrafos, o una lista de viñetas? ¿Tengo imágenes insertadas en mi página? ¿Tengo una tabla de datos? Sin abrumarte, Conceptos básicos de HTML proporciona suficiente información para que te familiarices con HTML.

CSS Básico

Cascading Stylesheets (CSS) es el código que se usa para dar estilo a tu sitio web. Por ejemplo, ¿deseas que el texto sea negro o rojo? ¿Dónde debe mostrarse el contenido en la pantalla? ¿Qué fondo de imágenes y colores deben usarse para decorar tu web? CSS Básico te lleva a través de lo que necesitas para empezar.

Fundamentos de JavaScript

JavaScript es el lenguaje de programación que se usa para agregar características interactivas a tu web, por ejemplo, juegos, cosas que suceden cuando se pulsan los botones o datos que se introducen en formularios, efectos dinámicos de estilo, animación y mucho más. Fundamentos de JavaScript te da una idea de lo que es posible con este lenguaje emocionante, y cómo empezar.

Publicar tu sitio web

Una vez terminado de escribir el código y organizado los archivos que componen tu sitio web, necesitas poner todo en la red para que la gente pueda encontrarlo. Publicar tu código de ejemplo te explica cómo obtener tu código de muestra simple en línea con el mínimo esfuerzo.

Cómo funciona la web

Cuando accedes a tu sitio web favorito, un montón de cosas complicadas suceden por detrás sin que te des cuenta. Cómo funciona la web describe lo que sucede cuando ves una página web en tu ordenador.
