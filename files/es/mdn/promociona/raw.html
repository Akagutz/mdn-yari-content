Plugin de WordPress

El plugin de wordpress para promocionar MDN ayuda a las personas que hacen blogs acerca de las tecnologias web o de productos de Mozilla; a promocionar tanto sus blogs como a MDN. Funciona de esta manera:

Cuando publicas un post, automaticamente se vinculan las palabras clave (como "HTML", "JavaScript", y "CSS", entre otras) a la respectiva pagina en MDN. Esto te permite escribir acerca de tecnologias web y crear links automatimaticamente a la documentación. ¡Es increiblemente útil!. Los terminos y sus paginas de destino en MDN están configuradas en la branch llamada def-list del repositorio Github -> https://github.com/mdn/wp-promote-mdn/tree/def-list.

Instalar este plugin es benefico para tí y para MDN, asi que, si todavia no lo has instalado, !Instalalo ahora¡ Si quieres contribuir en el codigo del plugin, dirigete al repositorio: wp-promote-mdn de Github.

Snippet de JavaScript

La manera recomendada de crear links entre las palabras clave y la documentacion en MDN es el plugin de Wordpress descrito anteriormente, pero si tu sitio o blog no se encuentra alojado en Wordpress, puedes insertar el codigo de Javascript que se encuentra un poco mas abajo para conseguir la misma funcion. Este script realiza la misma funcion que el plugin de Wordpress; crea links automaticamente en las palabras clave conocidas que se encuentran en el contenido de la página, hacia la documentacion respectiva en MDN.

Puedes cambiar el numero maximo de palabras clave que se convertiran en links cambiando el valor asignado a maxLinks que se encuentra en la var o. Tambien puedes especificar que elementos se buscaran, el nombre de la "class" para los links creados, y ademas puedes añadir mas palabras clave junto con la URL a la que estaran ligadas, para que el plugin las busque y creé los links.

Para usar este snippet, solo insertalo justo antes de esta etiqueta de cierre (closing tag): </body> al final de tu contenido.

<script type="text/javascript" >
    (function (d,f,a) {
        // Your settings here
        var o = {maxLinks: 4, searchElements: ['div', 'h'], linkClass: 'link-to-mdn', extraLinks: {'keywordx': 'http://example.com'}};

        var s=d.createElement("script");s.type="text/javascript";if(s.f)
        {s.a=function(){if(s.f=="loaded"||s.f=="complete")
        {s.a=null;PromoteMDN(o)}}}else{s.onload=function()
        {PromoteMDN(o)}}s.src="https://raw.githubusercontent.com/riverspirit/promote-mdn-script/master/promote-mdn.js";
        d.getElementsByTagName("head")[0].appendChild(s)
    })(document,'readyState','onreadystatechange');
</script>

Este snippet carga el codigo desde Github que realiza el verdadero trabajo. Si prefieres tener ese codigo en tu propio servidor (por seguridad, por rendimiento, o lo que sea) solo descargalo desde Github, añadelo a tu sitio, y corrige la URL que esta en la linea 9 del snippet anterior para que concuerde con el de tu servidor.
