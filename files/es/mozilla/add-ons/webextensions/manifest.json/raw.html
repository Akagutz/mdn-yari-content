El manifest.json es un archivo en formato JSON, y es el único archivo que toda WebExtension debe contener necesariamente.

Usando el manifest.json, puedes especificar los metadatos básicos de tu extensión como nombre y versión, así como aspectos funcionales de tu extensión, como scripts en segundo plano, y las acciones del navegador.

Este debe contener las siguientes claves:

applications. Solo soportado en Gecko.

manifest_version

name

version

Este también podría incluir cualquiera de las siguientes claves:

background

browser_action

page_action

content_scripts

default_locale. Debe estar presente solo si la carpeta _locales existe.

description

permissions

web_accessible_resources
