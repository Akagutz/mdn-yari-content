Firefox para dispositivos móviles

Tenemos varios productos móviles diferentes destinados a diferentes tareas, dispositivos y audiencias:

  • Construir Firefox para Android. (Codename: Fennec)
  • Construir Firefox para iOS, nuestro navegador de uso general para iOS con sincronización de escritorio incorporada.
  • Construir Firefox Focus,  nuestro navegador centrado en nuestra privacidad para iOS.

Para el desarrollo de escritorio y móvil, ten en cuenta lo siguiente:

  • Si bien se puede construir Firefox sobre hardware antiguo, puede llevar bastante tiempo compilar en máquinas más lentas. Se recomienda tener al menos 8 GB de RAM, y si es más mejor. El proceso de compilación es intensivo tanto en la CPU como en la E/S, por lo que también se recomienda que máquina tenga un SSD.
  • También se recomienda el acceso rápido a internet de banda ancha. Tanto el entorno de desarrollo como el repositorio de código fuente son bastante grandes.
  • Aunque puede compilar Firefox para que se ejecute en máquinas de 32 bits, el proceso de construcción de casi todos nuestros productos requiere un sistema operativo de 64 bits.
