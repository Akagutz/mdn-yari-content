JavaXPCOM provee un puente para incrustar aplicaciones Java en Gecko y usar los componentes XPCOM. Compilar  requiere tener instalado JDK y algunas opciones de configuración.

Requerimientos

Compilar JavaXPCOM requiere tener instalado un JDK.

  • Los compiladores de Macintosh normalmente ya lo traen.
  • Para Windows y Linux se puede descargar desde http://java.sun.com/j2se/1.5.0/download.jsp.

Opciones de configuración

Para activar la compilaciòn de componentes JavaXPCOM, debes compilar la extensión "java". Esto normalmente se logra colocando la siguiente etiqueta en tu .mozconfig:

ac_add_options --enable-javaxpcom

El código debe ser capaz de encontrar un JDK instalado. Tambièn debes configurar la variable de entorno JAVA_HOME<tt> para que apunte al directorio de tu JDK, o especificar adicionalmente la siguiente etiqueta: 

ac_add_options --with-java-include-path=dir
ac_add_options --with-java-bin-path=dir

Nota: Si estás trabajando en Mac OS X no debes especificar <tt>--with-java-include-path o --with-java-bin-path,por defecto /System/Library/Frameworks/JavaVM.framework/Headers.
