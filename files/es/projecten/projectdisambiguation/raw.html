Introducción

Desambiguación en la wiki de MDC, es el proceso de resolver ambigüedad, refiriendose a los conflictos que ocurren cuando los artículos acerca de dos o más temas diferentes tienen el mismo título "natural".

Cada página dentro de la wiki debe tener un título único. Esto puede causar problemas cuando hay dos páginas que pudieran, teoricamente, tener el mismo título. Por ejemplo, indexOf es un método para ambos objetos, Cadenas y Arrays en JavaScript.

NOTA: el contenido de esta política ha cambiado desde que se escribió por primera vez.

Como Wikipedia, diferenciamos páginas colocando identificadores en paréntesis después del título principal (ejemplo, "Mercurio (planeta)"). Como fuera, los usuarios que busquen información en el "indexOf" probablemente vendrán a Devmo y escribir "indexOf" en la caja de búsqueda. La búsqueda arrojará como resultado todas las páginas que contengan "indexOf", lo que puede volverse algo tedioso, dado que se mostrarán títulos, artículos, tutoriales, ejemplos de código, y referencias dentro de la wiki. Para ayudar a los lectores a encontrar lo que están buscando fácilmente, hemos empezado a agregar páginas de desambiguación en la wiki.

Cuándo crear una página de desambiguación

Las páginas de desambiguación deben ser creadas solo para resolver problemas de ambigüedad entre dos o más páginas.

Cómo crear una página de desambiguación

Las páginas de desambiguación son hechas a mano por nuestros editores para enlazar a las diferentes páginas que el lector podría estar buscando en el momento en que hacen una busqueda ambígua. Esto permite a los lectores buscar rápidamente la lista de páginas ambíguas y elegir la que está buscando.

Para crear una página de desambiguación, simplemente crea una página cuyo título es un término ambíguo; por ejemplo, indexOf.

Qué incluír en la página de desambiguación

Cuando estés creando una página de desambiguación por primera vez, necesitas hacer dos cosas:

Breve línea de introducción

Agrega una breve introducción en la parte superior de la página, como "indexOf es un método de varios objetos de JavaScript:", "indexOf es usado de varias formas:", o, más simple, "indexOf puede significar muchas cosas:".

La plantilla "desambigüar"

Agrega la plantilla desambigüar en el fondo de la página . Esto inserta una nota estándar y la categoría Desambiguación.

Lista enlazada de títulos de pagina ambíguos

Simplemente enlaza una lista de títulos de páginas ambíguas, con un breve resumen para cada uno. Te llevará a la página de ejemplos de desambiguación de indexOf.

Si una lista de los títulos y los resúmenes relacionados se vuelve difícil de manejar, eres bienvenido para organizar la lista dividiéndolo en secciones lógicas o agrupaciones.

Mejores prácticas

Las páginas de desambiguación solo deberán contener enlaces a páginas que estén dentro de la wiki.
No enlazar a otras páginas en el sumario en una página de desambiguación. Por ejemplo, no hagas esto: "indexOf - método para el objeto String en JavaScript". Solo déjalo como "indexOf - método para el objeto String en JavaScript". Enlaces extraños no tiene propósito en una página de desambiguación.
Acorta el enlace del título, si es necesario, al término quese está desambiguándo. En lugar de enlazar a to Core JavaScript 1.5 Reference:Objects:String:indexOf, usa la facilidad alt-text para acortarlo a indexOf.

Las páginas de desambiguación no están diseñadas para juegos "free association". Por favor úselas con cuidado y sólo cuando sea necesario.

Reparando enlaces a temas des-ambiguados

Después de que creas una página de desambiguación, haz clic en "Qué enlazar aquí" para ver si alguna página existente se enlaza a tu nueva página de desambiguación. Si es así, por favor ve a esas páginas y cambia los enlaces donde sea posible, así ellos señalan a la página correcta dentro de la wiki.

Por ejemplo, si una página que habla sobre el método JavaScript String object's indexOf se enlaza a la página de desambiguación indexOf, ese enlace debe ser removido para que así enlace directamente a la página Core JavaScript 1.5 Reference:Objects:String:indexOf.

Si estás creando una página de desambiguación que involucrará mover una página existente, haz clic en "Qué enlazar aquí" para encontrár todas las páginas que enlazan a la página que estás a punto de mover. Asegúrate de que esas páginas están arregladas o que no serán afectadas de manera significativa antes de que hagas el movimiento.

Un código de honor para la creación de páginas de desambiguación es arreglar todos los enlaces involucrados.
