HTML 4.0 añadió nuevos Elementos, eliminó algunos, desancosejó el uso de otros así como de algunos atributos y también añadió nuevos atributos. Esta nota técnica ofrece un resumen de la compatibilidad de los Elementos en HTML 4.0 sobre Netscape Navigator 4.x, Netscape 6, Internet Explorer 3.x-5.x y HTML 3.2.

Referencias

The definitive references for HTML 4.01 are:

  • <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
    "http://www.w3.org/TR/html4/strict.dtd">
  • <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
    "http://www.w3.org/TR/html4/loose.dtd">
  • <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN"
    "http://www.w3.org/TR/html4/frameset.dtd">
