Scripts y la política del mismo origen
La política del mismo origen previene que un documento o script cargado de un origen determinado acceda a las propiedades de un documento desde otro origen.
Configurando permisos de scripts
Las políticas de seguridad configurables de Mozilla permiten a los usuarios establecer políticas de seguridad para el navegador y además, tener diferentes políticas para distintos sitios de internet.
Vulnerabilidades y fixes conocidos en Mozilla
Esta página contiene una lista de las vulnerabilidades de seguridad conocidas que afectan a versiones particulares de productos  Mozilla e instrucciones sobre qué hacer para que los usuarios se protejan a si mismos. La lista será agregada cuando nuevos problemas de seguridad sean encontrados.
