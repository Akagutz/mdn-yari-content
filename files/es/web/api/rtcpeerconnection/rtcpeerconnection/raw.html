El constructor RTCPeerConnection()retorna un nuevo {{domxref("RTCPeerConnection")}}, el cual representa una conexion entre el dispositivo local y un par remoto.

Sintaxis

const pc = new RTCPeerConnection([configuration]);

Parámetros

configuration {{optional_inline}}
Un RTCConfiguration dictionary proveyendo opciones para configurar la nueva conexion.

Diccionario RTCConfiguration

{{page("/en-US/docs/Web/API/RTCConfiguration", "Properties")}}

Valor de retorno

Un nuevo objeto {{domxref("RTCPeerConnection")}}, configurado acorde a lo descripto por la configuracion, si esta fue especificada; De lo contrario, configurada usando sus opciones por defecto.
