Definición

El elemento q (cita) crea citas en linea, marca las citas a otros autores o documentos.

Sus etiquetas son: <q> y </q> (ambas obligatorias).
Crea una caja: en linea.
Está definido como: Elemento especial
Puede contener: cero o más elementos en linea

<p>
 El estándar dice: <q cite='http://html.conclase.net/w3c/html401-es/struct/text.html#h-9.2.2'>
 ... Q sirve para citas cortas (contenido en línea) que no necesitan cambios de párrafo.</q>
</p>
