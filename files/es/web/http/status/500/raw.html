The HyperText Transfer Protocol (HTTP) 500 Internal Server Error server error response code indicates that the server encountered an unexpected condition that prevented it from fulfilling the request.

El codigo 500 "Error Interno del Servidor", es un código de estado HTTP muy general que significa que algo salió mal en el servidor del sitio web, pero el servidor no podría ser más específico sobre cuál es el problema exacto.
