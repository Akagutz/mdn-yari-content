El Protocolo de Transferencia de HiperTexto (HTTP) 503 Servicio No Disponible código de error de respuesta del servidor indica que el servidor no está listo para manejar la solicitud.

Causas comunes son que un servidor esté abajo por mantención o esté con sobrecarga. Esta respuesta debería ser usada por condiciones temporales y la cabecera HTTP {{HTTPHeader("Retry-After")}} debería, si es posible, contener el tiempo estimado para la recuperación del servicio.

Nota: junto con esta respuesta, una página amistosa explicando el problema debería ser enviada.

Las cabeceras relacionadas con caché que son enviadas junto con esta respuesta deberían manejarse con cuidado, ya que un estado 503 es considerado una condición temporal y las respuestas no deberían usualmente ser almacenadas en caché .
