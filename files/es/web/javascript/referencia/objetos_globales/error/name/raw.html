La propiedad name otorga un nombre al tipo de error.

Descripción

Por defecto, las instancias de {{jsxref("Error")}} tienen establecido como nombre por defecto "Error". La propiedad name, además de la propiedad {{jsxref("Error.prototype.message", "message")}}, es utilizada por el método {{jsxref("Error.prototype.toString()")}} para crear una cadena de texto que represente al error.

Ejemplos

Arrojar un error personalizado

var e = new Error('Malformed input'); // e.name es 'Error'

e.name = 'ParseError';
throw e;
// e.toString() devolvería 'ParseError: Malformed input'
