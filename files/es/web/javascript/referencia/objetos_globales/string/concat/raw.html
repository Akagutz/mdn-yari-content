Resumen

El método concat() se utiliza para combinar dos o más cadenas. Este método no cambia las cadenas existentes, sino que retorna una nueva cadena.

Sintaxis

str.concat(cadena2,cadena3, ...,cadenaN)

Parámetros

cadena2...cadenaN
Cadenas que se concatenarán en la nueva cadena.

Descripción

concat crea una nueva cadena concatenando la cadena que invoca al método con el resto de cadenas pasadas como parámetro, ninguna de las cadenas originales se ve afectada.

Ejemplos

Ejemplo: Usando concat

El siguiente ejemplo combina cadenas en una nueva cadena.

cadena1="Oh "
cadena2="qué maravillosa "
cadena3="mañana'."
cadena4=cadena1.concat(cadena2,cadena3) // devuelve "Oh qué maravillosa mañana'."
