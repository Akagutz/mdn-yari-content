Scalable Vector Graphics, SVG, es un dialecto de marcado XML W3C para la generación de gráficos. Es parcialmente implementado en los principales navegadores como Firefox, Opera, navegadores WebKit, Internet Explorer y otros.

Este tutorial intenta explicar los detalles técnicos y las particularidades de SVG. Si pretendes dibujar hermosas imágenes, puedes encontrar más recursos en la documentación de Inkscape . Otro buena introducción a SVG te la ofrece la SVG Primer de W3C's.

El tutorial se encuentra en una etapa temprana de desarrollo. Si puedes, por favor ayuda introduciendo y escribiendo un párrafo o dos. ¡Puntos extra por escribir una página completa!
