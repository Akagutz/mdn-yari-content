المان <html> المان ریشه (المان مرتبۀ بالای) یک سند HTML را نمایش میدهد، بنابراین معمولاً به آن «المان ریشه» گفته میشود. تمام المان های دیگر، فرزندان این المان محسوب میشوند.

دسته بندی هیچ
محتوای قابل قبول

یک المان {{HTMLElement("head")}} که به دنبال آن یک المان {{HTMLElement("body")}} آمده است.
امکان حذف تگ ها

اگر اولین چیزی که در المان <html> قرار میگیرد، کامنت نباشد، میتوان تگ ابتدا را ننوشت.
اگر المان <html>  بلافاصله بعد از یک کامنت نیامده باشد، میتوان تگ انتها را ننوشت.
والدین قابل قبول

هیچ. این المان، المان ریشۀ سند است.

صفت ها

این المان شامل صفت های کلی میشود.

{{htmlattrdef("manifest")}} {{obsolete_inline}}
Specifies the {{glossary("URI")}} of a resource manifest indicating resources that should be cached locally. See Using the application cache for details.
{{htmlattrdef("version")}} {{obsolete_inline}}
Specifies the version of the HTML {{glossary("DTD", "Document Type Definition")}} that governs the current document. This attribute is not needed, because it is redundant with the version information in the document type declaration.
{{htmlattrdef("xmlns")}}
Specifies the {{glossary("XML")}} {{glossary("Namespace")}} of the document. Default value is "http://www.w3.org/1999/xhtml". This is required in documents parsed with XML {{glossary("parser", "parsers")}}, and optional in text/html documents.

مثال

<!DOCTYPE html>
<html lang="en">
  <head>...</head>
  <body>...</body>
</html>

Accessibility concerns

Providing a {{htmlattrxref("lang")}} attribute with a valid IETF identifying language tag on the <html> element will help screen reading technology determine the proper language to announce. The identifying language tag should describe the language used by the majority of the content of the page. Without it, screen readers will typically default to the operating system's set language, which may cause mispronunciations.

Including a valid lang declaration on the <html> element also ensures that important metadata contained in the page's {{HTMLElement("head")}}, such as the page's {{HTMLElement("title")}}, are also announced properly.
