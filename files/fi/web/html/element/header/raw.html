The HTML <header> element represents introductory content, typically a group of introductory or navigational aids. It may contain some heading elements but also other elements like a logo, a search form, an author name, and so on.

Käyttöohjeet

The <header> element is not sectioning content and therefore does not introduce a new section in the outline. That said, a <header> element is intended to usually contain the surrounding section's heading (an h1–h6 element), but this is not required.

Ominaisuudet

Tämä elementti sisältää vain globaalit ominaisuudet.

Esimerkkejä

Page Header

<header>
  <h1>Etusivun Otsikko</h1>
  <img src="mdn-logo-sm.png" alt="MDN logo">
</header>
