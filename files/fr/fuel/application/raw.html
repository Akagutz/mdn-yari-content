Synoptique

Attributs

Attribut Type Description
id Attribut AString en lecture seule L'id de l'application.
name Attribut AString en lecture seule Le nom de l'application.
version Attribut AString en lecture seule Le numéro de version de l'application.
console Attribut fuelIConsole en lecture seule L'objet console pour l'application.
extensions Attribut fuelIExtensions en lecture seule Les objets extensions pour l'application. Contient une liste de toutes les extensions installées.
prefs Attribut fuelIPreferenceBranch en lecture seule Les objets préférences pour l'application. Defaults to an empty root branch.
storage Attribut fuelISessionStorage en lecture seule L'objet storage pour l'application.
events Attribut fuelIEvents en lecture seule Les objets événements pour l'application. Accepte les valeurs : load, ready, quit, unload
bookmarks Attribut fuelIBookmarkFolder en lecture seule L'objet racine des marque-pages pour l'application.
windows Attribut nsIVariant en lecture seule Un tableau des fenêtres du navigateur dans l'application.
activeWindow Attribut fuelIWindow en lecture seule La fenêtre active du navigateur.
