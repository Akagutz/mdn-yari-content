AJAX (Asynchronous {{glossary("JavaScript")}} And {{glossary("XML")}}) est une pratique de programmation consistant à construire des pages Web plus complexes et dynamiques en utilisant une technologie connue sous le nom de {{glossary("XHR_(XMLHttpRequest)","XMLHttpRequest")}}.

Ce qu'AJAX vous permet de faire est de mettre à jour seulement certaines parties du {{glossary("DOM")}} d'une page web en {{glossary("HTML")}}, au lieu d'avoir à recharger toute la page. AJAX vous permet également de travailler de manière asynchrone, ce qui signifie que votre code continue à s'exécuter pendant que cette partie de votre page tente de se recharger (par opposition à synchrone, qui empêchera votre code de s'exécuter jusqu'à ce que cette partie de votre page web ait fini de se recharger).

Avec des sites Web interactifs et les standards web modernes, AJAX est progressivement remplacé par des fonctions dans des frameworks JavaScript et la norme officielle Fetch API.
