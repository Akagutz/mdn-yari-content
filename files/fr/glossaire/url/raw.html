Une URL (Uniform Resource Locator) est une chaîne de caractères indiquant où une ressource peut être trouvée sur Internet.

Dans le contexte {{Glossary("HTTP")}}, les URL sont appelées des « adresses Web » ou « liens ». Votre navigateur affiche les URL dans sa barre d'adresse, par exemple https://developer.mozilla.org

Les URL peuvent aussi être utilisées pour des transferts de fichiers ({{Glossary("FTP")}}), des courriers électroniques ({{Glossary("SMTP")}}) et d'autres applications.
