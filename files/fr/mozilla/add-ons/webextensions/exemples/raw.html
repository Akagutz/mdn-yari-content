Pour illustrer la façon de développer des extensions, nous maintenons un référentiel d'exemples simples des extensions à https://github.com/mdn/webextensions-examples. Cet article décrit les API des WebExtensions utilisées dans ce référentiel.

Ces exemples fonctionnent dans Firefox Nightly : la plupart travaillent dans les versions antérieures de Firefox, mais vérifiez la version minimum strict_min_version dans le fichier manifest.json de l'extension pour en être sur.

Si vous souhaitez essayer ces exemples, vous avez trois choix :

  1. Cloner le référentiel, puis charger l'extension directement à partir de son répertoire source, en utilisant la fonctionnalité "Charger temporairement l'extension". L'extension restera chargée jusqu'à ce que vous redémarriez Firefox.
  2. Cloner le référentiel, puis utiliser les outils de lignes de commande web-ext pour exécuter Firefox avec l'extension installée.
  3. Cloner le référentiel, puis accéder au répertoire du build. Cela contient des versions intégrées et signées de tous les exemples, afin que vous puissiez les ouvrir dans Firefox (en utilisant Fichier/Ouvrir fichier) et les installer en permanence, tout comme une extension que vous installez à partir addons.mozilla.org.

Important: Veuillez ne pas soumettre ces exemples de WebExtension à AMO (addons.mozilla.org), vous n'avez pas besoin de signer l'add-on pour exécuter les exemples de WebExtension. Il suffit de suivre les étapes ci-dessus.
