Cette page tente de conserver une trace des principales branches publiques afin d'aider les développeurs à accéder à la bonne version du code source lorsqu'ils travaillent sur un bogue spécifique à une version. Les entrées les plus récentes sont affichées les premières.

Informations complémentaires

Pour savoir sur quelle branche est basé un projet, consultez la carte des Trunk 1.9 et Branch 1.8 (en).

Lisez comment marquer une version de Mozilla (en) si vous créez votre propre version.
