Exemples de Code NSS

Les exemples de code ci-dessous montrent comment NSS peut être utilisé pour des opérations cryptographiques, de gestion de certificats, pour SSL, etc. Ils montrent aussi de bonnes pratiques pour l'application dans le domaine de la cryptographie.

 

  1. Exemple de code 1 : Génération de clef et transport entre les serveurs
  2. Exemple de code 2 :  Chiffrement symétrique
  3. Exemple de code 3 : Hachage, MAC
  4. Exemple de code 4 : Chiffrement PKI
  5. Exemple de code 5 : Chiffrement PKI avec clef publique et privée utilisant le format DER
  6. Exemple de code 6 : Clefs symétrique persistantes dans la base de données NSS
