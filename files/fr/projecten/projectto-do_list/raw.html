Vous cherchez un moyen d'aider à améliorer la documentation, mais ne savez pas par où commencer ? Nous vous suggérons de regarder les catégories suivantes, afin de voir si une des pages a été spécifiquement marquée pour attirer l'attention :

  • NeedsExample
  • NeedsTechnicalReview - utilisez la case à cocher "Needs Technical Review" dans l'éditeur pour marquer la page comme nécessitant une revue technique.
  • NeedsEditorialReview  - si vous voulez marquer la page comme nécessitant une revue éditorielle.
  • NeedsUpdate
  • NeedsRewrite
  • NeedsContent
  • NeedsMigration
  • NeedsMarkupWork
  • NeedsImageCleanup
  • NeedsMigrationReview - si vous avez migré une page et aimeriez une personne pour la valider, et l'ajouter à une categorie.
  • NeedsAttention - pour toutes les pages qui nécessitent une plus grande attention.

La suite contiend une liste d'articles dans ces catégories qui ont été marquées comme nécessitant un travail :

  • JavaScript
    • NeedsTechnicalReview
    • NeedsContent
    • NeedsBrowserCompatibility
    • NeedsEditorialReview
  • DOM
    • NeedsTechnicalReview
    • NeedsExample
    • NeedsContent
    • NeedsUpdate
    • NeedsBrowserCompatibility
    • NeedsEditorialReview

Taches diverses :

  • Intégrer cet example, compositing with alpha blending, dans le tutoriel de Canvas, vers la leçon de Compositing. L'exemple existant sur les formes opaques est bon pour montrer le concept de composition, alors que le nouveau montre comment alpha les affecte.

Si vous désirez travailler sur d'autres sujet, passez sur les documentation wishlist, ou how to help pour avoir encore plus d'idées.
