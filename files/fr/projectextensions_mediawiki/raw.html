Extension de navigation « fil d'ariane » (breadcrumbs)

Le wiki devmo a une extension propre, qui permet la génération automatique des « fils d'ariane » sur une page. Pour l'utiliser, ajoutez simplement l'élement suivant (avec ses deux balises, d'ouverture et de fermeture), sur votre page wiki.

<breadcrumbs></breadcrumbs>

 

Notes

  1. L'extension ne fonctionne pas lorsqu'elle est ajoutée comme une simple balise vide (par ex. : <breadcrumbs/>). Les deux parties, ouverture et fermeture de l'élément, sont nécessaires.
  2. L'extension a un bug pour l'instant :
    1. Elle ne fonctionne pas sur les page qui n'ont pas de sous-page, donc bien qu'elle fonctionne sur une page appelée "JavaScript:Articles:Foobar", elle ne fonctionnera pas sur une page dont le titre est juste "JavaScript". Un problème supplémentaire a été trouvé, et la correction qui semblait avoir été apportée devra être revue.

Extension de titre

Cette extension permet d'afficher le titre d'une page avec les caractères soulignés, ce qui n'est pas possible par défaut dans MediaWiki.

Pour le moment, cette extension ne change le titre seulement lors de son affichage dans la navigation breadcrumb, insérée par l'extension breadcrumb. Cela fonctionnera prochainement pour l'affichage des titres ailleurs.

Pour utiliser cette extension, ajoutez simplement le titre comme il devrait s'afficher, entouré de balises title :

<title>MAX_VALUE</title>

Vous pouvez voir comme l'extension fonctionne sur MAX_VALUE.

Notes

  • Suite à des limitation techniques liées à la mise à jour vers MediaWiki1.5, cette extension a été changée de <title> à <title-override> ; si vous rencontrez une page avec l'ancienne version, n'hésitez pas à la mettre à jour vers la nouvelle version.

Extension de flux RSS

L'extension PHP MagpieRSS et l'extension MediaWiki Mafs/RSS ont été installées.

Utilisation

charset=
Le jeu de caractères utilisé par le flux. iconv est utilisé pour le convertir.
short
Ne pas afficher le texte de description/sommaire pour chaque élement. ("long" est la valeur par défaut).
max=X
Affiche les X titres les plus récents.
highlight= term1 term2
Les termes séparés par des espaces seront surlignés à l'affichage.
filter= term1 term2
Affiche uniquement les éléments RSS qui contiennent au moins un des termes spécifiés.
reverse
Affiche les éléments RSS dans l'ordre inverse.

Extension pour l'élément Abbr

MDC a ajouté une extension permettant l'utilisation de l'élément <abbr>...</abbr> dans ses pages. Celui-ci doit être utilisé comme l'élément HTML standard abbr, pour indiquer lorsqu'un bout de texte est une abréviation et donner sa signification. De nombreux attributs HTML courants sont disponibles pour celui-ci (mais pas tous). (title, id, class, lang, dir).
