Les templates ont été améliorés d'une manière significative dans Firefox 3. Principalement, il est possible d'utiliser des processeurs de requêtes personnalisés pour gérer d'autres types de sources de données que RDF. Une nouvelle syntaxe d'interrogation rend cela possible. Un support pour les sources XML est également fourni. Une description complète des nouvelles fonctionnalités disponibles dans les templates {{ interwiki('wikimo', 'XUL:Template_Features_in_1.9', 'est disponible') }}. ({{ Bug(285631) }})

Autres améliorations des templates

  • Des conditions relationnelles ont été ajoutées pour permettre un contrôle plus précis des résultats correspondants à une règle. Ceci permet par exemple de sélectionner les résultats qui commencent ou se termines par certaines chaînes, ou qui sont avant ou après d'autres valeurs.
  • Un paramètre 'dont-recurse' a été ajouté pour empêcher la récursion de se produire afin qu'un seul niveau de résultats soit généré
  • Des API ont été ajoutées au constructeur de templates pour récupérer un objet de résultat représentant un élément de sortie.
  • Le service de tri XUL est plus robuste est trie mieux les arbres de contenu ou non. Il permet également de trier du contenu non construit par le template. ({{ Bug(335122) }})
