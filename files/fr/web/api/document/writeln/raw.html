Écrit une chaîne de caractères suivit d'un retour de ligne.

Syntax

document.writeln(line);

Parameters

  • line est la chaine de caractère contenant le text.

Example

document.writeln("<p>Entrer votre mot de passe :</p>");

Notes

document.writeln est identique à document.write, mais ajoute une nouvelle ligne.

Note: document.writeln (comme document.write) n'a aucun effet en XHTML documents (vous obtiendrais, "Operation is not supported" (NS_ERROR_DOM_NOT_SUPPORTED_ERR) erreur dans la console). Ceci est le cas si un fichier est ouvert en local avec .xhtml extension ou pour tous documents utilisant une application/xhtml+xml MIME type. Pour plus d'information visité W3C XHTML FAQ.
