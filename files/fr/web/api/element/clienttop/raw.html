Résumé

La largeur de la bordure supérieure d'un élément en pixels. Celle-ci ne comprend pas les marges ou le remplissage (padding) supérieur. clientTop est en lecture seule.

Les applications basées sur Gecko gèrent clientTop à partir de Gecko 1.9 (Firefox 3, implémentation dans le {{ Bug(111207) }}). Cette propriété n'est pas disponible dans Firefox 2 et les versions précédentes.

Syntaxe

vartop = element.clientTop;
