L'évènement mousedown est déclenché lorsqu'un dispositif de pointage (une souris par exemple) est pressé sur un élément (comme un bouton).

Informations générales

Spécification
DOM L3
Interface
{{domxref("MouseEvent")}}
Propagation
Oui
Annulable
Oui
Cible
Element
Action par défaut
Varie : Commencer une opération drag/drop ; commener une sélection de texte ; commencer une interaction de défilement (en combinaison avec la roulette de la souris, si supporté)

Propriétés

Propriété Type Description
target {{readonlyInline}} {{domxref("EventTarget")}} La cible de l'événement (la cible la plus haute dans l'arbre DOM).
type {{readonlyInline}} {{domxref("DOMString")}} Le type de l'événement.
bubbles {{readonlyInline}} Boolean Si l'événement se propage normalement ou non.
cancelable {{readonlyInline}} Boolean Si l'événement est annulable ou non ?
view {{readonlyInline}} {{domxref("WindowProxy")}} {{domxref("document.defaultView")}} (window du document)
detail {{readonlyInline}} long (float) Un compteur du nombre de clics consécutifs survenus dans un court laps de temps, incrémenté de un.
currentTarget {{readonlyInline}} {{domxref("EventTarget")}} The node that had the event listener attached.
relatedTarget {{readonlyInline}} {{domxref("EventTarget")}} Pour les événements  mouseover, mouseout, mouseenter et mouseleave: la cible de l'événement complémentaire (la cible de mouseleave dans le cas de l'événement mouseenter ). sinon null.
screenX {{readonlyInline}} long La coordonnée en X du pointeur de la souris dans le système de coordonnées globales (l'écran).
screenY {{readonlyInline}} long La coordonnée en Y du pointeur de la souris dans le système de coordonnées globales (l'écran).
clientX {{readonlyInline}} long La coordonnée en X du pointeur de la souris dans le système de coordonnées locales (l'élément du DOM).
clientY {{readonlyInline}} long La coordonnée en Y du pointeur de la souris dans le système de coordonnées locales (l'élément du DOM).
button {{readonlyInline}} unsigned short Le numéro du bouton préssé quand l'événement a été déclenché: Bouton gauche=0, Bouton du milieu=1 (si présent), Bouton droit=2. Pour les souries configurées pour les personnes gauchères et dont les actions des boutons sont inversées, les valeurs sont lues de droite à gauche.
buttons {{readonlyInline}} unsigned short Les boutons relachés quand l'événement a été déclenché: Bouton gauche=1, Bouton droit=2, Bouton du milieu (roulette)=4, 4ème bouton (Généralement, le bouton "Page précedente")=8, 4ème bouton (Généralement, le bouton "Page suivante")=16. Si deux boutons ou plus ont été relachés, retourne la somme logique des valeurs. E.g., si les boutons droit et gauche sont relachés, retourn 3 (=1 | 2). Plus d'infos.
mozPressure {{readonlyInline}} float La quantité de pression appliquée à une touche ou à une tabdevice; cette valeur est comprise entre 0.0 (pression minimum) et 1.0 (pression maximum).
ctrlKey {{readonlyInline}} Boolean true si la touche control était enfoncée quand l'événement a été declenché. sinonfalse.
shiftKey {{readonlyInline}} Boolean true si la touche shift était enfoncée quand l'événement a été declenché. sinonfalse.
altKey {{readonlyInline}} Boolean true si la touche alt était enfoncée quand l'événement a été declenché. sinonfalse.
metaKey {{readonlyInline}} Boolean true si la touche meta était enfoncée quand l'événement a été declenché. sinonfalse.
