L'événement mousemove se déclenche quand un dispositif de pointage (la plupart du temps une souris) se déplace alors qu'elle est au dessus d'un élément.

Informations générales

Spécification
DOM L3
Interface
MouseEvent
Propagation
Oui

Oui

Element
Action par défaut
Aucune

Propriétés

Propriété Type Description
target {{readonlyInline}} {{domxref("EventTarget")}} La cible de l'événement (la plus haute dans l'arbre DOM).
type {{readonlyInline}} {{domxref("DOMString")}} Le type d'événement.
bubbles {{readonlyInline}} Boolean Si l'événement se propage par défaut ou non.
cancelable {{readonlyInline}} Boolean Si l'événement peut être annulé ou non.
view {{readonlyInline}} {{domxref("WindowProxy")}} {{domxref("document.defaultView")}} (window du document)
detail {{readonlyInline}} long (float) Un compteur du nombre de clics consécutifs survenus dans un faible laps de temps, incrémenté par un.
currentTarget {{readonlyInline}} {{domxref("EventTarget")}} Le noeud auquel l'écoute d'événement est rattaché.
relatedTarget {{readonlyInline}} {{domxref("EventTarget")}} Pour les événements mouseover, mouseout, mouseenter et mouseleave : la cible de l'événement complémentaire ( la cible de l'événement mouseleave dans le cas d'un événement mouseenter). null sinon.
screenX {{readonlyInline}} long La coordonnée X du pointeur de la souris dans les coordonnées globales (celles de l'écran).
screenY {{readonlyInline}} long La coordonnée Y du pointeur de la souris dans les coordonnées globales (celles de l'écran).
clientX {{readonlyInline}} long La coordonnée X du pointeur de la souris dans les coordonnées locales (celles du contenu DOM).
clientY {{readonlyInline}} long La coordonnée Y du pointeur de la souris dans les coordonnées locales (celles du contenu DOM).
button {{readonlyInline}} unsigned short Toujours à 0 car aucun bouton ne déclenche l'événement mousemove.
buttons {{readonlyInline}} unsigned short Les boutons de la sourie pressés lors du déclenchement de l'événement: Clic gauche=1, Clic droit=2, Bouton du milieu (rouletteà=4, 4e bouton (généralement le bouton "retour en arrière du navigateur")=8, 5e boutone (généralement le bouton "retour en avant du navigateur")=16. Si deux boutons ou plus sont pressés, retourne la somme logique de ces valeurs. Ex : Si le clic droit et le clic gauches sont pressés, retourne 3 (=1 | 2). Plus d'infos.
mozPressure {{readonlyInline}} float La quantité de pression appliquée par un touché sur l'écran lors de la génération de l'événement ; cette donnée peut prendre n'importe quelle valeur allant de 0.0 (pression minimum) à 1.0 (pression maximum).
ctrlKey {{readonlyInline}} boolean truesi la clef de contrôle était enfoncée au moment du déclenchement de l'événement. false sinon.
shiftKey {{readonlyInline}} boolean truesi shift était enfoncée au moment du déclenchement de l'événement. false sinon.
altKey {{readonlyInline}} boolean truesi alt était enfoncée au moment du déclenchement de l'événement. false sinon.
metaKey {{readonlyInline}} boolean truesi la clef meta était enfoncée au moment du déclenchement de l'événement. false sinon.
