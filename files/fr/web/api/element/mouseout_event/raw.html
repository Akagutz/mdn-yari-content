L'événement mouseout est déclenché quand un outil de pointage (généralement une souris) est déplacé hors de l'élément mis sur écoute ou un de ses enfants. Notez qu'il est aussi déclenché lorsque vous quittez l'élément parent pour vous déplacer sur un élément enfant, puisque vous sortez de l'espace du parent.

Exemple

L'exemple suivant illustre la différence entre les événements mouseout et mouseleave.

<ul id="test">
  <li>objet 1</li>
  <li>objet 2</li>
  <li>objet 3</li>
</ul>

<script>
  var test = document.getElementById("test");
  
  
  // ce gestionnaire ne sera exécuté qu'une seule fois quand le curseur sortira de
  // de la liste non-organisée
  test.addEventListener("mouseleave", function( event ) {   
    // mise en valeur de la cible de mouseleave
    event.target.style.color = "purple";

    // réinitialisation la couleur après un court délai
    setTimeout(function() {
      event.target.style.color = "";
    }, 500);
  }, false);
  
  
  // ce gestionnaire sera exécuté à chaque fois que le curseur sortira d'un élément
  // de la liste
  test.addEventListener("mouseout", function( event ) {   
    // mise en valeur de la cible de mouseout
    event.target.style.color = "orange";

    // réinitialisation la couleur après un court délai
    setTimeout(function() {
      event.target.style.color = "";
    }, 500);
  }, false);
</script>
