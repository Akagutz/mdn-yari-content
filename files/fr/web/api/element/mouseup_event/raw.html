L'événement mouseup est déclenché quand un dispositif de pointage est relâché au dessus d'un élément.

buttons {{readonlyInline}} unsigned short The buttons depressed when the mouse event was fired: Left button=1, Right button=2, Middle (wheel) button=4, 4th button (typically, "Browser Back" button)=8, 5th button (typically, "Browser Forward" button)=16. If two or more buttons are depressed, returns the logical sum of the values. E.g., if Left button and Right button are depressed, returns 3 (=1 | 2). More info.
mozPressure {{readonlyInline}} float The amount of pressure applied to a touch or tabdevice when generating the event; this value ranges between 0.0 (minimum pressure) and 1.0 (maximum pressure).
ctrlKey {{readonlyInline}} boolean true if the control key was down when the event was fired. false otherwise.

altKey {{readonlyInline}} Boolean true if the alt key was down when the event was fired. false otherwise.
metaKey {{readonlyInline}} Boolean true if the meta key was down when the event was fired. false otherwise.
