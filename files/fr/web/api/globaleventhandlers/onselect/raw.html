{{ ApiRef("HTML DOM") }}

Un gestionnaire d'évènement pour l'élément de la fenêtre seléctionné.

Syntaxe

window.onselect = funcRef;
  • funcRef est une fonction.

Example

<html>
<head>

<title>onselect test</title>

<style type="text/css">
.text1 { border: 2px solid red; }
</style>

<script type="text/javascript">

window.onselect = selectText;

function selectText()
{
 alert("select event detected!");
}
</script>
</head>

<body>
<textarea class="text1" cols="30" rows="3">
Mettez en surbrillance une partie de ce texte 
avec le pointeur de la souris 
pour déclencher l'événement d'onselect.
</textarea>
</body>
</html>

Notes

L'évènement select se déclenche uniquement pour du texte à l'intérieur d'un text input un textarea, est sélectionné.
