Retourne un Document, le document actif dans le contexte de navigation intégré du cadre en ligne.

Exemple de contentDocument

var documentIframe = document.getElementsByTagName("iframe")[0].contentDocument;

documentIframe.body.style.backgroundColor = "blue";
// Cela rendra bleu le iframe.
