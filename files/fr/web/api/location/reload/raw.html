{{ APIRef("Location") }}

La méthode Location.reload() recharge la ressource depuis l'URL actuelle. Son unique paramètre optionnel est un {{domxref("Boolean")}}, qui, quand il vaut true, force la page à toujours être rechargée depuis le serveur. Si ce paramètre vaut false ou n'est pas spécifié, le navigateur peut éventuellement recharger la page depuis son cache.

Si l'action est impossible à cause d'une violation de sécurité, une {{domxref("DOMException")}} de type SECURITY_ERROR est lancée. Cela se produit si l'origine du script appelant est différente de l'origine de la page initialement désignée par l'objet {{domxref("Location")}}, principalement quand le script est hébergé sur un autre domaine.

Syntaxe

object.reload(forcedReload);

Paramètres

forcedReload {{optional_inline}}
Un flag {{domxref("Boolean")}}, qui, s'il vaut true, force la page à toujours être rechargée depuis le serveur. S'il vaut false ou n'est pas spécifié, le navigateur peut éventuellement recharger la page depuis son cache.

Exemples

// Recharge la page actuelle, sans utiliser le cache
document.location.reload(true);
