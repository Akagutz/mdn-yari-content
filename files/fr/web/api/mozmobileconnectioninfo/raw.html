The MozMobileConnectionInfo interface allows to access connection information for voice or data. The {{domxref("window.navigator.mozMobileConnection","navigator.mozMobileConnection")}} uses it through its {{domxref("MozMobileConnection.voice","voice")}} and {{domxref("MozMobileConnection.data","data")}} properties.

Interface overview

interface MozMobileConnectionInfo
{
  readonly attribute {{domxref("MozMobileCellInfo")}} cell;
  readonly attribute boolean connected;
  readonly attribute boolean emergencyCallsOnly;
  readonly attribute unsigned short lastKnownMcc;
  readonly attribute {{domxref("MozMobileNetworkInfo")}} network;
  readonly attribute unsigned short? relSignalStrength;
  readonly attribute boolean roaming;
  readonly attribute object? signalStrength;
  readonly attribute DOMString? state;
  readonly attribute DOMString type;
};

Properties

{{domxref("MozMobileConnectionInfo.cell")}} {{readonlyinline}}
A {{domxref("MozMobileCellInfo")}} object that gives access to cell location information.
{{domxref("MozMobileConnectionInfo.connected")}} {{readonlyinline}}
A boolean that indicates whether the connection is ready.
{{domxref("MozMobileConnectionInfo.emergencyCallsOnly")}} {{readonlyinline}}
A boolean that indicates whether only emergency calls are possible.
{{domxref("MozMobileConnectionInfo.lastKnownMcc")}} {{readonlyinline}}
A number representing the Mobile Country Code (MCC) of the last known network operator.
{{domxref("MozMobileConnectionInfo.network")}} {{readonlyinline}}
A {{domxref("MozMobileNetworkInfo")}} object that gives access to the network operator information.
{{domxref("MozMobileConnectionInfo.relSignalStrength")}} {{readonlyinline}}
A number that gives the signal strength, represented linearly as a number between 0 (weakest signal) and 100 (full signal) or null if no signal at all.
{{domxref("MozMobileConnectionInfo.roaming")}} {{readonlyinline}}
A boolean that indicates whether the connection is going through a foreign operator (roaming) or not.
{{domxref("MozMobileConnectionInfo.signalStrength")}} {{readonlyinline}}
A number that gives the signal strength in dBm, or null if no service is available.
{{domxref("MozMobileConnectionInfo.state")}} {{readonlyinline}}
A string that gives the state of the connection.
{{domxref("MozMobileConnectionInfo.type")}} {{readonlyinline}}
A string that gives the type of connection.

Methods

None.
