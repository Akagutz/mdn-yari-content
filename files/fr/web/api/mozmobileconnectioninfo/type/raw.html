Résumé

type est une propriété en lecture seule qui indique le type de connexion.

Syntaxe

var dataType  = navigator.mozMobileConnection.data.type;
var voiceType = navigator.mozMobileConnection.voice.type;

Valeur

Retourne une chaine.

Les valeurs possibles sont : gsm, cdma, gprs, edge, umts, hsdpa, evdo0, evdoa, evdob, etc.

Spécification

Ne fait partie d'aucune spécification.
