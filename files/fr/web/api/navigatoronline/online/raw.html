Résumé

Renvoie une valeur booléenne indiquant si le navigateur est en ligne ou non.

Syntaxe

online = window.navigator.onLine;
  • online est une valeur booléenne (true ou false).

Exemple

alert(navigator.onLine ? "Vous êtes en ligne" : "Vous êtes hors ligne");

Notes

Consultez Évènements online et offline‎ pour une description plus détaillée de cette propriété ainsi que des nouvelles fonctionnalités liées au travail hors ligne introduites dans Firefox 3.
