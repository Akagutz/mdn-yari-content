Sommaire

Retourne un objet {{domxref("MimeTypeArray")}} qui contient une liste de d'objets {{domxref("MimeType")}} representant les types MIME reconnus par le navigateur. 

Syntaxe

typesMime = navigator.mimeTypes;

mimeTypes est un objet MimeTypeArray qui a un attribut length et les méthodes item(index) et namedItem(name).

Exemple

function estJavaPresent() {
  return 'application/x-java-applet' in navigator.mimeTypes;
}

function getJavaPluginDescription() {
  var mimetype = navigator.mimeTypes['application/x-java-applet'];
  if (mimetype === undefined) {
    // no Java plugin present
    return undefined;
  }
  return mimetype.enabledPlugin.description;
}

Specification

Ceci ne fait objet d'aucune spécification.
