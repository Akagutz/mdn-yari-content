{{APIRef("Web Storage API")}}

La méthode removeItem() de l'interface {{domxref("Storage")}} , lorsque vous lui passez une clé en argument, va supprimer la ressource avec le nom de clé correspondant du storage.

Syntaxe

storage.removeItem(nomCle);

Paramètres

nomCle
Un  {{domxref("DOMString")}} qui contient le nom de la clé que vous voulez supprimer.

Valeur de retour

Pas de valeur de retour.

Exemple

La fonction suivante crée trois données dans le local storage, puis va supprimer la donnée image.

function populateStorage() {
  localStorage.setItem('bgcolor', 'red');
  localStorage.setItem('font', 'Helvetica');
  localStorage.setItem('image', 'myCat.png');

  localStorage.removeItem('image');
}

Note: Pour voir ce code en fonctionnement, voir Web Storage Demo.
