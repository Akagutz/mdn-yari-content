{{APIRef ("CSSOM")}}

Un objet qui implémente l'interface StyleSheet représente une seule feuille de style. Les feuilles de style vont en plus implanter l'interface plus spécialisée {{domxref ("CSSStyleSheet")}}.

Propriétés

{{Domxref ("StyleSheet.disabled")}}
Est un {{domxref ("Boolean")}} représentant si la feuille de style est appliquée ou non.
{{Domxref ("StyleSheet.href")}} {{readonlyInline}}
Renvoie un {{domxref ("DOMString")}} représentant l'emplacement de la feuille de style.
{{Domxref ("StyleSheet.media")}} {{readonlyInline}}
Renvoie un {{domxref ("MediaList")}} représentant le milieu de destination pour les  informations de style.
{{Domxref ("StyleSheet.ownerNode")}} {{readonlyInline}}
Renvoie un {{domxref ("Node")}} associant cette feuille de style avec le document en  cours.
{{Domxref ("StyleSheet.parentStyleSheet")}} {{readonlyInline}}
Renvoie un {{domxref ("StyleSheet")}} comprenant celui‐ci s'il existe ; retourne  null s'il n'y en a pas.
{{Domxref ("StyleSheet.title")}} {{readonlyInline}}
Renvoie un {{domxref ("DOMString")}} représentant le titre consultatif de la feuille de style en cours.
{{Domxref ("StyleSheet.type")}} {{readonlyInline}}
Renvoie une {{domxref ("DOMString")}} représentant la langue de feuille de style pour cette feuille de style.
