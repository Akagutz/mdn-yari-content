Sends an {{domxref("XDomainRequest")}} which has previously been opened calling {{domxref("XDomainRequest.open()")}}.

Syntax

xdr.send(data);

Parameters

data
le formulaire (data) à envoyer avec la demande. Ce paramètre est facultatif pour les requêtes GET.

Exemple

var xdr = new XDomainRequest();
xdr.open("POST", "http://example.com/api/method");
xdr.onload = function(){
  //Prendre en charge la réponse xdr.responseText
}
xdr.send("param1=value1&param2=value2");

Spécifications

Ne fait partie d'aucune spécification.
