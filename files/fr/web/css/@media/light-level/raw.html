{{CSSRef}}

light-level est une caractéristique média CSS (cf. {{cssxref("@media")}}) qui permet de vérifier le niveau de luminosité ambiante.

Syntaxe

dim
L'appareil est utilisé dans un environnement où la luminosité est faible et où un contraste excessif ou une luminosité trop importante serait gênante pour le lecteur. Cela correspond par exemple à une lecture de nuit ou à un environnement intérieur tamisé.
normal
L'appareil est utilisé dans un environnement où le niveau de lumière est idéal pour le rendu de l'écran, aucun ajustement n'est nécessaire.
washed
L'appareil est utilisé dans un environnement très lumineux ce qui rend l'écran difficile à lire. Cela correspond à une utilisation extérieure avec un soleil fort.

Exemples

Attention : Selon la compatibilité des appareils actuels, il est possible que ce fragment de code ne fonctionne pas sur certains.

@media (light-level: normal) {
  p {
    background: url("texture.jpg");
    color: #333;
  }
}

@media (light-level: dim) {
  p {
    background: #222;
    color: #ccc;
  }
}

@media (light-level: washed) {
  p {
    background: white;
    color: black;
    font-size: 2em;
  }
}
