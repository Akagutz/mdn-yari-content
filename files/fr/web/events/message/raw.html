Un évènement message informe la cible, un objet {{domxref("WebSocket")}}, {{domxref("RTCDataChannel")}}, {{domxref("EventSource")}}, ou {{domxref("BroadcastChannel")}}, qu'un message a été reçu.

Si la cible est un {{domxref("RTCDataChannel")}}, un event handler pour cet évènement peut être ajouté via la propiété {{domxref("RTCDataChannel.onmessage")}}.

Si la cible est un {{domxref("BroadcastChannel")}}, un event handler pour cet évènement peut être ajouté via la propriété {{domxref("BroadcastChannel.onmessage")}}.
