Un évènement  open informe le destinataire que la connection a été établie. Cet évènement est envoyé à l'objet {{domxref("RTCDataChannel")}}, quand la connexion est ouverte et à  l'objet {{domxref("EventSource")}} quand la connexion server-sent events est opérationnelle.

An event handler for this event can be added via the {{domxref("RTCDataChannel.onopen")}} property or {{domxref("EventSource.onopen")}} as appropriate..
