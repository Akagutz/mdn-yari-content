La méthode move() peut être utilisée pour copier une séquence du tableau à une position donnée, définie par le paramètre cible. Cette méthode, non-standard, a été remplacée par la méthode {{jsxref("TypedArray.prototype.copyWithin()")}}. TypedArray correspond à un des types de tableaux typés.

Syntaxe

typedarray.move(début, fin, cible)

Paramètres

début
La position à partir de laquelle considérer les éléments à copier.
fin
La position jusqu'à laquelle considérer les éléments à copier.
cible
La position à partir de laquelle on copiera les éléments.

Valeur de retour

Le tableau typé modifié.

Exemples

var buffer = new ArrayBuffer(8);
var uint8 = new Uint8Array(buffer);
uint8.set([1,2,3]);
console.log(uint8); // Uint8Array [ 1, 2, 3, 0, 0, 0, 0, 0 ]
uint8.move(0,3,3);
console.log(uint8); // Uint8Array [ 1, 2, 3, 1, 2, 3, 0, 0 ]

Spécifications

Cette méthode ne fait partie d'aucun standard. Elle a été remplacée par la méthode {{jsxref("TypedArray.prototype.copyWithin()")}}.
