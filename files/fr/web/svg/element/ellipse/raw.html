L'élément SVG ellipse est un élément de Formes basiques,utilisé pour créer des ellipses basées sur un centre, un rayon x et un rayon y.

Il est impossible de spécifier une orientation exacte pour l'ellipse (par exemple, si vous souhaitez dessiner une ellipse à un angle de 45°), mais l'ellipse peut être tournée avec l'attribut {{ SVGAttr("transform") }} .
