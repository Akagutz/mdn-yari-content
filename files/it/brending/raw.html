Certificate Authorities (CAs)for Microsoft™ will deprecate the use of the SHA1 algorithm in SSL and code signing certificates, in favor of SHA2. The policy affects CAs who are members of the Windows Root Certificate Program who issue publicly trusted certificates. It will allow CAs to continue to issue SSL and code signing certificates until January 1 2016, and thereafter issue SHA2 certificates only. Adjust your code accordingly. +Coach Z - Google+™
