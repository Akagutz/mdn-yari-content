Per poter effettuare modifiche ai contenuti di MDN (ad esempio modificare/creare pagine o cambiare i tag) devi essere autenticato. Questo articolo ti spiegherà il processo di autenticazione e tutti i metodi supporti per identificarti come un MDN contributor.

Attualmente é possibile effetuare l'autenticazione su MDN solamente tramite GitHub. Se non possiedi un account GitHub collegato al tuo account MDN, puoi leggere How to link a GitHub account to your MDN profile per capire come aggiornare il tuo account.

Autenticarsi

MDN utilizza servizi di terze parti per permettere l'autenticazione dei contributor. Se non sei autenticato, vedrai in alto nella pagina il box "Accedi", con un'icona per ogni servizio di autenticazione supportato.

Premi il bottone "Accedi" per iniziare il processo di autenticazione. Se non possiedi un account Github, dovrai crearne uno. Puoi leggere How to create an MDN account e How to link a GitHub account to your MDN profile per maggiori dettagli in caso di necessità.

Servizi di autenticazione

Malgrado MDN sia pensato per supportare svariati medoti di autenticazione, al momento l'unico supportato é GitHub.

Se il tuo account MDN non ha un account GitHub collegato, dovrai collegarne uno per poter creare/modificare contenuti su MDN. Vedi How to link a GitHub account to your MDN profile per maggiori dettagli.
