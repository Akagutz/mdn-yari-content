Plugin per WordPress

È possibilie utilizzare il Plugin di promozione MDN per i blog WordPress per aiutare le persone che scrivono blog sulle tecnologie web o su prodotti legati a Mozilla a promuovere sia il loro stesso blog che MDN stesso. Funziona in due modi:

  1. Quando pubblichi un articolo sul tuo blog, collega automaticamente varie parole chiave (ad esempio "HTML", "JavaScript", e "CSS") alla pagina appropriata su MDN. Questo ti permette di scrivere articoli sulle tecnologie web e avere collegamenti automatici alla documentazione rilevante senza dover fare ulteriori passi. È incredibilmente utile! I termini e le loro pagine di destinazione sono configurabili usando una pagina speciale qui su MDN.
  2. Una speciale casella "Notifica questo articolo a Mozilla" viene aggiunto alla pagina di modifica degli articoli; questa opzione permette di inviare una mail ai team PR e MDN di Mozilla così da permetterci, se appropriato, di menzionare il tuo articolo su materiale promozionale, inviare tweet su di esso e via dicendo. Spuntare questa casella non garantisce che verrai menzionato, ma sicuramente richiamerà l'attenzione delle persone giuste!

L'installazione di questo plugin è utile sia a te che a MDN, quindi se non l'hai ancora installato dovresti farlo ora! Se vuoi aiutare lo sviluppo del plugin visita il repository GitHub wp-promote-mdn.

Frammenti JavaScript

Il modo consigliato per aggiungere collegamenti alle pagine di MDN tramite le parole chiave è il plugin di Wordpress Promote MDN suggerito prima ma, nel caso il tuo sito/blog non sia un blog Wordpress, puoi inserire nel tuo sito il codice che trovi qui sotto e che dovrebbe funzionare con qualsiasi sito web.

Questo codice fa esattamente la stessa cosa del plugin Wordpress, ovvero converte le parole chiave presenti nel contenuto del tuo sito in collegamenti alla documentazione appropriata su MDN. Puoi configurare il numero massimo di parole chiave che vuoi convertire in collegamenti per pagina. Il numero predefinito è 3. Sentiti libero di modificarlo come ritieni appropriato!

Tutto quello che devi fare è inserire il codice seguente all'interno della pagina web, appena prima del tag di chiusura </body>

<script>
    (function (d,f,a) {
        // Inserisci i tuoi settaggi qui
        var o = {maxLinks: 4, searchElements: ['div', 'h'], linkClass: 'link-to-mdn', extraLinks: {'keywordx': 'http://example.com'}};

        var s=d.createElement("script");s.type="text/javascript";if(s.f)
        {s.a=function(){if(s.f=="loaded"||s.f=="complete")
        {s.a=null;PromoteMDN(o)}}}else{s.onload=function()
        {PromoteMDN(o)}}s.src="https://raw.githubusercontent.com/riverspirit/promote-mdn-script/master/promote-mdn.js";
        d.getElementsByTagName("head")[0].appendChild(s)
    })(document,'readyState','onreadystatechange');
</script>

Se per qualche ragione volessi ospitare lo script sul tuo server, puoi scaricarlo da qui, caricarlo sul tuo sito e modificare il collegamento nel codice qui sopra.
