{{APIRef('Shadow DOM')}}

L'interfaccia ShadowRoot del DOM API Shadow  è il nodo padre di un sotto albero del DOM che viene renderizzato separatamente dall'albero principale del DOM.

Properties

{{domxref("ShadowRoot.mode")}} {{readonlyinline}}
The mode of the ShadowRoot: either "open" or "closed".
Setting the mode of a shadow root to "closed" makes the shadow root’s implementation internals inaccessible and unchangeable from JavaScript—in the same way the implementation internals of, for example, the {{HTMLElement("video")}} element are inaccessible and unchangeable from JavaScript.
{{domxref("ShadowRoot.host")}} {{readonlyinline}}
A DOM element to which the ShadowRoot is attached.
{{domxref("ShadowRoot.innerHTML")}}
The DOM tree inside the ShadowRoot.
