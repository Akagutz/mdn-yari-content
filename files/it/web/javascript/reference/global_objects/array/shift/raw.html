Il metodo shift() rimuove il primo elemento da un array e ritorna l'elemento rimosso. Questo metodo cambia la lunghezza dell'array.

Sintassi

arr.shift()

Valore di ritorno

L'elemento rimosso dall'array; {{jsxref("undefined")}} se l'array è vuoto.

Descrizione

Il metodo shift rimuove l'elemento che si trova all'index zero e sposta verso il basso i valori degli indici consecutivi, quindi ritorna l'elemento rimosso. Se la proprietà {{jsxref("Array.length", "length")}} dell'array è uguale a 0 il valore di ritorno è {{jsxref("undefined")}}

shift è intenzionalmente generico; questo metodo può essere chiamato con {{jsxref("Function.call", "call", "", 1)}} o {{jsxref("Function.apply", "apply", "", 1)}} to objects resembling arrays. Objects which do not contain a length property reflecting the last in a series of consecutive, zero-based numerical properties may not behave in any meaningful manner.

Examples

Removing an element from an array

The following code displays the myFish array before and after removing its first element. It also displays the removed element:

var myFish = ['angel', 'clown', 'mandarin', 'surgeon'];

console.log('myFish before:', JSON.stringify(myFish));
// myFish before: ['angel', 'clown', 'mandarin', 'surgeon']

var shifted = myFish.shift(); 

console.log('myFish after:', myFish); 
// myFish after: ['clown', 'mandarin', 'surgeon']

console.log('Removed this element:', shifted); 
// Removed this element: angel

Using shift() method in while loop

The shift() method is often used in condition inside while loop. In the following example every iteration will remove the next element from an array, until it is empty:

var names = ["Andrew", "Edward", "Paul", "Chris" ,"John"];

while( (i = names.shift()) !== undefined ) {
    console.log(i);
}
// Andrew, Edward, Paul, Chris, John
