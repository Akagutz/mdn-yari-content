{{Non-standard_header}}

概要

ハイパーテキストターゲットとして利用される HTML アンカー を生成します。

String のメソッド
実装されたバージョン JavaScript 1.0
ECMAScript エディション 定義されていない

構文

string.anchor(nameAttribute)

引数

nameAttribute
文字列

詳細

文書中にアンカーをプログラム的に生成し、表示するためには、document.write 、あるいは、 document.writeln メソッドと共に、anchor メソッドを使用してください。anchor メソッドでアンカーを生成してから、write 、あるいは、writeln メソッドを呼び出して、文書中にアンカーを表示してください。サーバーサイド JavaScript では、write 関数を使って、アンカーを表示してください。

構文中で、テキストの文字列は、ユーザーに見てもらいたい リテラルテキストを表します。nameAttribute に指定された文字列は、A タグの NAME 属性を表します。

anchor メソッドで生成されたアンカーは、document.anchors の配列の要素となります。

例

例: anchor メソッドの使用

以下の例のコードは、HTML の script 要素内に書かれたものです。:

var myString = "目次";

document.writeln( myString.anchor("TOC") );

以下の HTML が生成されるでしょう。:

<A NAME="TOC">目次</A>
