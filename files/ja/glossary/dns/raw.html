DNS (Domain Name System) は、インターネットに接続されたリソースに対する階層的で分散化された名前付けシステムで、{{glossary("domain name","ドメイン名")}}と、関連リソース、例えば IP アドレスとの組み合わせのリストを管理するものです。

DNS の最も主流の機能は、人に優しいドメイン名 (例えば mozilla.org) を数値の {{Glossary("IP address")}} (例えば 151.106.5.172)に変換することです; このようにドメイン名を適切な IP アドレスに変換する処理は DNS ルックアップとして知られて言います。それと逆に、DNS逆引き (rDNS) IP アドレスに関連付けられたドメイン名を決定するのに使われます。
