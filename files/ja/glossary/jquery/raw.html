jQuery は {{Glossary("JavaScript")}} {{Glossary("Library", "ライブラリー")}} であり、{{Glossary("DOM")}} 操作を簡単にすること、{{Glossary("AJAX")}} 呼び出し、{{Glossary("Event", "イベント")}} ハンドリングに注力しています。JavaScript 開発者からよく使われます。

jQuery は $(selector).action() というフォーマットを使って要素をイベントに割り当てます。詳しく説明すると、$(selector) が jQuery を呼び出して selector 要素を選択し、  .action()というイベント {{Glossary("API")}} に割り当てます。

$(document).ready(function(){
  alert("Hello World!");
  $("#blackBox").hide();
});

上のコードは下のコードと同じ関数を実行します:

window.onload = function() {
  alert( "Hello World!" );
  document.getElementById("blackBox").style.display = "none";
};
