{{Glossary("CSS")}} (Cascading Style Sheets) の文脈において、用語 media は {{Glossary("rendering engine", "レンダリングエンジン")}} によって描画されるドキュメントの出力先を参照します。 典型的には、それはデバイスの画面ですが、プリンターや読み上げ装置、点字ディスプレイ、あるいはそのほかのデバイスの可能性もあります。

CSS はコンテンツの提示方法を変える様々な機能をサポートしており、それはレンダリングされている先の出力媒体の種類、もしくは幅や解像度、そのほかの値といった、出力媒体の個々のプロパティに基づいています。

技術的リファレンス

  • メディアクエリー: スタイル適用時に満たすべき、デバイスの特徴や引数を説明しています。
  • {{cssxref("@media")}} @-規則: メディアクエリーの結果に基づき、スタイルシートの一部を条件付きで適用します。
  • {{domxref("Window.matchMedia()")}}: メディアクエリーに関してウィンドウを検証します。
