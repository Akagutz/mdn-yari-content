MDN を愛していますか？オープン Web の素晴らしさを仲間の Web 開発者たちと共有しましょう！壁紙もステッカーもポスターもアニメーションも Web サイトのバッジも、何でもここからリンクを貼ってください。

ソーシャルメディアでリンクをシェアする

MDN の特に役立つ記事を見つけましたか？Twitter, Facebook, Hacker News, reddit, やお好みのすごいものでシェアしてください。@MozDevNet まで (英語で) お知らせ頂くと良いですが、必須ではありません。

MDN リンクで質問に答える

質問が Stack Overflow にあってもなくても、Twitterverse で叫んだりする人もいたり、共同作業者がチームの Slack チャンネルで聞いてきたり、しばしば ウェブ開発の質問への答えは MDN にあります。その質問に答える MDN のリンクを共有することで、ウェブ開発のベストな情報が見つかるよう、質問者を助けてあげましょう。

技術トークで MDN を引用する

地域の JavaScript ミートアップグループでしゃべっても大きな技術カンファレンスでしゃべる場合でも、しばしば MDN には話題を補完する記事があったり、聞き手にさらなる背景知識を与えられます。プレゼンテーションがオンラインで利用できる場合、リンクつきで @MozDevNet に伝えていただけるとすばらしいです。しかし最も大事なことは、必要とする人にすばらしい情報を共有することです。

WordPress プラグイン

MDN 宣伝のための WordPress ブログプラグイン が Web 技術と Mozilla 関連製品についてのブログ記事の執筆者のために用意されています。ブログの宣伝にも MDN の宣伝にもなるでしょう。次のやり方があります。

ブログに記事を投稿するとき、キーワード（"HTML" や "JavaScript"、"CSS" など）が自動的に MDN の記事にリンクされます。つまり、あなたが技術的な内容の記事を書くと、自分では何もしていなくても自動的に MDN のドキュメンテーションにリンクされるということです。便利すぎて信じられない！キーワードとどの記事がリンクされるかは Github リポジトリの https://github.com/mdn/wp-promote-mdn/tree/def-list の決定リストで調整されています。

このプラグインの導入は MDN にとってもあなたにとっても役に立つでしょう。まだインストールしていないなら、ぜひ導入してください。プラグインのコード作成を手伝いたいなら、wp-promote-mdn GitHub repo を参照しましょう。

JavaScript スニペット

MDN ページに自動的にリンクを生成するためには上記のやり方が望ましいですが、ブログが WordPress で動作していない場合は、以下の JavaScript コードを埋め込むことで同様のことが可能です。WordPress プラグインとやることは同じです。キーワードを MDN のドキュメンテーションにリンクします。

リンクするキーワードの上限を設定することが出来ます。オプションオブジェクト o の maxLinks の値を変更しましょう。検索する要素と作成されるリンクのクラスの指定、キーワードの追加、スキャンを実行する URL の指定もすることが出来ます。

このスニペットを使うためには、記事の末尾の </body> タグの直前に埋め込んでください。

<script>
    (function (d,f,a) {
        // Your settings here
        var o = {maxLinks: 4, searchElements: ['div', 'h'], linkClass: 'link-to-mdn', extraLinks: {'keywordx': 'http://example.com'}};

        var s=d.createElement("script");s.type="text/javascript";if(s.f)
        {s.a=function(){if(s.f=="loaded"||s.f=="complete")
        {s.a=null;PromoteMDN(o)}}}else{s.onload=function()
        {PromoteMDN(o)}}s.src="https://raw.githubusercontent.com/riverspirit/promote-mdn-script/master/promote-mdn.js";
        d.getElementsByTagName("head")[0].appendChild(s)
    })(document,'readyState','onreadystatechange');
</script>

このスニペットは Github から実働部分のコードを読み出します。このコードを（セキュリティの為やパフォーマンスなどの為に）自分のサーバー上に置きたい場合は、Github からコードをダウンロード して自分のサイトに追加してください。そして９行目の URL を適切に変更してください。
