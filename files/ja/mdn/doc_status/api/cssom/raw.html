タグ付け標準

  • グループ化用のタグは以下の通りです。
    • "{{Tag("CSSOM View")}}" は、 {{SpecName("CSSOM View")}} および {{SpecName("Screen Orientation")}} で定義されているインターフェイス、メソッド、プロパティに使用してください。
    • "{{Tag("CSSOM")}}" は、その他の関連する仕様書で定義されているインターフェイス、メソッド、プロパティに使用してください。
    • "{{Tag("CSS Typed Object Model API")}}" は、 {{SpecName("CSS Typed OM")}} で定義されているインターフェイス、メソッド、プロパティに使用してください。

その他の行うべきタスク

  • {{domxref("HTMLElement.style")}} は {{domxref("ElementCSSInlineStyle.style")}} に移動する必要があります。これは {{domxref("HTMLElement")}} および {{domxref("SVGElement")}} から使用されています。もちろん {{domxref("ElementCSSInlineStyle")}} インターフェイスを作成しなければなりません。
  • {{domxref("Window.screen")}} は実際の Window.screen と {{domxref("Screen")}} とに分割する必要があります。
  • {{domxref("Screen")}} にはいくつか (5-6) の標準外のメソッドやプロパティがあり、正しいタグが付いていません。 (現時点では {{domxref("Window.screen")}} の下にありますが、間違って DOM タグが付けられているようです)。
  • いくつかのページにはここで experimental とタグ付けする必要があります。
