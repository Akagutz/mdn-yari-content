タグ付け標準

  • すべてのページには {{Tag("API")}} および {{Tag("Reference")}} をタグ付けしてください。
  • Web Authentication API にリンクするページには、すべて {{Tag("WebAuthn")}} および {{Tag("WebAuthentication API")}} の両方をタグ付けしてください。
  • あるインターフェイスに所属するメソッドやプロパティについては、インターフェイスの名前 (例えば {{domxref("PublicKeyCredential.rawId")}} には {{Tag("PublicKeyCredential")}}) を追加してください。
  • メソッドには "{{Tag("Method")}}" を、プロパティには "{{Tag("Property")}}" タグを、インターフェイスには {{Tag("Interface")}} タグを、コンストラクターには {{Tag("Constructor")}} タグをつけてください。
  • ページに何らかの作業が必要であれば、 "Needs*" タグを追加してください。具体的には、例が存在しなければ、 {{Tag("NeedsExample")}} タグを追加してください。
