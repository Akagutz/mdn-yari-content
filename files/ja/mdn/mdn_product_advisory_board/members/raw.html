MDN プロダクトアドバイザリーボードの現在のメンバーは以下のとおりです。

Daniel Appelquist
Samsung Internet の Developer Advocacy ディレクター、W3C の Technical Architecture Group の共同議長

Daniel Appelquist は、Samsung のモバイルおよび VR Web ブラウザである Samsung Internet <https://samsunginter.net> の Web Developer Advocacy チームを率いています。2013年から W3C の Technical Architecture Group の共同議長も務めています。彼は20年以上にわたって Web に取り組んできました。最初は Web 出版の分野で新興企業を創設し、その後 (99年にニューヨークからロンドンに引っ越した後)、さまざまなプロジェクトとイニシアチブを通じて Web とモバイルの融合に取り組んできました。また彼は、英国政府のオープンスタンダード戦略を指揮し、英国政府オープンスタンダード委員会のメンバーであり続けています。この間、彼は活発なコミュニティビルダーおよびイベントオーガナイザーを務めています。

Jory Burson
COO, Bocoup

Jory は、いくつかの業界委員会および Bocoup の標準設定団体のメンバーとして、オープンソースおよびオープンスタンダードのコミュニティにおけるコラボレーションの向上に取り組んでいます。彼女は Ecma International の Executive Committee、JS Foundation Technical Architecture Committee、W3C Advisory Council、MDN Board のメンバーであり、Ecma TC39、Ecma TC53、および Web Platform Tests のプロジェクトに取り組んでいます。暇な時に、Jory は標準の歴史と Web プラットフォームの社会的構築について研究し、書いて、話します。

Dominique Hazael-Massieux
W3C Web 技術エキスパート、電気通信分野のバーティカルチャンピオン、Web リアルタイムコミュニケーションワーキンググループ、デバイスおよびセンサーワーキンググループ。

Dominique Hazael-Massieux は W3C スタッフの一員で、開発者との関係における W3C の取り組みを率いています。Dom は2000年から W3C に取り組んでいて、devrel に加えて、現在 WebRTC、デバイス API、および WebVR の標準化に携わっています。

Meggin Kearney
Google の Web Developer Relations のテクニカルライター

Meggin は、MDN の Web API ドキュメントを含む、Web 開発者の成功に役立つドキュメントを作成するチームを率いています。誇り高い Chromium コミッターでもある彼女は、Chrome 拡張機能 doc サーバーも共同開発しました。 Meggin はカリフォルニア大学バークレー校で学士号を取得しました。彼女はサンフランシスコに住んでおり、多様性をサポートし、Web 開発者コミュニティに参加する方法を常に模索しています。

Patrick Kettner
Microsoft

Patrick は、相互運用性と新しい Web テクノロジに重点を置いた、マイクロソフトのプログラムマネージャです。業界の Web 開発者である彼は、すべての主要なブラウザに取り組んでおり、Web 開発者の生活をより簡単で楽しいものにするために毎日取り組んでいます。仕事以外では、Modernizr のメンテナンスなど、数多くのオープンソースプロジェクトにも貢献しています。彼は、素晴らしいパートナー、カトリーナ、息子ホールデン、そして犬のバクスターと共にシアトルに住んでいます。

Chris Mills
Mozilla の MDN Web ドキュメントのコンテンツリーダーおよびライターチームマネージャ。

Chris は、MDN のコンテンツリーダーとして Mozilla に勤務しており、短期および長期の文書化が必要な場合の戦略をまとめるのに役立ちます。彼はまた、DOM API、HTML と CSS の機能、Web ゲーム、WebAssembly などをカバーする多数の初心者向けチュートリアルと参考記事を寄稿しています。彼はまた MDN ライターチームを管理します。

Erika Doyle Navara
Microsoft の Windows Web ドキュメントのシニア開発者ライター

Erika Doyle Navara は、IE7 の開発を開始した2006年以来、Microsoft のブラウザエンジンルームを取り巻いてきました。テストエンジニア兼テクニカルライターとして、W3C で CSS 2.1、CSS 3、SVG、および HTML 5 のテストスイートに対するマイクロソフトの貢献とブラウザデータの整理を支援しました。彼女は、MDN Web ドキュメントでクロスブラウザの相互運用性を引き続き推進していくことに興奮しています。

Robert Nyman
Google のプログラムとイニシアチブ、Web Developer Relations のグローバルリード。

Robert Nyman は、Google の Web Platform、開発者フィードバック＆コミュニティのグローバルリーダーです。彼の役割では、Web を開発者にとって最高のプラットフォームにするために働いています。Google に入社する前、Robert は Mozilla の技術的な伝道者でした。Open Web と同社のさまざまな製品や取り組みに焦点を当てていました。彼はストックホルムに住んでおり、旅行や人々との出会いに情熱を注いでいます。彼は42カ国で発表したLanyrdの「最もよく旅されたスピーカー」の称号を主張します。

Ali Spivak
Mozilla の開発者エコシステム責任者、ブロダクトアドバイザリーボードチェア。

Ali は Mozillaの開発者エコシステムの責任者で、以前は Mozilla の開発者マーケティングを担当していました。彼女は5年以上 MDN を管理しており、相互運用可能なクロスプラットフォームの Web に取り組んでいます。Mozilla 以前は、Cisco、Edmunds.com、および多数のスタートアップで Web 制作を管理していました。

Kadir Topal
Mozillaの MDN Web Docs プロダクトマネージャ。
