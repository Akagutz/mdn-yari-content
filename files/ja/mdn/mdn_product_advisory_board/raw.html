MDN Web Docs は、Wiki 技術に基づいたオープンソースの Web 開発ドキュメントプラットフォーム上に構築された Web 開発者向けの技術ドキュメントの信頼できるソースです。これにより、ほぼ誰でもコンテンツを作成および編集できます。

MDN プロダクトアドバイザリーボードに対する Mozilla のビジョンは、MDN コミュニティが最新のブラウザおよび Web 標準の最も重要な側面を文書化した最も包括的で完全かつ信頼できる参照として MDN を集合的に維持するのを助けるコラボレーションを構築することです。

プロダクトアドバイザリーボードは外部リーダーからのアドバイスを提供し、MDN がその使命の中で HTML、CSS、JavaScript、および Web API の公平でブラウザにとらわれないドキュメントを提供し、標準ベースの Web 開発のトップリファレンスになるのを支援します。
