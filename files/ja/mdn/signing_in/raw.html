MDN のコンテンツ (ページの編集、作成、追加など) を変更する場合は、どんな時でもサインインする必要があります。この記事では、MDN の投稿者としてあなたを識別するために、私たちがサポートしているサインインのプロセスと認証方法について説明します。

現在、MDN の認証とログインには GitHub のみをサポートしています。 MDN アカウントに GitHub ログインが添付されていない場合は、アカウントを更新する方法について GitHub アカウントを MDN プロファイルにリンクする方法を参照してください。

サインイン

MDNにコントリビュートするためにはサードパーティから提供される認証サービスを使用してログインします。もし現在ログインされていない場合、画面上部に現在サポートされている認証サービスのアイコンが表示された "Sign in" ボックスが出るかと思います。

Sign Inボタンを押すことでログインすることができます。もしGitHubアカウントにMDNアカウントが紐付けられていない場合は新規に作成されます。詳細は MDNアカウントを作成する方法 や GitHub アカウントと MDN プロファイルを紐付ける方法 を参照してください。

認証サービス

MDNは複数の認証方式をサポートするように設計されていますが、現在はGitHub ログインのみサポートされています。

もしGitHubのアカウントにMDNアカウントが登録されていない場合、MDNの記事を編集するにはMDNのアカウントを新規作成する必要があります。詳細についてはGitHub アカウントにMDN プロファイルを紐付ける方法 を参照してください。
