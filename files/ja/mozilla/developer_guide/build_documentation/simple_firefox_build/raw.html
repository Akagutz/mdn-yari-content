デスクトップ版 Firefox のビルド

Windows 版と macOS / Linux 版の Firefox はとても人気のある製品です。以下のリンクは自身のプラットフォームに合わせたビルドを始めるための手助けになります。

  • Windows 版 Firefox のビルド
  • macOS 版 Firefox のビルド
  • Linux 版 Firefox のビルド

モバイル版 Firefox

異なる目的、デバイス、ユーザーをケアするためにいくつかのモバイル製品があります。

  • Android 版 Firefox のビルド. (コードネーム：Fennec )
  • iOS 版 Firefox のビルド, デスクトップとシンクできる機能を持つ iOS 上の一般目的用途ブラウザーです。
  • Firefox Focus のビルド, iOS と Android 上のプライバシーに焦点をあてたブラウザーです。

デスクトップ版とモバイル版の両方とも、以下のことに注意してください。

  • 古いマシン上でも Firefox のビルドは可能ですが、より遅いマシンになれば極めてビルドに時間がかかります。少なくとも 8GB の RAM を推奨し、それ以上が良いでしょう。ビルドプロセスは CPU と I/O に高い負荷を与えます。そのため、SSD のマシンでビルドをすることを強く推奨します。
  • 早いインターネット環境も強く推奨します。開発環境とソースコードレポジトリは膨大です。
  • 32 ビットマシン上で Firefox を動作させることもできますが、ほとんどの製品では 64 ビット OS を動作推奨としています。
