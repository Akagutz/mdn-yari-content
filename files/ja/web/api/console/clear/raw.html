{{APIRef("Console API")}}{{Non-standard_header}}

コンソールをクリアします。

コンソールの内容が、"Console was cleared" といった情報メッセージに置き換えられます。

Google Chrome では、ユーザが Settings で "Preserve log upon navigation" を選択していると console.clear() は効果がありませんので注意してください。

構文

console.clear();
