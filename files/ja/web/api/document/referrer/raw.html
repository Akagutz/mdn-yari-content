Document.referrer プロパティは、このページへ「リンク」していたページの URI を返します。

構文

var referrer = document.referrer;

値

この値は、ユーザーが (リンクではなく、例えばブックマーク経由で) ページに直接移動した場合は空文字列になります。プロパティはただ文字列を返すものであり、参照先のページの DOM アクセスを提供するものではありません。

{{HTMLElement("iframe")}} の中では、 Document.referrer は最初、親ウィンドウの {{domxref("Window/location", "Window.location")}} の {{domxref("HTMLHyperlinkElementUtils/href", "href")}} と同じ値が設定されます。
