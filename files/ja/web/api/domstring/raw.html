DOMString は、UTF-16 文字列です。JavaScript では UTF-16 文字列を用いるため、DOMString は直接 {{jsxref("String")}} に対応します。

DOMString を受け付けるメソッドやパラメーターに null を渡すと、一般的に "null" として文字列化します。
