概要

hasAttributes は現在の要素が何らかの属性を持つか否かを示す真偽値を返します。

構文

result = targetNode.hasAttributes(attrString);
  • result : 真偽値 (true / false)

例

var t1 = document.getElementById("table-data"); 

if ( t1.hasAttributes() ) { 
  // t1 が属性を持つ場合の処理をここに記述
}
