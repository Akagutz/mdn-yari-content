removeAttribute は指定された要素から属性を削除します。

構文

element.removeAttribute(attrName)
  • attrName は要素から削除する属性の名前を表す文字列です。

例

// <div id="div1" align="left" width="200px"> 
document.getElementById("div1").removeAttribute("align"); 
// now: <div id="div1" width="200px">

注記

setAttribute を使って属性の値に null をセットする代わりに removeAttribute を使用すべきです。

要素にない属性を取り除こうと試みても例外は発生しません。

removeAttribute は返り値がないため、このメソッドをつなげて複数の属性を削除することはできません。
