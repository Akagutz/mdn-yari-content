Summary

イベントが浮上(bubble up)するものかどうかを示しています。

Syntax

bool = event.bubbles

bool はtrue か falseであり、イベントが浮上(bubble up)するものかどうかを示しています。

Notes

浮上(bubble up)するものは特定のイベントだけです。浮上するイベントはこのプロパティが true になっています。なので、これを調べることにより、そのイベントが浮上するものかどうかを知ることができます。

Example

 function goInput(e) {
  // checks bubbles and
  if not e.bubbles {
     // passes event along if it's not
     passItOn(e); 
  }
  // already bubbling
  doOutput(e)
}
