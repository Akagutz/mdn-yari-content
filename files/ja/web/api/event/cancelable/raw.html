概要

イベントがキャンセル可能かどうかを示す真偽値を返します。

構文

bool = event.cancelable
  • bool : 真偽値。イベントがキャンセル可能な場合は true となる。

例

var bool = event.cancelable;

注記

イベントがキャンセル可能かどうかは、そのイベントを初期化する際に判別されます。

イベントをキャンセルするには、そのイベントに対し {{domxref("event.preventDefault", "preventDefault()")}} メソッドを呼び出します。これは、実装がそのイベントに関連付けられた規定のアクションを実行しないようにします。
