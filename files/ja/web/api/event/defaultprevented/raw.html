Summary

イベントに対して {{ domxref("event.preventDefault()") }} が呼ばれているかを、 boolean で返します。

Note: getPreventDefault() は非標準かつ非推奨であるため、代わりにこの API を使うべきです。 (参照 {{ bug(691151) }}).

Syntax

bool = event.defaultPrevented

Example

 if (e.defaultPrevented) {
   /* the default was prevented */
 }
