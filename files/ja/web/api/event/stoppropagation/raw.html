概要

現在のイベントのさらなる伝播 (propagation) を止めます。

構文

event.stopPropagation();

例

この DOM のメソッドとイベントの伝播の詳しい例については、『 event.stopPropagation の例 - 例 5 : イベントの伝播』でご覧いただけます。

注意

イベントのフローに関する説明については DOM Level 2 仕様書 を参照してください。
(DOM Level 3 Events 草案 にはイラスト付きの説明があります)

preventDefault は、イベントの規定のアクションが発生するのを止めるのに使われる補完的なメソッドです。
