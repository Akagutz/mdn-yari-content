back メソッドは、セッションヒストリーを１つ前に移動します。１つ前のページが無い場合、このメソッドは何もしません。

Syntax

window.history.back()

Examples

下記の短い例では、ページ上のボタンを押したときにセッションヒストリーを１つ前のエントリーに戻します。

HTML

<button id="go-back">Go back!</button>

JavaScript

window.onload = function(e) {
  document.getElementById('go-back').addEventListener('click', e => {
    window.history.back();
  })
}
