HTMLBRElement インタフェースは HTML line break 要素 ({{htmlelement("br")}}) のインタフェースです。{{domxref("HTMLElement")}} の継承メソッドです。

プロパティ

{{domxref("HTMLElement")}} インタフェースから継承するものの他に、以下の固有プロパティを持ちます。

名称 型 説明
clear {{obsolete_inline}} {{domxref("DOMString")}} フローティングオブジェクトの周囲のテキストのフローを示します。

メソッド

固有のメソッドはありません。 {{domxref("HTMLElement")}} インタフェースのメソッドを継承しています。
