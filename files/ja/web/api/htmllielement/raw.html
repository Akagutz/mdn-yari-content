DOM の HTMLLIElement インタフェースは、リストアイテム要素 ( {{HTMLElement("li")}} ) を操作するための特定のプロパティを公開しています。
（リストアイテム要素は、このインターフェースの継承元である {{domxref("HTMLElement")}} オブジェクトインターフェースからの操作も可能となっています）

プロパティ

{{domxref("HTMLElement")}} から継承するものの他に、以下のプロパティを持ちます。

名称 型 説明
type {{obsolete_inline}} {{domxref("DOMString")}} ビュレットの種類 ("disc"、 "square"、 "circle")。現在の標準では CSS の {{cssxref("list-style-type")}} プロパティを用いるのが正しい方法です。スクリプトで設定する場合には CSSOM のメソッドを使用して下さい。
value long {{HTMLElement("ol")}} 内でのリストアイテム要素の位置を表す序数。これは {{HTMLElement("li")}} 要素の {{htmlattrxref("value", "li")}} 属性を反映し、負数も許容します。 {{HTMLElement("li")}} 要素が {{HTMLElement("ol")}} 要素の子要素ではない場合は、このプロパティは意味を成しません。

メソッド

{{domxref("HTMLElement")}} から継承するメソッドのみを持ちます。
