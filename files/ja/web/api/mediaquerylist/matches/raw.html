{{domxref("MediaQueryList")}} の matches 読み取り専用プロパティは、{{domxref("document")}} がメディアクエリーのリストに一致していれば true を返し、そうでなければ false を返す {{domxref("Boolean")}} 値です。

構文

var matches = MediaQueryList.matches;

値

{{domxref("Boolean")}}。{{domxref("document")}} が現在メディアクエリーのリストに一致していれば true を返し、そうでなければ false を返します。

例

var mql = window.matchMedia('(max-width: 600px)');

if(mql.matches) {
  // media query test returning true
}
