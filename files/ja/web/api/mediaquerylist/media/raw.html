{{domxref("MediaQueryList")}} インターフェイスの media 読み取り専用プロパティは、シリアライズされたメディアクエリーを表す {{domxref("DOMString")}} です。

構文

var media = MediaQueryList.media;

値

シリアライズされたメディアクエリーを表す {{domxref("DOMString")}}。

例

var mql = window.matchMedia('(max-width: 600px)');

console.log(mql.media);
