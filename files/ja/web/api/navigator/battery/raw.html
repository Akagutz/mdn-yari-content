{{ ApiRef("Battery API") }}{{deprecated_header}}

読み取り専用の battery プロパティは、システムの充電量についての情報を提供する {{domxref("BatteryManager")}} を返します。これは、充電状態の監視を扱ういくつかの新しいイベントも提供します。これは、Battery Status API を実装します。詳細な追加情報や API の使い方、サンプルコードについては、この Battery Status API のドキュメントを参照してください。

battery プロパティは非推奨であり、battery の {{domxref("Promise")}} を返す {{domxref("Navigator.getBattery()")}} に置き換わりました。このサポートは部分的です。

構文

var battery = navigator.battery;

navigator.battery は {{domxref("BatteryManager")}} オブジェクトです。
