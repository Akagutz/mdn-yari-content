{{ ApiRef() }} {{ 英語版章題("Summary") }}

概要

ブラウザでクッキーが有効かどうかを示す真偽値を返します（読み込み専用）。

{{ 英語版章題("Syntax") }}

構文

var cookieEnabled = navigator.cookieEnabled;
  • cookieEnabled は、true、または、false のいずれかである真偽値です。

{{ 英語版章題("Example") }}

例

if (!navigator.cookieEnabled) {
  // クッキーを有効にするとウェブページをより便利に利用できることをユーザに知らせる処理
}

{{ 英語版章題("Notes") }}

注記

{{ Bug(230350) }} のために、cookieEnabled は、Gecko ベースのアプリケーションで誤った結果を返す可能性があります。
