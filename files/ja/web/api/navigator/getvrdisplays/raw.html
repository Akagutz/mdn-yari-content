{{domxref("Navigator")}} インターフェイスの getVRDisplays() メソッドは，コンピュータに接続されている利用可能なVRデバイスを表す {{domxref("VRDisplay")}} オブジェクトの配列へ解決するためのpromiseを返します．

シンタックス

navigator.getVRDisplays().then(function(displays) {
  // Do something with the available VR displays
});

パラメータ

なし．

戻り値

{{domxref("VRDisplay")}} オブジェクトの配列へ解決するためのpromise．

例

WebVR API を用いて構築されたアプリは，完全なデバイスの利用可能機能へとアクセスするために，返されたリストを繰返し処理して， {{domxref("VRDisplay.displayId")}} を共有するディスプレイを相互に関連付けしなければなりません．
