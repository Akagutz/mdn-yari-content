概要

ふたつのノードが同一ノードの場合は true を、異なる場合は false を返します。

構文

var isEqualNode = node.isEqualNode(arg);
  • arg : 比較ノード

例

targetElm を id として持つ要素が文書内の最初の div 要素であるかどうかをテストし、結果の真偽値を表示する例を以下に示します。

var targetElm = document.getElementById("targetElm");
var firstDiv = document.getElementsByTagName("div")[0];

alert( targetElm.isEqualNode(firstDiv) );
