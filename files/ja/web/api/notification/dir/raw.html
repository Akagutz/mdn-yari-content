{{APIRef("Web Notifications")}}

{{domxref("Notification")}}インターフェースのdir読み取り専用プロパティは、{{domxref("Notification.Notification","Notification()")}}コンストラクターのdirオプションで指定した、通知テキストの方向を表します。

{{AvailableInWorkers}}

構文

var direction = Notification.dir;

値

テキストの方向を指定する{{domxref("DOMString")}} です。以下の値が設定できます。

  • auto:  ブラウザの言語設定の振る舞いが適用されます（既定値）。
  • ltr: 左から右。
  • rtl : 右から左。

注: ほとんどのブラウザは、明確なltrとrtl設定を無視して、ブラウザの設定どおりに実行するようです。

例

次のスニペットは、通知を発火します。単純なoptionsオブジェクトを生成してから、Notification()コンストラクターを使用して通知を発火します。

var options = {
  body: 'Do you like my body?',
  dir: 'rtl'
}

var n = new Notification('Test notification',options);

n.dir // should return 'rtl'
