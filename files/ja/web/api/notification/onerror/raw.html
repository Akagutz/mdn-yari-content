要約

{{domxref("Notification")}}インターフェースのonerrorプロパティは、{{event("error")}} イベントを受け取るエラーイベントリスナを指定します。これらのイベントは、 {{domxref("Notification")}} に何か問題があるときに発生します。（多くの場合、エラーは通知の表示を妨げます）。

構文

Notification.onerror = function() { ... };
