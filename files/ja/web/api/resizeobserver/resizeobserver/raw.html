{{APIRef("Resize Observer API")}}{{SeeCompatTable}}

ResizeObserver コンストラクタは、{{domxref('Element')}} のコンテンツ矩形または {{domxref('SVGElement')}} のバウンディングボックスに対する変更を監視する新しい {{domxref("ResizeObserver")}} オブジェクトを作成します。

構文

var ResizeObserver = new ResizeObserver(callback)

パラメータ

callback
サイズ変更が発生するたびに呼び出されるメソッド。メソッドは {{domxref('ResizeObserverEntry')}} オブジェクトの配列で呼び出されます。

例

次の例では、サイズ変更時のテキストとして、その中のボックスの寸法を示しています。

const resizeObserver = new ResizeObserver(entries => {
  for (let entry of entries) {
    const boxEl = entry.target;
    const dimensions = entry.contentRect;
    boxEl.textContent = `${dimensions.width} x ${dimensions.height}`;
  }
});
resizeObserver.observe(document.querySelector('.box:nth-child(2)'));
