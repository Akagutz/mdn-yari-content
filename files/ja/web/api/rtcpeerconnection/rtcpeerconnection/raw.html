RTCPeerConnection() コンストラクターは{{domxref("RTCPeerConnection")}}を新たに作成し、返します。これは、ローカルデバイスとリモートピアの間の接続を表現します。

構文

pc = new RTCPeerConnection([configuration]);

引数

configuration {{optional_inline}}
An RTCConfiguration dictionary providing options to configure the new connection.

RTCConfiguration dictionary

{{page("/en-US/docs/Web/API/RTCConfiguration", "Properties")}}

返値

新たに作成された{{domxref("RTCPeerConnection")}}オブジェクトを返します。configurationが設定されていればそれに従って、設定されていなければデフォルトの値が適切に設定されます。
