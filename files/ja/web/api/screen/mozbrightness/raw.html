スクリーンのバックライドの明るさを、0 (とっても暗い) から 1 (明るさ最大) の尺度で指定します。小数点以下2桁の float で表します。

この属性は、スクリーンが無効になっていても読み書きできます。ただし、スクリーンが無効な時はバックライトもオフになっています。この属性に X という値を一度書き込んでも、あとで読み取った時には、同じ X という値ではないかもしれません。というのも、ほとんどのスクリーンでは、0 から 1 までの小数点以下2桁の精度の異なる明るさレベルをサポートしていないからです。値の精度は、値の保存前に削減される可能性もあります。

構文

screenBrightness = window.screen.mozBrightness;
