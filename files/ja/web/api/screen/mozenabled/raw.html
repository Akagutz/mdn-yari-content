デバイスのスクリーンを制御するブール値の属性です。false に設定すると、スクリーンをオフにします。

構文

screenEnabled = window.screen.mozEnabled

利用するには

screen.mozEnabled を使うには、about:config から "dom.screenEnabledProperty.enabled" を true に設定する必要があります。
