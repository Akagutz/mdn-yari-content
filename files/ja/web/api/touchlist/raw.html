TouchList はタッチ面の全ての接触ポイントのリストを表します。
（例えば、ユーザが画面（またはトラックパッド）を 3 本指で操作している場合、それぞれの指の接触点に対応する {{domxref("Touch")}} オブジェクトの合計で TouchList は 3 エントリとなります。）

属性

{{domxref("TouchList.length")}} {{readonlyInline}}
TouchList 中の {{domxref("Touch")}} オブジェクトの個数

メソッド

{{domxref("TouchList.identifiedTouch()")}}
識別子が指定された値と一致するリスト内の、最初の {{domxref("Touch")}} エントリを返す
{{domxref("TouchList.item()")}}
リスト内の指定したインデックス位置の {{domxref("Touch")}} オブジェクトを返す。配列構文を使用して TouchList 内の各エントリを参照する事も可能（ touchList[X] ）
