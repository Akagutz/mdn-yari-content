概要

水平スクロールバー（表示されている場合）を含む、ブラウザウィンドウの {{ 原語併記("ビューポート", "viewport") }} の高さを返します。

{{ 英語版章題("Syntax") }}

構文

var intViewportHeight = window.innerHeight;

{{ 英語版章題("Value") }}

戻り値

intViewportHeight は、window.innerHeight プロパティの値を保持します。

window.innerHeight プロパティは、読み込み専用です。また、デフォルトの値はありません。window.innerHeight プロパティは、ピクセル数を表す整数を保持します。

{{ 英語版章題("Notes") }}

注記

innerHeight プロパティは、ウィンドウ、フレーム、フレームセット、もしくは、副ウィンドウのような、あらゆるウィンドウオブジェクトでサポートされます。

{{ 英語版章題("Example") }}

例

フレームセットを考えてください
---------------------

var intFrameHeight = window.innerHeight; // または
var intFrameHeight = self.innerHeight; /* フレームセット内の フレームのビューポートの高さを返します */
var intFramesetHeight = parent.innerHeight; /* 最も近いフレームセットのビューポートの高さを返します。 */
var intOuterFramesetHeight = top.innerHeight; /* 最も離れたフレームセットのビューポートの高さを返します。 */
