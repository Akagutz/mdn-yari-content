概要

デバイスの環境光センサが光の強度の変化を検出した際に発生する {{event("devicelight")}} イベントを受け取るイベントリスナーを指定します。

構文

window.ondevicelight = funcRef
  • funcRef : {{event("devicelight")}} イベント発生時に呼び出す関数への参照。 このイベントは {{domxref("DeviceLightEvent")}} のひとつです。

Gecko 固有の注意事項

{{event("devicelight")}} イベントは、Android 用 Firefox Mobile (15.0) および Firefox OS (B2G) に実装されており、設定が有効になっています。Gecko 22.0 {{geckoRelease("22.0")}} 以降では、Mac OS X デスクトップでも有効です。現在 Windows 7 でのサポートが進められています (※参照 : {{bug("754199")}})
