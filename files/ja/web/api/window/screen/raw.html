{{ApiRef}}

概要

window オブジェクトに紐付く screen オブジェクトへの参照を返します。

screen オブジェクトは現在のウィンドウがレンダリングされている画面のプロパティを調べるための特別なオブジェクトです。

構文

screenObj = window.screen;

プロパティ

{{domxref("window.screen.availTop", "screen.availTop")}}
固定あるいは半固定されているユーザインタフェース部分は含まない、ウィンドウ内の最初のピクセルの y 座標
{{domxref("window.screen.availLeft", "screen.availLeft")}}
画面の左端から利用可能な最初の有効ピクセル
{{domxref("window.screen.availHeight", "screen.availHeight")}}
Windows でのタスクバーのような、オペレーティングシステムによって表示されている、固定あるいは半固定されているユーザインタフェース部分を引いた画面の高さ (ピクセル単位)
{{domxref("window.screen.availWidth", "screen.availWidth")}}
ウィンドウで利用可能である水平空間の大きさ (ピクセル単位)
{{domxref("window.screen.colorDepth", "screen.colorDepth")}}
画面の色深度
{{domxref("window.screen.height", "screen.height")}}
画面の高さ (ピクセル単位)
{{domxref("window.screen.left", "screen.left")}}
メイン画面の左端から現在の画面の左端までの距離 (ピクセル単位)
{{domxref("window.screen.orientation", "screen.orientation")}}
画面の現在の向きを取得
{{domxref("window.screen.pixelDepth", "screen.pixelDepth")}}
画面のビット深度を取得
{{domxref("window.screen.top", "screen.top")}}
現在の画面の上端からの距離 (ピクセル単位)
{{domxref("window.screen.width", "screen.width")}}
画面の幅
{{domxref("window.screen.mozEnabled", "screen.mozEnabled")}} {{gecko_minversion_inline("12.0")}}
真偽値。false に設定する事によりデバイス画面がオフになる。
{{domxref("window.screen.mozBrightness", "screen.mozBrightness")}} {{gecko_minversion_inline("12.0")}}
デバイス画面の 明るさ (brightness) を調整。0 から 1.0 までの double を設定可能。

イベントハンドラ

{{domxref("window.screen.onorientationchange", "screen.onorientationchange")}}
{{event("orientationchange")}} イベントに呼応するハンドラ

メソッド

{{domxref("window.screen.lockOrientation", "screen.lockOrientation")}}
画面の向き（オリエンテーション）をロック (※フルスクリーンもしくはインストール済みアプリケーションでのみ動作)
{{domxref("window.screen.unlockOrientation", "screen.unlockOrientation")}}
画面の向き（オリエンテーション）のロックを解除 (※フルスクリーンもしくはインストール済みアプリケーションでのみ動作)

if (screen.pixelDepth < 8) {
  // use low-color version of page
