概要

base64 形式でエンコードされたデータの文字列をデコードします。

ユニコードまたは UTF-8 文字列に対してこのメソッドを用いる場合の問題や解決方法については、『window.btoa - ユニコード文字列』または『Base64 のエンコードとデコード』の『ユニコードの問題』の章をご覧下さい。

構文

decodedData = window.atob(encodedData);

例

encodedData = window.btoa("Hello, world");  // 文字列のエンコード
decodedData = window.atob(encodedData);     // 文字列のデコード

注記

{{domxref("window.btoa()")}} メソッドは、そのまま送信すると問題を起こす可能性のあるデータをエンコードしてから送信する用途に用いる事が出来ます。送信したデータは、受信側で window.atob メソッドを使って元のデータにデコードすることができます。例えば、0 から 31 までの ASCII 値のような制御文字をエンコードすることができます。

{{domxref("window")}} は、コンポーネント内ではグローバルオブジェクトではありませんが、atob() を JavaScript で実装された XPCOM コンポーネントで利用する事もできます。
