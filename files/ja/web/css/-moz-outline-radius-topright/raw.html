Mozilla アプリケーションにおいて、 CSS の -moz-outline-radius-topright プロパティは、要素の輪郭線 ({{cssxref("outline")}}) の右上の角を丸くするために使用することができます。

{{cssinfo}}

例

HTML

<p>段落の右上の角を見てください。</p>

CSS

p {
  margin: 5px;
  border: solid cyan;
  outline: dotted red;
  -moz-outline-radius-topright: 2em;
}
