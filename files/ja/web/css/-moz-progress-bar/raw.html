{{CSSRef}}{{Non-standard_header}}

CSS の ::-moz-progress-bar 擬似要素は Mozilla 拡張で、 {{HTMLElement("progress")}} 要素の中のプログレスバーを表します。 (バーは進捗した量を表します。)

構文

{{csssyntax}}

例

HTML

<progress value="30" max="100">30%</progress>
<progress max="100">Indeterminate</progress>

CSS

::-moz-progress-bar {
  background-color: red;
}

/* 不確実なバーは強制的に幅をゼロにする */
:indeterminate::-moz-progress-bar {
  width: 0;
}
