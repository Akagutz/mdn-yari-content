{{CSSRef}}

CSS の font-family 記述子は、{{cssxref("@font-face")}} ルールで指定されたフォントのフォントファミリーを、作者が指定することを可能にします。

{{cssinfo}}

構文

/* <string> values */
font-family: "font family";
font-family: 'another font family';

/* <custom-ident> value */
font-family: examplefont;

値

<family-name>
フォントファミリーの名前を指定します。

形式文法

{{csssyntax}}

例

@font-face {
  font-family: examplefont;
  src: url('examplefont.ttf');
}
