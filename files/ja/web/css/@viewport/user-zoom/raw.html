{{ CSSRef }}

CSS の user-zoom 記述子は、 {{cssxref("@viewport")}} で定義された文書の拡大率をユーザーが変更できるかどうかを制御します。

/* キーワード値 */
user-zoom: zoom;
user-zoom: fixed;

{{cssinfo}}

構文

値

zoom
ユーザーはズームイン、またはズームアウトできる。
fixed
ユーザーはズームイン、またはズームアウトできない。

形式文法

{{csssyntax}}

 

アクセシビリティの考慮事項

ズーム機能を無効にすると、弱視の人がページコンテンツを読んだり理解したりすることを妨げます。

  • MDN "WCAG を理解する ― ガイドライン 1.4 の解説"
