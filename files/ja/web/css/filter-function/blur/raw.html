CSS の blur() 関数は、入力画像に Gaussian blur を適用します。返値は {{cssxref("<filter-function>")}} です。

構文

blur(radius)

引数

radius
ぼかしの半径で、 {{cssxref("<length>")}} で指定します。ガウス関数の標準偏差の値を定義します。つまり、画面上のピクセルをいくつ互いに混合するかということです。すなわち、大きい値になるほどよりぼかしが入ります。 0 の値は入力を変更しません。補間時に欠けた場合の値は 0 です。

例

blur(0)        /* 効果なし */
blur(8px)      /* 半径 8px でぼかす */
blur(1.17rem)  /* 半径 1.17rem でぼかす */
