CSS の sepia() 関数は、入力画像をセピア色に、より暖かい、より黄色と茶色の表現に変換します。結果は {{cssxref("<filter-function>")}} です。

構文

sepia(amount)

引数

amount
変換の度合いで、 {{cssxref("<number>")}} 又は {{cssxref("<percentage>")}} で指定します。 100% の値では完全にセピア色になり、 0% の値では入力が変更されないままになります。 0% と 100% の間の値は、効果の割合です。補間時の欠損値は 0 です。

例

sepia(0)     /* 効果なし */
sepia(.65)   /* 65% セピア色 */
sepia(100%)  /* 完全にセピア色 */
