 CSS の ID セレクター (ID selector) は、 {{htmlattrxref("ID")}} 属性の値に基づいて要素とマッチします。選択される要素の ID 属性は、セレクターで指定した値と完全一致していなければなりません。

/* The element with id="demo" */
#demo {
  border: red 2px solid;
}

構文

#ID値 { スタイルプロパティ }

なお、これは以下の{{Cssxref("Attribute_selectors", "属性セレクター")}}と等価です。

[id=ID値] { スタイルプロパティ }

例

CSS

#identified {
  background-color: skyblue;
}

HTML

<div id="identified">これは特別な ID がついています！</div>
<div>これは単なる普通の div です。</div>
