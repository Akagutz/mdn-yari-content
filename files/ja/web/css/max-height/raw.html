CSS の max-height プロパティは要素の最大の高さを設定するのに使われます。{{cssxref("height")}} プロパティの使用値は、max-height に設定した値よりも大きくなりません。

The source for this interactive example is stored in a GitHub repository. If you'd like to contribute to the interactive examples project, please clone https://github.com/mdn/interactive-examples and send us a pull request.

{{cssxref("max-height")}} は {{cssxref("height")}} を上書きしますが、{{cssxref("min-height")}} は {{cssxref("max-height")}} を上書きします。

構文

/* <length> 値 */
max-height: 3.5em;

/* <percentage> 値 */
max-height: 75%;

/* キーワード値 */
max-height: none;
max-height: max-content;
max-height: min-content;
max-height: fit-content;
max-height: fill-available;

/* グローバル値 */
max-height: inherit;
max-height: initial;
max-height: unset;

値

{{cssxref("<length>")}}
固定の、高さの最大値です。利用可能な単位については {{cssxref("<length>")}} をご覧ください
{{cssxref("<percentage>")}}
{{cssxref("<percentage>")}} は、包含ブロックの高さを基準に計算されます。包含ブロックの高さが明示的に定義されていないときは、パーセンテージ値はゼロとみなされます

キーワード値

none
ボックスの高さに制限を設けません
max-content{{experimental_inline()}}
コンテンツ本来の望ましい高さを確保します。
min-content{{experimental_inline()}}
コンテンツの本質を崩さない範囲で最小の高さを確保します。
fill-available{{experimental_inline()}}
包含ブロックの高さから垂直方向のマージン、ボーダー、パディングを引いたものです (一部のブラウザでは、このキーワードに古い名称、available を与えていることに注意してください)。
fit-content{{experimental_inline()}}
max-content と同じです。

形式的構文

{{csssyntax}}

例

table { max-height: 75%; }

form { max-height: none; }

アクセシビリティへの配慮

ユーザーが文字を大きくしようとしてページを拡大した時に、max-height を指定した要素が切り詰められたり、他の要素をおおい隠したりしないように注意してください。
