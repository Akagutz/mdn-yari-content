CSS の min-height プロパティは要素の最小の高さを設定するのに使われます。{{Cssxref("height")}} プロパティの 使用値 は、min-height に指定した値よりも小さくなりません。

The source for this interactive example is stored in a GitHub repository. If you'd like to contribute to the interactive examples project, please clone https://github.com/mdn/interactive-examples and send us a pull request.

min-height の値が {{Cssxref("max-height")}} および {{Cssxref("height")}} の値より大きい場合は、min-height の値が要素の幅になります。

構文

/* <length> 値 */
min-height: 3.5em;

/* <percentage> 値 */
min-height: 10%;

/* キーワード値 */
min-height: max-content;
min-height: min-content;
min-height: fit-content;
min-height: fill-available;

/* グローバル値 */
min-height: inherit;
min-height: initial;
min-height: unset;

値

{{cssxref("<length>")}}
高さの最小値を {{cssxref("<length>")}} で表します。負の値は宣言を無効にします。
{{cssxref("<percentage>")}}
最小の高さを、包含ブロックの幅に対する {{cssxref("<percentage>")}} で表します。負の値は宣言を無効にします。

キーワード値

auto
flex アイテム向けの既定の最小の高さであり、他のレイアウトにおける既定値 0 よりも合理的な既定値を提供します。
max-content {{experimental_inline}}
コンテンツにあわせた最適な高さ（The intrinsic preferred height）
min-content {{experimental_inline}}
コンテンツにあわせた最小の高さ（The intrinsic minimum height）
fill-available{{experimental_inline}}
包含ブロックの高さから水平方向のマージン、ボーダー、パディングを引いたもの。一部ブラウザーで、このキーワードに古い名前、available を与えています
fit-content {{experimental_inline}}
CSS3 Box では、これは min-content の同義語です。CSS3 Sizing ではもっと複雑なアルゴリズムを定義していますが、実験的な方法を含めても、実装しているブラウザーはありません。

形式文法

{{csssyntax}}

例

table { min-height: 75%; }

form { min-height: 0; }
