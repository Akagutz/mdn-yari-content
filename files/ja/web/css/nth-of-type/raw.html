{{CSSRef}}

CSS の :nth-of-type() 疑似クラスは、兄弟要素のグループの中で指定された型の要素を、位置に基づいて選択します。

/* 兄弟の <p> 要素の中で、
   3つおきに選択 */
p:nth-of-type(4n) {
  color: lime;
}

構文

nth-of-type 疑似クラスは、要素を選択する最後から数えるパターンを表す引数を1つ取ります。

構文の詳しい説明は {{Cssxref(":nth-child")}} を参照してください。

形式文法

{{csssyntax}}

例

基本的な例

HTML

<div>
  <div>This element isn't counted.</div>
  <p>1st paragraph.</p>
  <p>2nd paragraph.</p>
  <div>This element isn't counted.</div>
  <p>3rd paragraph.</p>
  <p>4th paragraph.</p>
</div>

CSS

/* 奇数の段落 */
p:nth-of-type(2n+1) {
  color: red;
}

/* 偶数の段落 */
p:nth-of-type(2n) {
  color: blue;
}

/* 最初の段落 */
p:nth-of-type(1) {
  font-weight: bold;
}
