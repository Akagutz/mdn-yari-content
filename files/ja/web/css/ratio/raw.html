概要

<ratio> CSS データ型は、メディアクエリ における画面比率の記述に使用され、単位無しの 2 値の比率で示します。この値は、厳密な正の {{cssxref("<integer>")}}、スラッシュ ('/', Unicode U+002F SOLIDUS)、2 番目の厳密な正の {{cssxref("<integer>")}} の順に記述します。スラッシュの前後にはスペースを入れてもかまいません。最初の数値は横幅を、2番目の数値は縦幅をあらわします。

例

  比率 用例
4/3 20 世紀における伝統的な TV 画面の形式です。
16/9 現代の 'ワイドスクリーン' の TV 画面の形式です。
185/100 = 91/50
(非整数値の配分や除算は許容されていません)		 1960 年代から用いられている最も一般的な映画スクリーンの形式です。
239/100
(非整数値の配分や除算は許容されていません)		 'ワイドスクリーン' のアナモルフィックな映画スクリーンの形式です。
