{{CSSRef}}

CSS の :root 疑似クラスは、文書を表すツリーのルート要素を選択します。 HTML では、 :root は {{HTMLElement("html")}} 要素を表し、詳細度が高いことを除けば html セレクターと同等です。

/* 文書のルート要素（HTML の場合は <html>）を
   選択編集できる <input> 要素を選択 */
:root {
  background: yellow;
}

構文

{{csssyntax}}

例

:root はグローバルの CSS 変数を宣言するのに便利です。

:root {
  --main-color: hotpink;
  --pane-padding: 5px 42px;
}
