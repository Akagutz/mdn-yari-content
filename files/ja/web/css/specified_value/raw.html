概要

CSS プロパティの指定値は、次の3つの方法のいずれかで設定されます。

  1. 文書のスタイルシートがプロパティに値を指定した場合は、その値が使用されます。例えば、{{ cssxref("color") }} プロパティに green を設定した場合、対応する要素のテキストの色は緑になります。
  2. 文書のスタイルシートが値を指定しなかった場合、可能であれば親要素から値を継承します。例えば、 {{ HTMLElement("div") }} の内部に段落（{{ HTMLElement("p") }}）があり、 {{ HTMLElement("div") }} が値が "Arial" の font プロパティを持ち、 {{ HTMLElement("p") }} が定義された font プロパティを持たない場合、 Arial フォントを継承します。
  3. 上記のいずれも利用できない場合、CSS 仕様によって指定された初期値がその要素に適用されます。
