<noembed> 要素は廃止された、標準外の方法であり、{{HTMLElement("embed")}} 要素に対応していないブラウザーや、ユーザーが仕様とした種類の埋め込みコンテンツに対応していないブラウザーで代替または「フォールバック」コンテンツを提供するものです。これは HTML 4.01 で非推奨となり、フォールバックコンテンツは {{HTMLElement("object")}} 要素の開始タグと終了タグの間に配置されるようになりました。

現在のところ、この要素はまだ多くのブラウザーで動作しますが、廃止されており、使用するべきではありません。代わりに {{HTMLElement("object")}} 要素の開始タグと終了タグの間にフォールバックコンテンツを入れて使用して下さい。
