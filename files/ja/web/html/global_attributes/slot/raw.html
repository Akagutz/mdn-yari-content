slot グローバル属性 は shadow DOM シャドーツリー内の slot をある要素に割り当てます。 slot 属性をもつ要素は、{{htmlattrxref("name", "slot")}} 属性の値がその slot 属性の値と一致する {{HTMLElement("slot")}} 要素により生成された slot に割り当てられます。

例えば、我々の Using templates and slots ガイドをみてください。
