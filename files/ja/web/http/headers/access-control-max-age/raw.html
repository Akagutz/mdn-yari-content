Access-Control-Max-Age 応答ヘッダーは、{{glossary("preflight request", "プリフライト要求")}} の結果 (つまり {{HTTPHeader("Access-Control-Allow-Methods")}} 及び {{HTTPHeader("Access-Control-Allow-Headers")}} ヘッダーに含まれる情報) をキャッシュすることができる時間の長さを示します。

ヘッダー種別 {{Glossary("Response header", "応答ヘッダー")}}
{{Glossary("Forbidden header name", "禁止ヘッダー名")}} いいえ

構文

Access-Control-Max-Age: <delta-seconds>

ディレクティブ

<delta-seconds>
結果をキャッシュすることができる最長の秒数です。
Firefox は24時間 (86400秒) キャッシュすることができ、 Chromium は10分 (600秒) です。 Chromium は既定値を5秒に指定することもできます。
-1の値はキャッシュを無効にし、すべての呼び出しをチェックするためにプリフライトの OPTIONS が必要になります。

例

プリフライト要求の結果は10分間キャッシュされます。

Access-Control-Max-Age: 600
