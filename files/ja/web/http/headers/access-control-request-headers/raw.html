Access-Control-Request-Headers リクエストヘッダーは {{glossary("preflight request")}} を発行するときに使用され、実際のリクエストが行われたときにどの HTTP ヘッダーが使用されるかをサーバーに知らせます。

ヘッダータイプ {{Glossary("Request header")}}
{{Glossary("Forbidden header name")}} はい

構文

Access-Control-Request-Headers: <header-name>, <header-name>, ...

ディレクティブ

<header-name>
リクエストに含まれる HTTP ヘッダーのコンマ区切りのリスト。

例

Access-Control-Request-Headers: X-PINGOTHER, Content-Type
