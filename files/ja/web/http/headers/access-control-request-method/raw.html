実際のリクエストが行われたときにどの HTTP メソッドが使用されるかをサーバーに知らせるために、{{glossary("preflight request")}} を発行するときに Access-Control-Request-Method リクエストヘッダーが使用されます。プリフライトリクエストは常に {{HTTPMethod("OPTIONS")}} であり、実際のリクエストと同じメソッドを使用しないので、このヘッダは必要です。

ヘッダータイプ {{Glossary("Request header")}}
{{Glossary("Forbidden header name")}} はい

構文

Access-Control-Request-Method: <method>

ディレクティブ

<method>
{{HTTPMethod("GET")}}、{{HTTPMethod("POST")}}{{HTTPMethod("DELETE")}} のような HTTP リクエストメソッドの1つ。

例

Access-Control-Request-Method: POST
