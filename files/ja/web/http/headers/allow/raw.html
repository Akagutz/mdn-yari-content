Allow ヘッダーには、リソースによってサポートされるメソッドの集合が一覧表示されます。

リクエストメソッドが使用されていることをサーバーの{{HTTPStatus("405")}} Method Not Allowed ステータスコードが示している場合、このヘッダーは必ず送信される必要があります。例えば空の Allow ヘッダーは、リソースがリクエストメソッドを許可しないことを示しており、これは特定のリソースに対して一時的に発生する可能性があります。

ヘッダータイプ {{Glossary("空のヘッダー")}}
{{Glossary("禁じられたヘッダーネーム")}} なし

構文

Allow: <http-methods>

ディレクティブ

<http-methods>
許可された HTTP リクエストメソッド の一覧。

例

Allow: GET, POST, HEAD
