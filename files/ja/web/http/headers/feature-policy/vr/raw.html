このディレクティブはxr として定義されていますが、現在は vr として Chrome で実装されています。

HTTP の {{HTTPHeader("Feature-Policy")}} ヘッダーにおける vr ディレクティブは、現在の文書が WebVR API を使用することを許可するかどうかを制御します。このポリシーが有効であれば、 {{domxref("Navigator.getVRDisplays()")}} から返却された {{jsxref("Promise")}} が {{domxref("DOMException")}} で拒否されます。

構文

Feature-Policy: vr <allowlist>;
<allowlist>
{{page("/ja/docs/Web/HTTP/Feature_Policy/Using_Feature_Policy", "allowlist")}}既定値は 'self' です。
