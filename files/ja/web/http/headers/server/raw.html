Server ヘッダーには、要求を処理するオリジンサーバが使用するソフトウェアについての情報が含まれます。

過度に長く詳細なサーバー値は、攻撃者が既知のセキュリティホールを見つけて悪用しやすくするための内部実装の詳細を潜在的に明らかにするので避けるべきです。

ヘッダータイプ {{Glossary("Response header")}}
{{Glossary("Forbidden header name")}} no

構文

Server: <プロダクト>

ディレクティブ

<プロダクト>
要求を処理するソフトウェアまたは、（サブ）プロダクトの名前。

例

Server: Apache/2.4.1 (Unix)
