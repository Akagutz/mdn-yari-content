何がうまくいかなかったのか？

特定の言語構造が予想されている箇所に、ほかのものが提供されています。これは単純なタイプミスの可能性があります。

例

式が予想される

たとえば関数を呼び出すとき、末尾のカンマは許可されていません。JavaScript は、実際は任意の式をとりえる、他の引数があることを予想します。

Math.max(2, 42,);
// SyntaxError: expected expression, got ')'

正しくは、カンマを省略するか、引数を追加します：

Math.max(2, 42);
Math.max(2, 42, 13 + 37);
