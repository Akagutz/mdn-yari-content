ArrayBuffer.prototype プロパティは {{jsxref("ArrayBuffer")}} オブジェクトのプロトタイプを表します。

{{js_property_attributes(0,0,0)}}

解説

{{jsxref("ArrayBuffer")}} インスタンスは ArrayBuffer.prototype から継承します。すべてのコンストラクターと同様に、すべての ArrayBuffer インスタンスを変更するために、コンストラクタープロトタイプオブジェクトを変更できます。

プロパティ

ArrayBuffer.prototype.constructor
オブジェクトのプロトタイプを生成する関数を指定します。既定値は標準で組み込まれた ArrayBuffer コンストラクターです。
{{jsxref("ArrayBuffer.prototype.byteLength")}} {{readonlyInline}}
配列のバイト単位の大きさです。これは、配列が構築されるときに設定され、変更されません。読み取り専用です。

メソッド

{{jsxref("ArrayBuffer.prototype.slice()")}}
新しい ArrayBuffer を返し、これにはこの ArrayBuffer の begin から end の手前までのバイト単位の内容がコピーされます。 begin または end のどちらかが負の数の場合、これは配列の先頭からではなく、末尾からのインデックスを表します。
