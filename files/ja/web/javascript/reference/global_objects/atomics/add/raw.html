静的な Atomics.add() メソッドは、配列内の指定した位置に値を追加して、その場所の古い値を返します。この atomic 操作は、修正された値が書き戻されるまで、他の書き込みが起こらないことを保証します。

構文

Atomics.add(typedArray, index, value)

パラメータ

typedArray
共有された整数の TypedArray。{{jsxref("Int8Array")}} か {{jsxref("Uint8Array")}}、{{jsxref("Int16Array")}}、{{jsxref("Uint16Array")}}、{{jsxref("Int32Array")}}、{{jsxref("Uint32Array")}} のいずれか。
index
value を追加する typedArray の位置
value
追加する数値。

戻り値

指定した位置の古い値（typedArray[index]）。

例外

  • typedArray が許可された整数型ではなかった場合、{{jsxref("TypeError")}} をスローする。
  • typedArray が共有された TypedArray 型ではなかった場合、{{jsxref("TypeError")}} をスローする。
  • typedArray の index が境界から外れていた場合、{{jsxref("RangeError")}} をスローする。

例

var sab = new SharedArrayBuffer(1024);
var ta = new Uint8Array(sab);

Atomics.add(ta, 0, 12); // 古い値である 0 を返す。
Atomics.load(ta, 0); // 12
