静的な Atomics.sub() メソッドは、配列内の所定の位置に指定された値を取り除いた上で、その位置にある古い値を返します。この atomic 操作は、修正された値が書き戻されるまで、ほかの書き込みが発生しないことを保証します。

構文

Atomics.sub(typedArray, index, value)

引数

typedArray
shared integer typed array。{{jsxref("Int8Array")}} か {{jsxref("Uint8Array")}}、{{jsxref("Int16Array")}}、{{jsxref("Uint16Array")}}、{{jsxref("Int32Array")}}、{{jsxref("Uint32Array")}} のいずれか。
index
取り除く value の typedArray 内での位置。
value
取り除く数値。

戻り値

指定した位置（typedArray[index]）の古い値。

例外

  • typedArray が整数型を受け入れないない場合、{{jsxref("TypeError")}} をスローする。
  • typedArray が shared typed array 型ではない場合、{{jsxref("TypeError")}} をスローする。
  • index が typedArray の境界から外れている場合、{{jsxref("RangeError")}} をスローする。

例

var sab = new SharedArrayBuffer(1024);
var ta = new Uint8Array(sab);
ta[0] = 48;

Atomics.sub(ta, 0, 12); // returns 48, the old value
Atomics.load(ta, 0); // 36
