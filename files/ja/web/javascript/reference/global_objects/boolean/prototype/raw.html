説明

{{jsxref("Boolean")}} インスタンスは、 Boolean.prototype を継承します。コンストラクターのプロトタイプオブジェクトを用いて、{{jsxref("Boolean")}} インスタンスのすべてにプロパティやメソッドを加えることができます。

プロパティ

Boolean.prototype.constructor
インスタンスのプロトタイプを生成する関数を返します。これは、既定では {{jsxref("Boolean")}} 関数です。

メソッド

{{jsxref("Boolean.prototype.toSource()")}} {{non-standard_inline}}
{{jsxref("Boolean")}} オブジェクトのソースを含む文字列を返します。この文字列を使って同等のオブジェクトを生成することができます。 {{jsxref("Object.prototype.toSource()")}} メソッドを上書きしています。
{{jsxref("Boolean.prototype.toString()")}}
オブジェクトの値によって、"true" か "false" のどちらかの文字列を返します。{{jsxref("Object.prototype.toString()")}} メソッドを上書きしています。
{{jsxref("Boolean.prototype.valueOf()")}}
{{jsxref("Boolean")}} オブジェクトのプリミティブな値を返します。{{jsxref("Object.prototype.valueOf()")}} メソッドを上書きしています。
