概要

{{jsxref("Global_Objects/Collator", "Intl.Collator")}}コンストラクタに対するプロトタイプオブジェクトを表します。

{{js_property_attributes(0, 0, 0)}}

説明

Intl.Collatorインスタンスの説明に対して{{jsxref("Collator")}}を確認して下さい。

{{jsxref("Global_Objects/Collator", "Intl.Collator")}} インスタンスは Intl.Collator.prototypeから継承します。 プロトタイプオブジェクトへの修正はすべての{{jsxref("Global_Objects/Collator", "Intl.Collator")}}インスタンスによって継承されます。
