{{JSRef("Global_Objects", "DataView")}}

概要

{{jsxref("DataView")}}の開始位置から特定のバイトオフセットで符号付き16-bit integer (short)を取得します。

構文

dataview.getInt16(byteOffset [, littleEndian])

引数

byteOffset
データを読み込むビューの開始位置からのバイトオフセット
littleEndian
{{optional_inline}} 16-bit intが{{Glossary("Endianness", "little- or big-endian")}} フォーマットで設定されているかどうかを表します。falseかundefinedの場合、 ビッグエンディアン値が読み込まれます。

エラースロー

{{jsxref("RangeError")}}
ビューの終わりを越えて読み込まれるようにbyteOffsetが設定されている場合、スローされます。

説明

アライメント制約はありません。マルチバイト値はどんなオフセットからでも取得されます。

例

getInt16 メソッドの使い方

var buffer = new ArrayBuffer(8);
var dataview = new DataView(buffer);
dataview.getInt16(1); // 0
