getMilliseconds() メソッドは、ローカル時間に基づき、指定された日時の「ミリ秒」を返します。

構文

dateObj.getMilliseconds()

戻り値

ローカル時間に基づき、指定された日時の「ミリ秒」を表す 0 から 999 までの間の数値。

例

getMilliseconds() を使う

次の例は、現在時刻のミリ秒部を変数 milliseconds に代入します:

var today = new Date();
var milliseconds = today.getMilliseconds();
