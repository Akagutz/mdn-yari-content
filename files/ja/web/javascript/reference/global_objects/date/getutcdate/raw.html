getUTCDate() は、協定世界時に基づき、指定された日付の「日」を返します。

{{EmbedInteractiveExample("pages/js/date-getutcdate.html")}}

構文

dateObj.getUTCDate()

戻り値

協定世界時に基づき、与えられた日付の「日」を表す 1 から 31 までの間の整数値です。

例

getUTCDate() を使う

次の例は、現在日時の「日」部分を day 変数へ代入します。

var today = new Date();
var day = today.getUTCDate();
