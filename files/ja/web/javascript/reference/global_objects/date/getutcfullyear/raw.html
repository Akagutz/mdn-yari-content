構文

dateObj.getUTCFullYear()

戻り値

協定世界時に基づき、与えられた日付の「年」を表す数値。

説明

getUTCFullYear() が返す値は、例えば 1995 などの 2000 年表記に従う絶対値です。

例

getUTCFullYear() を使う

次の例は、現在の年を表す 4 桁の値を変数 year に代入します。

var today = new Date();
var year = today.getUTCFullYear();
