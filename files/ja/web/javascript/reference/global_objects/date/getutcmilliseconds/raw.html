getUTCMilliseconds() メソッドは、協定世界時に基づき、指定された日時の「ミリ秒」を返します。

{{EmbedInteractiveExample("pages/js/date-getutcmilliseconds.html")}}

構文

dateObj.getUTCMilliseconds()

戻り値

協定世界時に基づき、与えられた日時の「ミリ秒」を表す 0 から 999 までの間の整数値。

例

getUTCMilliseconds() を使う

次の例は、現在時刻の「ミリ秒」部を変数 milliseconds に代入します。

var today = new Date();
var milliseconds = today.getUTCMilliseconds();
