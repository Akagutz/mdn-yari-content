columnNumber プロパティは、このエラーを起こしたファイルの行内の列番号を含みます。

例

columnNumber を使う

var e = new Error('Could not parse input');
throw e;
console.log(e.columnNumber) // 0

仕様

仕様の一部ではありません。非標準。
