{{JSRef}}

name プロパティは、エラーの種類の名称を表します。初期値は "Error" です。

説明

デフォルトで、{{jsxref("Error")}} インスタンスは "Error" という名称を与えられます。{{jsxref("Error.prototype.message", "message")}} プロパティに加え、name プロパティは、Error の文字列表現を生成するため、{{jsxref("Error.prototype.toString()")}} メソッドにより使用されます。

例

カスタムエラーを投げる

var e = new Error('Malformed input'); // e.name is 'Error'

e.name = 'ParseError';
throw e;
// e.toString() would return 'ParseError: Malformed input'
