概要

挿入順にMapオブジェクトの要素のキーを含む新しいIteratorオブジェクトを返します。

構文

myMap.keys()

例

例: keys()を使う

var myMap = new Map();
myMap.set("0", "foo");
myMap.set(1, "bar");
myMap.set({}, "baz");

var mapIter = myMap.keys();

console.log(mapIter.next().value); // "0"
console.log(mapIter.next().value); // 1
console.log(mapIter.next().value); // Object
