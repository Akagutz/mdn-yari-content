{{JSRef("Global_Objects", "Map")}}

概要

{{jsxref("Map")}}オブジェクトの要素の数を表します。

説明

sizeの値はどのくらいのエントリーをMapオブジェクトが持っているかを表す整数値です。sizeに対するセット関数はundefinedです。このプロパティを変更することは出来ません。

例

例: sizeを使う

var myMap = new Map();
myMap.set("a", "alpha");
myMap.set("b", "beta");
myMap.set("g", "gamma");

myMap.size // 3
