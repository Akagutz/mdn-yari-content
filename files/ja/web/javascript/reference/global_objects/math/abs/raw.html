Math.abs() 関数は、数値の絶対値を返します。すなわち、

構文

var abs = Math.abs(x);

引数

x
数値 (数字)。

返り値

与えられた数値 (数字) の絶対値。

詳細

abs は Math オブジェクトの静的なメソッドなので、自ら生成した Math オブジェクトのメソッドとしてではなく、常に、Math.abs() として使用するようにしてください。

例

例: Math.abs の動作

数ではない文字列、もしくは、{{jsxref("undefined")}}/空 である変数を渡した場合は、{{jsxref("NaN")}} を返します。{{jsxref("null")}} を渡した場合は、0 を返します。

Math.abs('-1');     // 1
Math.abs(-2);       // 2
Math.abs(null);     // 0
Math.abs('');       // 0
Math.abs([]);       // 0
Math.abs([2]);      // 2
Math.abs([1,2]);    // NaN
Math.abs({});       // NaN
Math.abs('string'); // NaN
Math.abs();         // NaN
