Math.cos() 静的関数は、指定された（ラジアン単位の）角度の{{原語併記("余弦","cosine")}}を返します。この値は length adjacent length hypotenuse です。

構文

Math.cos(x)

引数

x
余弦を返す角度（ラジアン単位）です。

戻り値

与えられた数値の余弦です。

説明

Math.cos() メソッドは、角度の余弦を表す、-1 から 1 までの数値を返します。

cos() は Math オブジェクトの静的なメソッドなので、自ら生成した Math オブジェクトのメソッドとしてではなく、常に、Math.cos() として使用するようにしてください。

例

Math.cos() を使う

Math.cos(0);           // 1
Math.cos(1);           // 0.5403023058681398

Math.cos(Math.PI);     // -1
Math.cos(2 * Math.PI); // 1
