Math.E プロパティは自然対数の底（e）、約 2.718 を表します。

Math.E=e≈2.718\mathtt{\mi{Math.E}} = e \approx 2.718

詳細

E は静的プロパティなので、生成した Math オブジェクトのプロパティとしてではなく、常に、Math.E として使用するようにしてください（Math にはコンストラクターではありません）。

例

例: Math.E の使用

以下の関数は、e を返します。

function getNapier() {
  return Math.E;
}

getNapier(); // 2.718281828459045
