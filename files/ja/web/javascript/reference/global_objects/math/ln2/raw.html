Math.LN2 プロパティは2の自然対数、およそ0.693を表します。

Math.LN2=ln(2)≈0.693\mathtt{\mi{Math.LN2}} = \ln(2) \approx 0.693

詳細

LN2 は静的プロパティなので、生成した Math オブジェクトのプロパティとしてではなく、常に、Math.LN2 として使用するようにしてください（Math にはコンストラクターがありません）。

例

例: Math.LN2 の使用

以下の関数は、2の自然対数を返します。:

function getNatLog2() {
  return Math.LN2;
}

getNatLog2(); // 0.6931471805599453
