Math.PI プロパティは円周率、約3.14159を表します。

Math.PI=π≈3.14159\mathtt{\mi{Math.PI}} = \pi \approx 3.14159

詳細

PI は静的プロパティなので、生成した Math オブジェクトのプロパティとしてではなく、常に、Math.PI として使用するようにしてください（Math にはコンストラクターではありません）。

例

例: Math.PI の使用

以下の関数は Math.PI を使用して、与えられた半径の円周を計算します。

function calculateCircumference(radius) {
  return Math.PI * (radius + radius);
}

calculateCircumference(1);  // 6.283185307179586
