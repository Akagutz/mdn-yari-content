Math は、数学的な定数と関数を提供するプロパティとメソッドを持つ、組み込みのオブジェクトです。関数オブジェクトではありません。

説明

他のグローバルオブジェクトと異なり、Math はコンストラクタではありません。Math オブジェクトのすべてのプロパティとメソッドは、静的です。円周率を表す定数πは、Math.PI として参照でき、{{原語併記("正弦", "sine")}} 関数は、Math.sin(x) として呼び出すことができます (ここでの x は、メソッドの引数です)。定数は、JavaScript における実数での完全精度で定義されています。

メソッド

三角関数 (sin(), cos(), tan(), asin(), acos(), atan(), atan2()) の角度は、引数も戻り値も {{interwiki('wikipedia', 'ラジアン')}} で表すことに注意してください。ラジアンを度 (°) に変換するには、(Math.PI / 180) で除算し、逆の場合はこの値で乗算します。

多くの数学関数の精度は実装に依存するので注意してください。これは、ブラウザーごとに結果が異なる場合があることを意味し、同じ JS エンジン上でさえも、OS やアーキテクチャが異なれば結果が異なる場合があります。

Math オブジェクトを拡張する

JavaScriptの多くのビルドインオブジェクトがそうであるように, Math オブジェクトもプロパティやメソッドを拡張することができる. Math オブジェクトを拡張するには,  prototypeを使い、 Math:の代わりに直接拡張します

Math.propName = propValue;
Math.methodName = methodRef;

For instance, the following example adds a method to the Math object for calculating the greatest common divisor of a list of arguments.

/* Variadic function -- Returns the greatest common divisor of a list of arguments */
Math.gcd = function() {
    if (arguments.length == 2) {
        if (arguments[1] == 0)
            return arguments[0];
        else
            return Math.gcd(arguments[1], arguments[0] % arguments[1]);
    } else if (arguments.length > 2) {
        var result = Math.gcd(arguments[0], arguments[1]);
        for (var i = 2; i < arguments.length; i++)
            result = Math.gcd(result, arguments[i]);
        return result;
    }
};

Try it:

console.log(Math.gcd(20, 30, 15, 70, 40)); // `5`
