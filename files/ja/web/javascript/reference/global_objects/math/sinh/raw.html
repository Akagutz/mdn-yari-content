{{JSRef("Global_Objects", "Math")}}

概要

引数として与えた数のハイパーボリックサインを返します。{{jsxref("Math.E", "constant e", "", 1)}}を用いて表現できます。:

Math.sinh(x)=ex-e-x2\mathtt{\operatorname{Math.sinh(x)}} = \frac{e^x - e^{-x}}{2}

構文

Math.sinh(x)

引数

x
数値

説明

sinh()はMathオブジェクトの静的なメソッドなので、自ら生成したMathオブジェクトのメソッドとしてではなく、常に、Math.sinh()として使用するようにしてください(Mathオブジェクトにはコンストラクタはありません)。

例

例: Math.sinh()を使う

Math.sinh(0); // 0
Math.sinh(1); // 1.1752011936438014

ポリフィル

{{jsxref("Math.exp()")}}関数の助けを借りて、エミュレートできます。:

Math.sinh = Math.sinh || function(x) {
  return (Math.exp(x) - Math.exp(-x)) / 2;
}

または {{jsxref("Math.exp()")}}関数の呼び出しを１回だけ使用します。:

Math.sinh = Math.sinh || function(x) {
  var y = Math.exp(x);
  return (y - 1 / y) / 2;
}
