Number.EPSILON プロパティは、1 と 1 より大きい浮動小数点の最小値の差を表します。

この静的なプロパティにアクセスするために {{jsxref("Number")}} オブジェクトを生成する必要はありません（Number.EPSILON を使ってください）。

説明

EPSILON プロパティは およそ 2.2204460492503130808472633361816E-16、または 2-52 の値を持っています。

例

同等なことをテストする

x = 0.2;
y = 0.3;
z = 0.1;
equal = (Math.abs(x - y + z) < Number.EPSILON);

Polyfill

if (Number.EPSILON === undefined) {
    Number.EPSILON = Math.pow(2, -52);
}
