{{JSRef("Global_Objects", "Number")}}

概要

文字列引数をパースし指定された基数または底の整数値を返します。

構文

Number.parseInt(string[, radix])

引数

{{page("/ja/docs/Web/JavaScript/Reference/Global_Objects/parseInt", "引数")}}

戻り値

{{page("/ja/docs/Web/JavaScript/Reference/Global_Objects/parseInt", " 戻り値")}}

説明

このメソッドはグローバル関数{{jsxref("Global_Objects/parseInt", "parseInt()")}}と完全に同じ関数です。:

Number.parseInt === parseInt; // true

ECMAScript 2015でサポートされております(目的はグローバル関数のモジュール化です)。詳細や例について{{jsxref("Global_Objects/parseInt", "parseInt()")}}を確かめて下さい。

ポリフィル

if (Number.parseInt === undefined) {
    Number.parseInt = window.parseInt;
}
