Proxy の handler オブジェクトは、{{jsxref("Proxy", "proxy", "", 1)}} に対するトラップを保持するプレースホルダーオブジェクトです。

メソッド

すべてのトラップはオプションです。トラップが定義されていない場合、デフォルトの振る舞いはターゲットに操作を転送することです。
