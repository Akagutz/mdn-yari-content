{{JSRef}}

Reflect は、インターセプトが可能な JavaScript 操作に対するメソッドを提供するビルトインオブジェクトです。メソッドは proxy handlers のメソッドと同じです。Reflect は関数オブジェクトではありませんので、構築ができません。

説明

他のグローバルオブジェクトとは異なり、Reflect はコンストラクタではありません。new 演算子で使用したり、関数として Reflect オブジェクトを呼び出すことはできません。Reflect のすべてのプロパティとメソッドは（ {{jsxref("Math")}} オブジェクトの場合のように）静的です。

メソッド

Reflect オブジェクトは proxy handler methods と同じ名前をもち、{{jsxref("Object")}} のメソッドのいくつかと重複する静的な関数を提供します。

{{jsxref("Reflect.apply()")}}
args 引数によって指定される値をもつターゲット関数を呼び出します。{{jsxref("Function.prototype.apply()")}}も確かめてください。
{{jsxref("Reflect.construct()")}}
new 演算子を関数にしたものです。new target(...args) を呼び出すのと同等です。
{{jsxref("Reflect.defineProperty()")}}
{{jsxref("Object.defineProperty()")}}に似ています。{{jsxref("Boolean")}}を返します。
{{jsxref("Reflect.deleteProperty()")}}
delete 演算子を関数にしたものです。delete target[name] を呼び出すのと同等です。
{{jsxref("Reflect.get()")}}
プロパティの値を返す関数です。
{{jsxref("Reflect.getOwnPropertyDescriptor()")}}
{{jsxref("Object.getOwnPropertyDescriptor()")}} に似ています。指定したプロパティが存在する場合にプロパティディスクリプタを返すか、存在しない場合には {{jsxref("undefined")}} を返します。
{{jsxref("Reflect.getPrototypeOf()")}}
{{jsxref("Object.getPrototypeOf()")}} と同じです。
{{jsxref("Reflect.has()")}}
in 演算子を関数にしたものです。独自か継承したプロパティが存在するかどうかを示す boolean を返します。
{{jsxref("Reflect.isExtensible()")}}
{{jsxref("Object.isExtensible()")}} と同じです。
{{jsxref("Reflect.ownKeys()")}}
独自の（継承されない）プロパティのキーをもつ文字列の配列を返します。
{{jsxref("Reflect.preventExtensions()")}}
{{jsxref("Object.preventExtensions()")}} に似ています。{{jsxref("Boolean")}} を返します。
{{jsxref("Reflect.set()")}}
値をプロパティに割り当てる関数です。更新が成功したら true である {{jsxref("Boolean")}} を返します。
{{jsxref("Reflect.setPrototypeOf()")}}
オブジェクトのプロトタイプを設定する関数です。
