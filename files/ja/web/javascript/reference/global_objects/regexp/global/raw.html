global プロパティは "g" フラグが正規表現で使われているかどうかを返します。global は個々の正規表現インスタンスの読取専用プロパティです。

説明

global の値は {{jsxref("Boolean")}} です。"g" フラグが使われているなら true で、そうでなければ false です。"g" フラグは、正規表現が文字列での可能なマッチのすべてに対してテストされるべきであることを示します。A regular expression defined as both global("g") and sticky ("y") will ignore the global flag and perform sticky matches.

このプロパティを直接変更することはできません。

例

global を使う

var regex = new RegExp('foo', 'g');

console.log(regex.global); // true
