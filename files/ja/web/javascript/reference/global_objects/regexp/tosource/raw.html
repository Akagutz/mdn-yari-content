概要

オブジェクトのソースコードを表す文字列を返します。

構文

regexObj.toSource()

引数

無し。

説明

toSource メソッドは以下の値を返します。:

  • 組み込み RegExp オブジェクトでは、 toSource はソースコードが存在しないことを示す以下の文字列を返します。:
function RegExp() {
   [native code]
}
  • RegExp オブジェクトのインスタンスでは、 toSource はそのソースコードを表す文字列を返します。

このメソッドは、通常、 JavaScript によって内部的に呼ばれ、コード中で明示的に呼ばれることはありません。

参照

Object.prototype.toSource
