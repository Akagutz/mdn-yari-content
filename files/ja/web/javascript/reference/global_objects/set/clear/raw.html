clear() メソッドは、Set オブジェクトからすべての要素を取り除きます。

構文

mySet.clear();

戻り値

{{jsxref("undefined")}} です。

例

clear メソッドを使う

var mySet = new Set();
mySet.add(1);
mySet.add("foo");

mySet.size;       // 2
mySet.has("foo"); // true

mySet.clear();

mySet.size;       // 0
mySet.has("bar")  // false
