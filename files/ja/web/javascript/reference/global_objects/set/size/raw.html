説明

size の値は、Set オブジェクトがどのくらいのエントリーを持つかを表す整数値です。size に対する Set のアクセス関数は undefined です。よって、このプロパティは変更できません。

例

size を使う

var mySet = new Set();
mySet.add(1);
mySet.add(5);
mySet.add('some text')

mySet.size; // 3
