values() メソッドは、挿入順で Set オブジェクト内の各要素の値を含んだ、新しい Iterator オブジェクトを返します。

keys() メソッドは（ {{jsxref("Map")}} オブジェクトとの類似性のため）このメソッドに対するエイリアスです。まったく同じように動作し、Set の各要素の値を返します。

構文

mySet.values();

戻り値

挿入順で Set オブジェクト内の各要素の値を含んだ、新しい Iterator オブジェクトです。

例

values() を使う

var mySet = new Set();
mySet.add('foo');
mySet.add('bar');
mySet.add('baz');

var setIter = mySet.values();

console.log(setIter.next().value); // "foo"
console.log(setIter.next().value); // "bar"
console.log(setIter.next().value); // "baz"
