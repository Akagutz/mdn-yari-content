SharedArrayBuffer.prototype プロパティは、{{jsxref("SharedArrayBuffer")}} オブジェクトのプロトタイプを表します。

{{js_property_attributes(0,0,0)}}

説明

SharedArrayBuffer インスタンスは、 SharedArrayBuffer.prototype を継承します。すべてのコンストラクタと同様に、すべての SharedArrayBuffer インスタンスを変更するためにコンストラクタの prototype オブジェクトを変更できます。

プロパティ

SharedArrayBuffer.prototype.constructor
オブジェクトの prototype を生成する関数を指定する。初期値は標準ビルトインの SharedArrayBuffer コンストラクタである。
{{jsxref("SharedArrayBuffer.prototype.byteLength")}} {{readonlyInline}}
配列のバイト単位でのサイズ。これは配列が生成されたときに決定され、変更できない。 読み取り専用。

メソッド

{{jsxref("SharedArrayBuffer.prototype.slice(begin, end)")}}
begin から end までの SharedArrayBuffer バイトのコピーを含む新しい SharedArrayBuffer を返す。begin か end が負数だった場合、最初から参照するのとは対照的に、配列の終わりからインテックスを参照します。
