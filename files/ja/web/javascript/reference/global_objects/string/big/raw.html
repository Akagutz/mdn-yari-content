big() メソッドは、文字列を大きなフォントサイズで表示させる {{HTMLElement("big")}} HTML 要素を生成します。

使用時の注意: <big> 要素は HTML5 ではすでに廃止されており、使うべきではありません。Web 開発者は代わりに CSS プロパティを使うべきです。

構文

str.big()

戻り値

{{HTMLElement("big")}} HTML 要素を含む文字列です。

説明

big() メソッドは <big> の中に文字列を埋め込みます。つまり "<big>str</big>" のようになります。

例

big() の使用

以下の例では、文字の大きさを変化させるために string オブジェクトのメソッドを使用しています:

var worldString = 'Hello, world';

console.log(worldString.small());     // <small>Hello, world</small>
console.log(worldString.big());       // <big>Hello, world</big>
console.log(worldString.fontsize(7)); // <fontsize=7>Hello, world</fontsize>

{{domxref("HTMLElement.style", "element.style")}} を使えば要素の style 属性を取得でき、より汎用的に操作できます。例えば、:

document.getElementById('yourElemId').style.fontSize = '2em';
