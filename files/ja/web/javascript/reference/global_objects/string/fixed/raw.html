概要

文字列を {{htmlelement("tt")}} タグ で囲んだときのように、等幅フォントで表示します。

構文

str.fixed()

詳細

文書中で文字列を表示するためには、write 、あるいは、writeln メソッドと共に、 fixed メソッドを使用してください。サーバーサイド JavaScript では、write 関数を使って、文字列を表示してください。

例

var worldString = "世界のみなさん、こんにちは！";

document.write( worldString.fixed() );

この例は、以下の HTML と同様の出力を生成します。

<tt>世界のみなさん、こんにちは！</tt>
