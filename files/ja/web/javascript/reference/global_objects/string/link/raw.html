概要

他の URL をリクエストする HTML ハイパーテキストリンクを生成します。

構文

str.link(url)

引数

hrefAttribute
A タグの HREF 属性に指定する任意の文字列。妥当な（相対あるいは絶対） URL でなければならない。

詳細

ハイパーテキストリンクを表す HTML の断片 を生成するためには、link メソッドを使用してください。戻り値の文字列は、{{ Domxref("document.write") }}、あるいは、{{ Domxref("element.innerHTML") }} を通して、文書中に追加することができます。

link メソッドによって生成されたリンクは　document オブジェクトの links 配列の要素になります。{{ Domxref("document.links") }} を参照してください。

例

例: link メソッドの使用

"MDN" という文字列を、ユーザを MOZILLA DEVELOPER NETWORK に導くハイパーテキストリンクとして表示する例を以下に示します。

var hotText = "MDN";
var URL = "https://developer.mozilla.org/";

document.write( "Click to return to " + hotText.link(URL) );

この例は、以下の HTML と同様の出力を生成します。:

Click to return to <A HREF="https://developer.mozilla.org/">MDN</A>
