{{jsxref("Symbol")}}コンストラクタに対するプロトタイプを表します。

{{js_property_attributes(0,0,0)}}

説明

{{jsxref("Symbol")}}インスタンスは{{jsxref("Symbol.prototype")}}から継承します。すべてのSymbolインスタンスにプロパティやメソッドを追加するためにコンストラクタのプロトタイプオブジェクトを使用してください。

プロパティ

Symbol.prototype.constructor
インスタンスのプロトタイプを生成した関数を返します。デフォルトでは{{jsxref("Symbol")}}関数です。

メソッド

{{jsxref("Symbol.prototype.toSource()")}} {{Non-standard_inline}}
{{jsxref("Global_Objects/Symbol", "Symbol")}}オブジェクトのソースコードを含む文字列を返します。{{jsxref("Object.prototype.toSource()")}}メソッドをオーバーライドします。
{{jsxref("Symbol.prototype.toString()")}}
Symbolの説明を含む文字列を返します。{{jsxref("Object.prototype.toString()")}} メソッドをオーバーライドします。
{{jsxref("Symbol.prototype.valueOf()")}}
{{jsxref("Symbol")}}オブジェクトのプリミティブ値を返します。{{jsxref("Object.prototype.valueOf()")}}メソッドをオーバーライドします。
