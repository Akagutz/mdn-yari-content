Symbolオブジェクトのプリミティブ値を返します。

構文

Symbol().valueOf();

説明

{{jsxref("Symbol")}}オブジェクトのvalueOfメソッドはSymbolデータ型としてSymbolオブジェクトのプリミティブ値を返します。

JavaScriptでは、オブジェクトをプリミティブ値に変換するためにvalueOfメソッドを呼びます。自らvalueOfメソッドを呼び出す必要はめったにありません。JavaScriptではプリミティブ値が期待されるオブジェクトに出くわすと自動的に呼び出されます。
