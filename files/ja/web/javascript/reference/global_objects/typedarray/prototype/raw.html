概要

{{jsxref("TypedArray")}}コンストラクタに対するプロトタイプを表します。

{{js_property_attributes(0,0,0)}}

説明

{{jsxref("TypedArray")}}インスタンスは{{jsxref("TypedArray.prototype")}}から継承します。プロパティやメソッドをすべてのTypedArrayインスタンスに追加するためのコンストラクタのプロトタイプオブジェクトを使用できます。TypedArrayは、typed array typesの一つです。

継承の詳細については、TypedArrayについての説明も確かめて下さい。

プロパティ

TypedArray.prototype.constructor
インスタンスのプロトタイプを作る関数を返します。これは、デフォルトで対応するtyped array type関数です。
{{jsxref("TypedArray.prototype.buffer")}} {{readonlyInline}}
typed arrayによって参照される{{jsxref("ArrayBuffer")}}を返します。構築時に設定され、読取専用となります。
{{jsxref("TypedArray.prototype.byteLength")}} {{readonlyInline}}
{{jsxref("ArrayBuffer")}}の開始位置からのtyped arrayの(バイト単位の)長さを返します。構築時に設定され、読取専用となります。
{{jsxref("TypedArray.prototype.byteOffset")}} {{readonlyInline}}
{{jsxref("ArrayBuffer")}}の開始位置からのtyped arrayの(バイト単位の)オフセットを返します。構築時に設定され、読取専用となります。
{{jsxref("TypedArray.prototype.length")}} {{readonlyInline}}
typed array内に保持された要素の数を返します。構築時に設定され、読取専用となります。
