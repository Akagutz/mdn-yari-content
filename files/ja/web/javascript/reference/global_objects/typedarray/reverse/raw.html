概要

typed arrayの位置を反転させます。最初のtyped arrayの要素は最後になり、最後の要素は最初になります。このメソッドは{{jsxref("Array.prototype.reverse()")}}と同じアルゴリズムを持ちます。 TypedArrayはここではtyped array types の一つです。

構文

typedarray.reverse();

例

var uint8 = new Uint8Array([1, 2, 3]);
uint8.reverse();

console.log(uint8); // Uint8Array [3, 2, 1]
