{{JSRef("Global_Objects", "WeakSet")}}

概要

新しいオブジェクトをWeakSetオブジェクトの最後に追加します。

構文

ws.add(value);

引数

value
必須。WeakSetコレクションに追加するオブジェクト

例

例: add メソッドを使う

var ws = new WeakSet();

ws.add(window); // add the window object to the WeakSet

ws.has(window); // true
