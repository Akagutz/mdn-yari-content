概要

WeakSetオブジェクト内にオブジェクトが存在しているかどうかを示すbooleanを返します。

構文

ws.has(value);

引数

value
必須。WeakSetに存在しているかテストするオブジェクト。

戻り値

Boolean
指定された値を持つ要素がWeakSetオブジェクト内に存在する場合、trueを返します。さもなければ、falseを返します。

例

例: has メソッドを使う

var ws = new WeakSet();
var obj = {};
ws.add(window);

mySet.has(window);  // returns true
mySet.has(obj);     // returns false
