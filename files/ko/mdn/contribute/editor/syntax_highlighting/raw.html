MDN의 글에 코드 예제를 추가할 때, 읽는 이를 위해 시각적으로 관련 코드 조각을 식별하기 쉽게 구문 강조를 사용(enable)하는 것이 유용합니다. 여기 코드 예제에서 구문 강조를 사용하는 법이 있습니다:

  1. 글 속에 코드를 치거나 붙여넣으세요. 예를 들어:

    void main(int argc, char **argv) {

    printf("Hello world\n");

    }

  2. 텍스트를 선택하고 편집기 툴바에서 PRE 아이콘을 클릭하세요:

    void main(int argc, char **argv) {

printf("Hello world\n");

}
  3. 구문 강조(기) 아이콘을 클릭하고 코드 예제를 위한 해당 프로그래밍 언어의 이름을 고르세요. 이 경우에는, C/C++입니다: 
    void main(int argc, char **argv) {

printf("Hello world\n");

}

주의: 편집기는 구문 강조를 표시하지 않습니다. 미리보기 또는 바꾼 내용을 저장하고 구문 강조를 실제로 보기 위해 글을 살펴봐야 합니다.
