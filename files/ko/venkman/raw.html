Getting Venkman

  • Mozilla Suite and SeaMonkey: Venkman is an install option (choose Advanced install).
  • Mozilla Suite, SeaMonkey, Firefox, and Thunderbird: the latest version of Venkman is available from Mozilla Addons.
  • Nvu version is available from glazman.org.
