AbstractWorker 인터페이스는 {{domxref("SharedWorker")}}나 {{domxref("Worker")}}와 같은 모든 종류의 워커에 공통적으로 들어가는 속성과 메소드들을 추상적으로 나타냅니다.

속성

AbstractWorker 인터페이스는 어떠한 속성도 상속하지 않습니다.

{{domxref("AbstractWorker.onerror")}}
는 워커를 통해 발생하는 error 버블 타입의 {{domxref("ErrorEvent")}}가 나타날때마다 호출되는  {{domxref("EventListener")}}입니다.

메서드

AbstractWorker 인터페이스는 어떠한 메서드도 상속받거나 이행하지 않습니다.
