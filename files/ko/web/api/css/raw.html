CSS 인터페이스는 유용한 CSS 관련 메서드를 가집니다. CSS 인터페이스는 정적 속성과 메서드만 가지고 있으며 아무런 객체도 CSS 인터페이스를 구현하지 않습니다. 

속성

정적 속성

{{domxref("CSS.paintWorklet")}} {{experimental_inline}}
페인팅painting에 관련된 모든 클래스를 지닌 워크렛worklet에 접근할 수 있습니다.

메서드

정적 메서드

{{domxref("CSS.supports()")}}
매개변수로 주어진 속성-값 쌍 또는 조건이 지원되는지 여부를 나타내는 {{domxref("Boolean")}}을 반환합니다.
{{domxref("CSS.escape()")}} {{experimental_inline}}
CSS 선택자로 사용할 문자열을 이스케이프해서 반환합니다.
