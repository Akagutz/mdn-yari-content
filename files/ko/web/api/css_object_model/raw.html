CSS Object Model은 자바 스크립트에서 CSS를 조작 할 수있게 해주는 API 세트입니다.  CSS의 경우 DOM와 HTML API의 펜던트이며, 이를 통해 CSS양식을 동적으로 읽고 수정할 수 있습니다.
