웹 어플리케이션이 history navigation에서 명시적으로 기본 스크롤 복원 동작을 설정할 수 있도록 허용합니다. 이 속성은 문자열이며 유효한 값은 'auto' 혹은 'manual' 입니다.

문법

let scrollRestore = History.scrollRestoration;

Value

이 문자열 속성의 값으로 다음 두 가지를 사용할 수 있습니다.

"auto"
사용자가 스크롤한 페이지의 위치가 복원됩니다.
"manual"
사용자가 스크롤한 페이지의 위치는 복원되지 않습니다. 사용자는 위치로 직접 스크롤해야 합니다.
