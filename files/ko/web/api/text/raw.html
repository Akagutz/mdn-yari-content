Text 인터페이스는 {{domxref("Element")}} 또는 {{domxref("Attr")}} 의 문자형 컨텐츠를 표현합니다. 엘리먼트의 컨텐츠에 마크업이 없을 경우 엘리먼트의 텍스트를 갖는 Text 를 구현하는 단일 자식을 갖습니다. 반면에 엘리먼트에 마크업이 있을 경우 자식을 구성하는 정보 항목과 Text 노드로 파싱됩니다.

새 다큐먼트는 각각의 텍스트 블록에 대해 단일 Text 노드를 갖습니다. 다큐먼트의 컨텐츠가 변경됨에 따라 더 많은 Text 노드가 생성될 것입니다. {{domxref("Node.normalize()")}} 메소드는 인접한 Text 객체에 대한 각각의 텍스트 블록을 다시 단일 노드로 병합합니다.
