USVString 은 유니코드 스칼라 값의 모든 가능한 시퀀스 집합에 해당합니다. USVString 은 JavaScript로 반환될 때, {{jsxref("String")}} 에 매핑됩니다; 작동할 유니코드 스칼라 값 문자열을 필요로하고, 일반적으로 텍스트 처리를 수행하는 API에만 사용됩니다. USVString 은 {{domxref("DOMString")}}와 같습니다. 다만, 지원되지 않는 대체 코드 포인트를 허용하는 않습니다. USVString 에서 처리되지 않은 대체 코드 포인트는 브라우저에서 유니코드 'replacement character' U+FFFD, (�) 로 변환됩니다.
