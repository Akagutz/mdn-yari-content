요약

동적으로 창의 크기를 변경

문법

window.resizeTo(aWidth, aHeight)

매개변수

  • aWidth는 스크롤바 및 타이틀바등을 포함하는 픽셀단위의 새로운 {{domxref("window.outerWidth","outerWidth")}} 정수값이다.
  • aHeight는 스크롤바 및 타이틀바등을 포함하는 픽셀단위의 새로운 {{domxref("window.outerHeight","outerHeight")}} 정수값이다.

예제

// 현재 창을 화면의 1/4을 차지하는 크기로 변경

function quarter() {
  window.resizeTo(
    window.screen.availWidth / 2,
    window.screen.availHeight / 2
  );
}

주의사항

아래의 사항에 대항되는 경우, Firefox 7 버전 이후로는 브라우져상에서 윈도우의 기본 크기를 변경하는것은 더 이상 가능하지 않음.

  1. window.open함수에 의해서 생성되지 않은 윈도우나 탭의 크기를 변경할 수 없음.
  2. 해당 윈도우가 한개 이상의 탭을 포함하고 있을 경우, 해당 윈도우나 탭의 크기를 조정할 수 없음.

참조 :  {{domxref("window.resizeBy")}}.
