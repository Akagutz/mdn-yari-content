일반 대열, 또는 대열 조판은 블록 및 인라인 요소가 웹 페이지에 표시되는 방식이다. 차후 해당 조판에 변경이 이뤄질 수도 있다. 일반 대열은 본질적으로 모두 협력 관계에 있는 사물의 집합으로 당신의 조판에 소속된 사물들은 상대를 서로 인지하고 있다. 어떤 요소가 대열에서 이탈하는 해당 요소는 독립적으로 작동한다.

일반 대열에 포함된 인라인 요소는 인라인 방향으로 표시된다. 말인즉 문서의 쓰기 모드에 의거해 단어가 문장의 형태로 표시되는 방향을 말한다. Block 요소는 당해 문서에 쓰기 모드에 소속된 단락의 사례처럼 하나씩 차례대로 표시된다. 따라서 영문의 경우 인라인 요소는 왼쪽부터 시작해 하나씩 차례대로 표시되며, 블록 요소는 최상단에서 시작하여 하나씩 차례대로 페이지 아래로 이동하며 표시된다.
