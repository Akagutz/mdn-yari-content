요약

:first {{cssxref("@page")}} CSS pseudo-class 는 문서를 인쇄할 때, 첫번째 페이지의 스타일링을 표현합니다.

Note:  모든 CSS properties를 바꿀 수는 없습니다.  "margins, orphans, widows, page breaks"에 대한 속성만을 바꿀 수 있습니다.  다른 css 속성은 무시됩니다.

예시

@page :first {
  margin: 2in 3in;
}
Note:  마진을 정의할 때, 절대적 length 단위를 사용할수도 있습니다.  자세한 정보는 length 에서 확인하세요.
