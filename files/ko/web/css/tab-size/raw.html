요약

tab-size CSS 속성은 탭 문자(U+0009)의 너비를 조절하는데 사용됩니다.

{{cssinfo}}

문법

문법 형식: {{csssyntax("tab-size")}}

tab-size: 4
tab-size: 0
tab-size: 8

tab-size: inherit

값

{{cssxref("<integer>")}}
하나의 탭이 표현될 공백의 수입니다. 음수일 수 없습니다.
{{cssxref("<length>")}}
탭의 너비입니다. 음수일 수 없습니다.
inherit
부모 요소로부터 상속받습니다.

예제

pre {
  tab-size: 4; /* 탭 크기를 4 칸의 공백으로 정함 */
}

pre {
  tab-size: 0; /* 들여쓰기를 없앰 */
}

pre {
  tab-size: 99; /* 탭을 쓰지 마세요! */
}
