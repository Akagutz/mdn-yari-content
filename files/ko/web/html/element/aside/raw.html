HTML <aside> 요소는 문서의 주요 컨텐츠에 별도로 이어진 컨텐츠가 있는 문서의 한 구획을 말합니다. 주로 사이드바로 나타냅니다.

{{EmbedInteractiveExample("pages/tabbed/aside.html", "tabbed-standard")}}

The source for this interactive example is stored in a GitHub repository. If you'd like to contribute to the interactive examples project, please clone https://github.com/mdn/interactive-examples and send us a pull request.

  • 컨텐츠 범주 플로우 컨텐츠, 구획 컨텐츠, palpable 컨텐츠.
  • 허용된 컨텐츠플로우 컨텐츠.
  • 태그 생략 {{no_tag_omission}}
  • 허용된 부모 요소 플로우 컨텐츠를 허용하는 요소.
    <aside> 요소는 {{HTMLElement("address")}} 요소의 후손이 될수 없다는 점을 주의.
  • DOM 인터페이스 {{domxref("HTMLElement")}}

속성

이 요소는 전역 속성만을 포함합니다.

사용 일람

  • <aside> 요소를 주요 흐름의 일부로 파악되는 문단으로서 사용하지 마십시오.

예제

<article>
  <p>
    The Disney movie <em>The Little Mermaid</em> was
    first released to theatres in 1989.
  </p>
  <aside>
    The movie earned $87 million during its initial release.
  </aside>
  <p>
    More info about the movie...
  </p>
</article>
