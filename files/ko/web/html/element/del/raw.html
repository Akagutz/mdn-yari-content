HTML <del> 요소는 문서에서 삭제된 텍스트의 범위를 나타냅니다. 이 요소는 보통 (그럴 필요는 없지만) 가운데줄 텍스트로 렌더링 됩니다.

The source for this interactive example is stored in a GitHub repository. If you'd like to contribute to the interactive examples project, please clone https://github.com/mdn/interactive-examples and send us a pull request.

  • 컨텐트 카테고리Phrasing 컨텐트 혹은 flow 컨텐트.
  • 허용되는 컨텐트 Transparent.
  • 태그 생략 {{no_tag_omission}}
  • 허용되는 부모 요소들 phrasing 컨텐트를 허용하는 모든 태그들
  • DOM 인터페이스 {{domxref("HTMLModElement")}}

속성

이 요소는 전역 속성을 포함합니다.

{{htmlattrdef("cite")}}
변경을 설명하는 리소스의 URI (예를 들자면, 회의록).
{{htmlattrdef("datetime")}}
이 속성은 변경이 일어난 시간과 날짜를 나타내며 올바른 날짜와 선택적인 시간 문자열도 함께)여야 합니다. 만약 값이 날짜와 선택적인 시간 문자열로 분석될수 없는경우, 요소는 관련된 타임스탬프를 가지고 있지 않습니다.

예제

<p><del>This text has been deleted</del>,
here is the rest of the paragraph.</p>
<del><p>This paragraph has been deleted.</p></del>
