개요

HTML 세부요소 (<details>) 는 사용자가 추가 정보를 검색할 수 있는 공시 위젯으로 사용된다.

콘텐츠 범주 플로우 콘텐츠, sectioning root, 대화형 콘텐츠, 가시성 콘텐츠.
허용된 콘텐츠 플로우 콘텐츠가 뒤따르는 {{HTMLElement("summary")}}요소.
태크 생략 {{no_tag_omission}}
허용된 부모 요소 플로우 콘텐츠를 허용하는 모든 요소.
DOM 인터페이스 {{domxref("HTMLDetailsElement")}}

속성

이 요소는 글로버 속성을 포합합니다.

{{htmlattrdef("open")}}
이 Boolean 속성은 페이지 로드에 있는 사용자에게 세부사항들을 보여줄지에 대해 나타낸다.

예제

<details>
  <summary>Some details</summary>
  <p>More info about the details.</p>
</details>
