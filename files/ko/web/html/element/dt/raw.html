HTML <dt> 요소 (or HTML Definition Term Element)는 정의 목록에서 용어를 구분합니다. 이 요소는 {{HTMLElement("dl")}} 요소의 자식으로서만 등장할수 있습니다. 보통 이것은 {{HTMLElement("dd")}}요소 전에 나오는데, 연속적으로 나오는 여러개의 <dt>요소는 바로 다음에 나오는 {{HTMLElement("dd")}}요소에 의해 정의되는 몇개의 용어를 가르킵니다.

  • 컨텐츠 범주 없음
  • 허용된 컨텐츠{{todo}} Flow content, but with no {{HTMLElement("header")}}, {{HTMLElement("footer")}}, sectioning content or heading content descendants.
  • 태그 생략 시작 태그는 반드시 있어야함.
    종료 태그는 이 요소 바로 뒤에 다른 <dd> 태그가 따라오거나, 부모 요소에 더 이상 다른 요소가 없을떄 생략 가능함.
  • 허용된 부모 요소{{HTMLElement("dl")}}안에서 {{HTMLElement("dt")}} 또는 {{HTMLElement("dd")}} 앞에
  • DOM 인터페이스 {{domxref("HTMLElement")}}
    Gecko 1.9.2 (Firefox 4) 까지, Firefox는 이 요소를 위해  HTMLSpanElement 인터페이스를 구현했음.
