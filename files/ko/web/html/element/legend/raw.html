개요

HTML <legend> 요소는(또는 HTML Legend Field 요소) 부모 요소 {{HTMLElement("fieldset")}}의 캡션을 의미합니다.

  • 콘텐츠 범주 없음
  • 허용되는 콘텐츠 Phrasing content.
  • 태그 생략 {{no_tag_omission}}
  • 허용되는 부모 요소 {{HTMLElement("fieldset")}}
  • DOM 인터페이스 {{domxref("HTMLLegendElement")}}

속성

이 요소는 전역 요소만을 포함합니다.
