HTML <param> 요소는 {{ HTMLElement("object") }}의 매개 변수를 정의합니다.

  • 컨텐츠 범주 없음.
  • 허용된 컨텐츠 없음. 이것은 {{Glossary("empty element")}}.
  • 태그 생략 이것은 빈 요소 이기떄문에, 시작 태그는 있어야하며, 종료 태그는 있어서는 안됩니다.
  • 허용된 부모 요소들 {{ HTMLElement("object") }} before any flow content.
  • DOM 인터페이스 {{domxref("HTMLParamElement")}}

속성

이 요소는 전역 속성을 포함합니다.

{{ htmlattrdef("name") }}
매개 변수의 이름
{{ htmlattrdef("type") }} {{ obsolete_inline() }}
Only used if the valuetype is set to "ref". Specifies the MIME type of values found at the URI specified by value.
{{ htmlattrdef("value") }}
매개 변수의 값을 지정합니다.
{{ htmlattrdef("valuetype") }} {{ obsolete_inline() }}
Specifies the type of the value attribute. Possible values are:
  • data: Default value. The value is passed to the object's implementation as a string.
  • ref: The value is a URI to a resource where run-time values are stored.
  • object: An ID of another {{ HTMLElement("object") }} in the same document.
