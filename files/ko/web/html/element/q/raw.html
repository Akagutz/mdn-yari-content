HTML <q>요소는 둘러싼 텍스트가 짧은 인라인 인용문이라는것을 나타냅니다. 이 요소는 단락 구분이 필요없는 짧은 인용문을 위해 사용되며, 긴 인용문에는 {{HTMLElement("blockquote")}} 요소를 사용하세요.

속성

이 요소는 전역 속성을 포함합니다.

{{htmlattrdef("cite")}}
이 속성의 값은 인용된 정보의 원본 문서나 메세지를 가리키는 URL를 나타냅니다. This attribute is intended to point to information explaining the context or the reference for the quote.

예제

<p>According to Mozilla's website,
  <q
  cite="https://www.mozilla.org/en-US/about/history/details/">Firefox 1.0
  was released in 2004 and became a big success.</q></p>
