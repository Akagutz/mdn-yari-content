Summary

HTML 밑줄 요소(<u>)는 밑줄이 있는 글자를 표현합니다.

이 요소는 HTML5에서 분명하게 표현되는 발음이 잘못된 문자나, 중국 문자()에 적합한 이름같은 비 문자 주석, 오탈자 등을 나타내는 것을 말합니다.

사용 사항: {{ HTMLElement("u") }}는 순수하게 꾸미는 요소로서 HTML 4와 XHTML 1에 사장되었으나, HTML5에서 다른 시맨틱 요소와 함께 재정의되었습니다. 만약 의미 없이 문자에 밑줄을 그을 때, {{ HTMLElement("span") }} 요소나 그 외 의미론적으로 적절한 요소에 CSS {{ cssxref("text-decoration") }} 속성의 underline 값으로 꾸며야 합니다.

속성

이 속성은 전역 속성만을 포함합니다.

유의사항

Tip: {{ HTMLElement("u") }} 요소를 하이퍼링크와 헛깔릴 만한 곳에 사용되는 것을 피해야 합니다.

Note: HTML 5 스펙에선 개발자들에게 {{ HTMLElement("u") }}보다 다른 요소들이 거의 아주 적절하다고 상기시켜줍니다.

DOM 인터페이스

이 요소는 HTMLElement 인터페이스를 구현합니다.

구현 사항: Gecko 1.9.2 버전 이상, 파이어폭스는 이 요소에 {{ domxref("HTMLSpanElement") }} 인터페이스를 추가했습니다.

예시

<u>Today's Special</u>: Salmon<br />
<span style="text-decoration:underline;">Today's Special</span>: Salmon
