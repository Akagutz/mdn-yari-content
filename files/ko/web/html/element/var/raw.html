개요

HTML 변수요소 (<var>) 는 수학적 표현이나 프로그래밍 변수를 나타냅니다.

사용 문맥

허용된 컨텐츠 구문 컨텐츠
태그 생략 가능 여부 시작,종료 태그 모두 필요합니다.
허용된 부모 요소  구문 컨텐츠를 허용하는 요소
규범문서 HTML, "The var element"

속성

이 요소는 전역 속성만을 포함합니다.

DOM 인터페이스

이 요소는 HTMLElement 인터페이스를 구현합니다.

구현 참고: Gecko 1.9.2 까지는, 파이어폭스 인터넷 브라우저는 이 요소를 위해 HTMLSpanElement 인터페이스를 구현했습니다.

이 태그의 일반적인  스타일

var {
  font-style: italic; 
}

예제

<p> A simple equation: <var>x</var> = <var>y</var> + 2 </p>
