contenteditable 전역 속성(Global attributes)은 사용자가 요소를 편집 할 수 있는지 여부를 나타내는 열거형 속성입니다. 이 경우 브라우저는 위젯을 수정하여 편집을 허용하도록 합니다. 속성값은 다음 값 중 하나이어야 합니다.

  • true 혹은 빈 문자열 : 요소를 편집 가능하도록 지정합니다.
  • false, 요소를 편집 할 수 없도록 지정합니다.

이 속성을 설정하지 않으면 기본값은 상위 요소에서 상속됩니다.

이 속성은 Boolean 속성이 아니라 열거형 속성입니다. 이는 속성값으로 true, false 또는 빈 문자열 중 하나를 명시적으로 사용하는 것이 필수적이라는 것을 의미하며, <label contenteditable>Example Label</label>와 같은 축약적인 표현은 허용되지 않습니다.  올바른 사용법은 <label contenteditable="true">Example Label</label> 입니다.
