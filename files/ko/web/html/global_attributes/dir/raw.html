dir 전용 속성은 열거형 속성으로 요소의 텍스트에 대한 방향성을 나타냅니다. 이 속성은 다음과 같은 값을 가질 수 있습니다:

  • ltr, 왼쪽에서 오른쪽 방향을 의미하며, 영어와 같이 왼쪽에서 오른쪽으로 기술되는 언어에 사용됩니다;
  • rtl, 오른쪽에서 왼쪽 방향을 의미하며, 아랍어와 같이 오른쪽에서 왼쪽으로 기술되는 언어에 사용됩니다;
  • auto, 사용자 에이전트가 결정하도록 합니다. 이 방식은 요소의 내부에 있는 문자를 강한 방향성(strong directionality)을 갖는 문자를 찾을 때까지 파싱을 계속해 나가며, 찾은 방향성을 요소 전체에 적용하는 기본적인 알고리즘을 사용합니다.

Usage notes

이 속성은 의미적으로 다른 뜻을 가지는 {{ HTMLElement("bdo") }} 요소에 필수 속성입니다.

  • 이 속성은 {{ HTMLElement("bdi") }} 요소의 속성을 상속하지 않습니다. 따로 설정되지 않으면 그 값은 auto입니다.

  • CSS 페이지가 활성화되고 해당 요소가 CSS 속성인 {{ cssxref("direction") }}와 {{ cssxref("unicode-bidi") }}을 지원하는 경우, 이들 속성에 의해 오버라이드될 수 있습니다.

  • 텍스트의 방향은 내용과 의미적으로 관련되고 표시 방식과는 관련이 없으므로 웹 개발자는 가능한 관련 CSS속성 대신 이 속성을 사용하는 것이 좋습니다. 이렇게 하면 CSS를 지원하지 않거나 CSS가 비활성화된 브라우저에서도 텍스트가 올바르게 표시됩니다.

  • auto 값은 사용자 입력 데이터와 같이 데이터의 방향성을 모르며 최종적으로 데이터베이스에 저장되는 데이터일 경우에만 사용되어야 합니다.
