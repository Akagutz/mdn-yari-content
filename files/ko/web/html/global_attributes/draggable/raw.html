draggable 전역 속성은 열거형 속성으로 해당 요소가 {{domxref("HTML_Drag_and_Drop_API","HTML Drag and Drop API")}}를 이용해 드래그(drag)될 수 있는지를 나타냅니다. 이 속성은 다음과 같은 값을 가질 수 있습니다:

  • true, 해당 요소를 드래그할 수 있음.
  • false, 해당 요소를 드래그할 수 없음.

이 속성이 설정되지 않은 경우 기본값은 auto이며, 브라우저에 기본으로 정의된 동작을의미합니다.

이 속성은 열거형(enumerated) 속성이지 부울(Boolean) 속성은 아닙니다. 이는 true 또는 false 값을 명시적으로 사용해야 한다는 것을 의미하며, <label draggable>Example Label</label> 와 같이 축약해서 사용하는 것은 허용되지 않습니다. 이에 대한 정확한 사용법은 <label draggable="true">Example Label</label>입니다.

기본적으로 선택된 텍스트와 이미지 그리고 링크에 대해서만 드래그할 수 있습니다. 모든 다른 요소에 대해 드래그 & 드랍(drag & drop) 매커니즘이 작동하도록 하기 위해서는 종합 예제(comprehensive example)와 같이 {{domxref('GlobalEventHandlers.ondragstart','ondragstart')}} 이벤트가 반드시 설정되어야 합니다.
