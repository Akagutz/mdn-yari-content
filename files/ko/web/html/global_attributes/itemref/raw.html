itemscope 속성을 가진 요소의 하위 요소가 아닌 전역 속성 itemref 프로퍼티는 itemref를 이용하는 아이템과 연관될 수 있습니다.  itemref는 문서 내의 다른 곳에 있는 추가 프로퍼티를 가진 요소의 id(itemid가 아님) 목록을 제공합니다.

itemref 속성은 itemscope 속성이 지정된 요소에 대해서만 지정할 수 있습니다.

참고: itemref 속성은 마이크로데이터 데이터 모델의 포함되어 있지 않습니다.  단지, 편리한 트리 구조(conbenient tree structure)를 따르지 않는 주석을 추가할 데이터가 존재하는 페이지에 주석을 추가는데 작성자에게 도움을 주는 문법적인 구조일 뿐입니다. 예를 들어, 작성자는 각 열이 별도의 아이템을 나타내고 셀이 프로퍼티를 갖도록 정의된 테이블의 데이터에 표식을 남길 수 있습니다.

예제

HTML

<div itemscope id="amanda" itemref="a b"></div>
<p id="a">Name: <span itemprop="name">Amanda</span> </p>
<div id="b" itemprop="band" itemscope itemref="c"></div>
<div id="c">
    <p>Band: <span itemprop="name">Jazz Band</span> </p>
    <p>Size: <span itemprop="size">12</span> players</p>
</div>

구조화 데이터

(JSON-LD 형식)

{
  "@id": "amanda",
  "name": "Amanda",
  "band": {
    "@id": "b",
    "name": "Jazz Band",
    "size": 12
  }
}
