style 전역 속성은 요소에 적용할 CSS 스타일 선언을 담습니다. 스타일은 별도의 파일에 정의하는 것이 권장된다는 점을 참고하세요. 이 속성과 {{HTMLElement("style")}} 요소는 주로 테스트 등 빠른 스타일링을 위한 목적으로 사용됩니다.

{{EmbedInteractiveExample("pages/tabbed/attribute-style.html","tabbed-shorter")}}

The source for this interactive example is stored in a GitHub repository. If you'd like to contribute to the interactive examples project, please clone https://github.com/mdn/interactive-examples and send us a pull request.

참고: 이 속성은 의미 정보를 전달하는데 사용되어선 안됩니다. 모든 스타일링이 제거되더라도, 페이지는 의미적으로 바르게 남아있어야 합니다. 일반적으로 무관한 정보를 숨기는데 사용되지 않아야 합니다; 이건 hidden 속성으로 수행되어야 합니다.
