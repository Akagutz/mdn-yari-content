{{JSRef}}

toTimeString() 메서드는 사람이 읽을 수 있는 형태로 {{jsxref("Date")}}의 시간 부분을 반환합니다.

구문

dateObj.toTimeString()

반환값

주어진 날짜의 시간 부분을 사람이 읽을 수 있는  미국 영어의 문자열.

설명

{{jsxref("Date")}} instances refer to a specific point in time. Calling {{jsxref("Date.prototype.toString()", "toString()")}} will return the date formatted in a human readable form in American English. In SpiderMonkey, this consists of the date portion (day, month, and year) followed by the time portion (hours, minutes, seconds, and time zone). Sometimes it is desirable to obtain a string of the time portion; such a thing can be accomplished with the toTimeString()method.

The toTimeString() method is especially useful because compliant engines implementing ECMA-262 may differ in the string obtained from {{jsxref("Date.prototype.toString()", "toString()")}} for {{jsxref("Date")}} objects, as the format is implementation-dependent; simple string slicing approaches may not produce consistent results across multiple engines.

예제

toTimeString() 사용하기

var d = new Date(1993, 6, 28, 14, 39, 7);

console.log(d.toString());     // logs Wed Jul 28 1993 14:39:07 GMT-0600 (PDT)
console.log(d.toTimeString()); // logs 14:39:07 GMT-0600 (PDT)
