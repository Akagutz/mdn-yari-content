The Error.prototype property represents the prototype for the {{jsxref("Error")}} constructor.

{{js_property_attributes(0, 0, 0)}}

설명

All {{jsxref("Error")}} instances and instances of {{jsxref("Global_Objects/Error", "non-generic errors", "#Error_types", 1)}} inherit from Error.prototype. As with all constructor functions, you can use the prototype of the constructor to add properties or methods to all instances created with that constructor.
