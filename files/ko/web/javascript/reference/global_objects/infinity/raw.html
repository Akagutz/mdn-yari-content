설명

Infinity는 전역 객체의 속성입니다. 즉, 전역 스코프의 변수입니다.

Infinity의 초기값은 {{jsxref("Number.POSITIVE_INFINITY")}}입니다. Infinity(양의 무한대)는 다른 어떤 수보다 더 큽니다. 수학적으로 Infinity는 무한대와 동일하게 행동합니다. 예컨대 임의의 양의 수에 Infinity를 곱하면 Infinity와 같고, 임의의 수를 Infinity로 나누면 0과 같습니다.

Infinity는 ECMAScript 5 명세에 따라 읽기 전용입니다. (JavaScript 1.8.5 / Firefox 4에서 구현)

예제

console.log(Infinity);          /* Infinity */  
console.log(Infinity + 1);      /* Infinity */  
console.log(Math.pow(10,1000)); /* Infinity */  
console.log(Math.log(0));       /* -Infinity */  
console.log(1 / Infinity);      /* 0 */
