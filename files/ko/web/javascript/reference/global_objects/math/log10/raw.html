Math.log10() 함수는 숫자를 log10(숫자)로 반환합니다.

∀x>0,Math.log10(x)=log10(x)=the uniqueysuch that10y=x\forall x > 0, \mathtt{\operatorname{Math.log10}(x)} = \log_10(x) = \text{the unique} \; y \; \text{such that} \; 10^y = x

문법

Math.log10(x)

매개변수

x
숫자.

반환 값

주어진 숫자는 log10(숫자)로 계산합니다. 만약 숫자가 음수라면 NaN를 반환합니다.

설명

만약 x 의 값이 0보다 작다면(음수) 값은 항상 NaN로 반환합니다.

log10()는 Math의 정적 메서드이므로 만든 Math 객체의 메서드가 아니라 항상 Math.log10()함수를 사용해야합니다. (Math는 생성자가 없습니다.)

이 함수는 Math.log(x) / Math.log(10)와 같습니다.

따라서 log10(e)는 Math.LOG10E와 같습니다. 

그리고 위 함수는 1 /{{jsxref("Math.LN10")}}과 같습니다.

예제

Math.log10()

Math.log10(2);      // 0.3010299956639812
Math.log10(1);      // 0
Math.log10(0);      // -Infinity
Math.log10(-2);     // NaN
Math.log10(100000); // 5

Polyfill

This can be emulated with the following function:

Math.log10 = Math.log10 || function(x) {
  return Math.log(x) * Math.LOG10E;
};
