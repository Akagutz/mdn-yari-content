TypeError.prototype 프로퍼티는 {{jsxref("TypeError")}} 생성자의 프로토타입을 의미합니다.

Description

모든 {{jsxref("TypeError")}} 인스턴스들은 TypeError.prototype로부터 상속받습니다. 프로토타입에 다른 프로퍼티나 메소드들을 추가할 수 있습니다.

오류 메세지. 비록 ECMA-262가 {{jsxref("TypeError")}}는 반드시 자체적으로 message 프로퍼티를 제공해야 한다고 기술하지만, SpiderMonkey에서는 {{jsxref("Error.prototype.message")}}를 상속합니다.

Methods

비록 {{jsxref("TypeError")}} 프로토타입 객체가 자체적으로는 아무런 메소드도 가지고 있지 않지만, {{jsxref("TypeError")}} 인스턴스들은 프로토타입 체인(prototype chain)을 통해 메소드들을 상속받을 수 있습니다..
