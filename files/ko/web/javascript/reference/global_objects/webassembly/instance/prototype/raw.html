WebAssembly.Instance.prototype 속성은 {{jsxref("WebAssembly.Instance()")}} 생성자의 프로토 타입을 나타냅니다.

{{js_property_attributes(0, 0, 0)}}

Description

모든 {{jsxref("WebAssembly.Instance")}} 인스턴스는 Instance.prototype에서 상속받습니다. {{jsxref("WebAssembly.Instance()")}} 생성자의 프로토 타입 객체는 모든 {{jsxref( "WebAssembly.Instance")}} 인스턴스에 영향을 미치도록 수정할 수 있습니다.

Properties

Instance.prototype.constructor
이 객체의 인스턴스를 생성 한 함수를 돌려줍니다. 기본적으로 이것은 {{jsxref("WebAssembly.Instance()")}} 생성자입니다.
Instance.prototype.exports {{readonlyinline}}
JavaScript로 액세스하여 사용할 수 있도록 WebAssembly 모듈 인스턴스에서 내 보낸 모든 함수를 멤버로 포함하는 객체를 반환합니다.
