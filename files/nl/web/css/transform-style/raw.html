Samenvatting

De transform-style CSS eigenschap, bepaalt of de kinderen van het element in de 3D ruimte geplaatst worden, of worden samengevoegd met het vlak van het element.

Indien samengevoegd, bestaan de kinderen niet in hun eigen 3D ruimte.

Deze eigenschap wordt niet overgeërfd, en moet dus ingesteld worden voor alle non-leaf ondergeschikte elementen van het element.

{{cssinfo}}

Syntax

/* Keyword waarden */
transform-style: preserve-3d;
transform-style: flat;

/* Globale waarden */
transform-style: inherit;
transform-style: initial;
transform-style: unset;

Waarden

preserve-3d
Geeft aan dat de kinderen van het element in de 3D ruimte moeten geplaatst worden.
flat
Geeft aan dat de kinderen van het element in het vlak van het element zelf liggen.
