W sekcji Instalacja podstawowego oprogramowania, pokażemy ci podstawowe narzędzia jakich potrzeba do stworzenia prostej witryny, oraz jak je właściwie zainstalować.

Jakich narzędzi używają profesjonaliści?

  • Komputer. Być może wydaje się to oczywiste dla wielu ludzi, ale część z was czyta ten artykuł z telefonu, tabletu czy komputera w bibliotece. W przypadku poważnego tworzenia stron internetowych lepiej zainwestować w komputer stacjonarny lub laptop (Windows, Mac, or Linux).
  • Edytor tekstowy, aby pisać w nim kod. To może być edytor tekstowy (e.g. Brackets, Atom or Visual Studio Code), lub edytor hybrydowy (e.g. Dreamweaver). Biurowe edytory tekstu (np. Word) nie nadają się do tworzenia kodu, ponieważ używają wiele ukrytych elementów formatujących wpisywany tekst co zakłóca potem przetwarzanie tak stworzonego pliku przez silniki przeglądarek internetowych.
  • Przeglądarka internetowa, aby testować w niej kod. Obecnie najczęściej używanymi przeglądarkami internetowymi są Firefox, Chrome, Opera, Safari, Internet Explorer and Microsoft Edge. Powinieneś także testować jak twoja strona prezentuje się na urządzeniach mobilnych oraz starszych przeglądarkach, których wciąż mogą używać twoi odbiorcy (takich jak IE 8–10.)
  • Edytor graficzny, taki jak GIMP, Paint.NET, lub Photoshop, by tworzyć obrazy na potrzeby twojej strony.
  • System kontroli wersji, do zarządzania plikami na serwerach, zepołowej współpracy nad projektem, udostępniania kodu i zasobów, oraz unikania konfiktów w edycji. Obecnie Git jest najpopularniejszym narzędziem systemu kontroli wersji, a GitHub również bardzo popularną, opartą o Git, usługą hostingową dla kodu.
  • Program FTP, używany do transferu plilkó dla starszych wersji usług hostingowych (Git w coraz większym stopniu zastępuje FTP w tym celu). Istnieje mnóstwo dostępnych programów (S)FTP jak na przykład Cyberduck, Fetch, and FileZilla.
  • System automatyzacji taki jak Grunt czy Gulp do automatycznego wykonywania powtarzalnych czynności, np. minimalizacji kodu i uruchamiania testów.
  • Szablony, biblioteki, frameworki itp., Aby przyspieszyć pisanie wspólnej funkcjonalności .
  • Oraz wiele więcej narzędzi!

Jakich właściwie narzędzi potrzebuję na teraz?

Wygląda to jak przerażająca lista, ale na szczęście możesz zacząć tworzyć strony internetowe, nie wiedząc nic o większości z nich. W tym artykule przygotujemy dla Ciebie minimum - edytor tekstu i kilka nowoczesnych przeglądarek internetowych.

Instalacja edytora tekstowego

Prawdopodobnie posiadasz już podstawowy edytor tekstowy na swoim komputerze. Takim porstawowym edytorem w systemie Windows jest Notatnik a na macOS TextEdit. W systemach Linux to zależy od wydania; na Ubuntu jest to gedit.

Do tworzenia stron internetowych przydałby ci się jednak lepszy edytor tekstowy. Zalecamy rozpocząć prace z Brackets, który jest darmowym edytorem i oferuje podgląd na żywo tworzonych stron (live preview) oraz podpowiada składnię kodu.

Instalacja współczesnych przeglądarek

Na razie zainstalujemy tylko kilka przeglądarek internetowych, aby przetestować nasz kod. Poniżej wybierz swój system operacyjny i kliknij odpowiedni link aby pobrać plik instalacyjny swoich ulubionych przeglądarek:

  • Linux: Firefox, Chrome, Opera.
  • Windows: Firefox, Chrome, Opera, Internet Explorer, Microsoft Edge (domyślną przeglądarką w Windows 10 jest Edge; jeśli masz Windows 7 lub nowszy, możesz zainstalować Internet Explorer 11; w przeciwnym wypadku powinieneś zainstalować alternatywne przeglądarki)
  • Mac: Firefox, Chrome, Opera, Safari (w systemach iOS and macOS domyślną przegląradką jest Safari)

Zanim przejdziesz dalej, zainstaluj co najmniej dwie z tych przeglądarek, tak aby były już gotowe do testów.
