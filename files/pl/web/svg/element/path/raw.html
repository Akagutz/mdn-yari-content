Element <path> SVG służy do definiowania kształtu. Za pomocą tego element można utworzyć wszystkie podstawowe kształty.
 
Pierwsze kroki
Ten samouczek pomoże Ci zacząć używać ścieżek SVG.

DOM model

Ten element jest implementowany z {{domxref("SVGPathElement")}}.

Przykład

SVG

<svg width="100%" height="100%" viewBox="0 0 400 400"
     xmlns="http://www.w3.org/2000/svg">

  <path d="M 100 100 L 300 100 L 200 300 z"
        fill="orange" stroke="black" stroke-width="3" />
</svg>
