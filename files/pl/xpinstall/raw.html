Cross-Platform Install (XPInstall) jest technologią używaną w Mozilla Application Suite, Mozilli Firefox, Mozilli Thunderbird oraz innych aplikacjach bazujących na XUL-u służącą do instalacji rozszerzeń. Moduł instalacyjny XPI (czytany "zippy" i wywodzący się od XPInstall) jest plikiem ZIP zawierającym w głównym katalogu paczki skrypt instalacyjny lub manifest (w formie pliku install.js lub install.rdf).

Jak dodać moduł instalacyjny do okna instalacji
An installer module is a package in Phil Katz zip format that contains an install script (entitled install.js) at the root of an XPI. The package contains some files and the install script contains some javascript directives for actions to take during an install including adding files and directories, removing old or obsolete file and directories, executing command line tools, etc.
