Gecko pozwala zewnętrznym programistom na używanie technologii tworzonych przez Mozillę. Oznacza to, że możesz zagnieździć przeglądarkę w swojej aplikacji, otwierając kanały i strumienie przez warstwę sieciową, operować na drzewie DOM i tak dalej. Możesz nawet stworzyć całkiem nową aplikację używając chrome.

There's an improved embedding API in early development - see {{ interwiki('wikimo', 'Embedding/NewApi', 'Embedding/NewApi') }}
If you're building an application that uses Mozilla technology, we'd love to hear about it!
Check out mozilla.org/poweredby for more information.
