Plugin para Wordpress

O Promova o MDN plugin para Wordpress blogs são válidos para ajudar pessoas que blogam sobre tecnologias web ou Mozilla-relacionados produtos os dois promovem os seus blogs tão bem quanto o MDN. Isso funciona de duas maneiras:

  1. Quando você publica no seu blog, ele automaticamente faz o link de termos chave (como "HTML", "JavaScript", e "CSS", entre outros) para as páginas apropriadas no MDN. Isso ajuda você a escrever sobre tecnologias web e automaticamente faz o link para documentos relevantes sem precisar ser feito nada por você. Isso é extremamente útil! Os termos e suas páginas destinos estão no MDN e são configurados usando páginas especiais aqui no MDN.
  2. Um nota para caixa "Notifique o Mozilla dessa publicação" é adicionada a publicação editando páginas; essa opção envia um email para os times PR e MDN no Mozilla, então nós podemos, se apropriado, mencionar sua publicação no blog com material promocional, twitar sobre isso, e mais. Verificando a caixa não garante uma menção, mas isso irá certamente chamar atenção das pessoas certas!

A instalação desse plugin ajuda você e o MDN, então se você não o tem instalado, você deve! Se você precisa de ajuda com o código do plugin, vá até ao wp-promote-mdn GitHub repo.

JavaScript snippet

A recomendação automaticamente faz o link de palavras-chave para as páginas MDN e promove o plugin MDN Wordpress, mas se seu site ou blog não roda no Wordpress, você pode colocar um código JavaScript para fazer a mesma função. Esse script faz o mesmo que o plugin para Wordpress; ele converte palavras-chave na página para links para os documentos do MDN.

Você pode configurar um máximo número de palavras-chaves que tornam-se links mudando o valor do ofmaxLinks na opção de objetos. Você pode também especificar os elementos para pesquisa, a classe para uso do creador de links, e o conjunto de palavras-chave e URLs locais para pesquisa.

Para usar o snippet, só adicione isso antes da tag de fechamento </body> no final de seu conteúdo.

<script>
    (function (d,f,a) {
        // Your settings here
        var o = {maxLinks: 4, searchElements: ['div', 'h'], linkClass: 'link-to-mdn', extraLinks: {'keywordx': 'http://example.com'}};

        var s=d.createElement("script");s.type="text/javascript";if(s.f)
        {s.a=function(){if(s.f=="loaded"||s.f=="complete")
        {s.a=null;PromoteMDN(o)}}}else{s.onload=function()
        {PromoteMDN(o)}}s.src="https://raw.githubusercontent.com/riverspirit/promote-mdn-script/master/promote-mdn.js";
        d.getElementsByTagName("head")[0].appendChild(s)
    })(document,'readyState','onreadystatechange');
</script>

Esse snippet carrega o código primário que faz todo trabalho real do Github. Se você prefere ser host do código em seu servidor (para segurança, performance, etc.) só faça o download do código pelo Github, adicione isto em seu site, e ajuste a URL na linha 9 do código acima para finalizar.
