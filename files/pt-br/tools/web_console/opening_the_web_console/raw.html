Para abrir o Web Console:

  • selecione "Web Console" no submenu Web Developer no menu do Firefox (ou no menu Ferramentas se você exibir a barra de menus ou estiver no Mac OS X)
  • ou pressione o atalho do teclado Ctrl + Shift + K (Command + Option + K no OS X).

As Ferramentas vão aparecer na parte inferior da janela do navegador, com o Web Console ativado (é chamado apenas de "Console" nas Ferramentas):

A interface do Console é dividida em três sessões:

  • Ferramentas: na parte superior, há uma barra de ferramentas contendo dois botões. Clique no botão de lata de lixo para limpar o conteúdo do Console. Clique no ícone do funil para filtrar as mensagens exibidas no Console
  • Linha de Comando: a linha de comando começa com dois sinais de maior. Use-a para inserir comandos JavaScript
  • Mensagens do Console: as mensagens geradas pelo código na página e pelos comandos digitados na linha de comando são exibidos após cada comando

Observação: Você pode limpar o conteúdo do Console pressionando o atalho Ctrl + L (Windows, macOS, e Linux) ou Cmd + K no macOS.
