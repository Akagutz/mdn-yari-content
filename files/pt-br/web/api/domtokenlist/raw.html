{{APIRef("DOM")}}

A interface DOMTokenList  representa um set de tokens separados por espaço. Tais sets são retornados por {{domxref("Element.classList")}}, {{domxref("HTMLLinkElement.relList")}}, {{domxref("HTMLAnchorElement.relList")}} ou {{domxref("HTMLAreaElement.relList")}}. É indexada começando com 0 assim como os objetos tipo {{jsxref("Array")}} JavaScript. DOMTokenList é sempre case-sensitive.

Properties

{{domxref("DOMTokenList.length")}} {{ReadOnlyInline}}
É um inteiro representando o número de objetos armazenados no objeto.
{{domxref("DOMTokenList.value")}} {{ReadOnlyInline}}
Retorna o valor da lista como um {{domxref("DOMString")}}.

Methods

{{domxref("DOMTokenList.item()")}}
Retorna um item da lista pelo seu index (retorna undefined se o número é maior ou igual ao comprimento da lista).
{{domxref("DOMTokenList.contains()")}}
Retorna true se a lista contém o token especificado, caso contrário false.
{{domxref("DOMTokenList.add()")}}
Adiciona o token em especificado à lista.
{{domxref("DOMTokenList.remove()")}}
Remove o token especificado da lista.
{{domxref("DOMTokenList.replace()")}}
Substitui  um token exsitente por um novo token.
{{domxref("DOMTokenList.supports()")}}
Retorna true se um token especificado está na lista de tokens suportados pelos atributos associados.
{{domxref("DOMTokenList.toggle()")}}
Remove um determinado token da lista e retorna false. Se o token não esiste, é adicionado e a função retorna true.
{{domxref("DOMTokenList.entries()")}}
Retorna um {{jsxref("Iteration_protocols","iterator")}} permitindo que se percorra todas os pares chave/valor contidos nesse objeto.
{{domxref("DOMTokenList.forEach()")}}
Executa uma função especificada para cada elemento no DOMTokenList.
{{domxref("DOMTokenList.keys()")}}
Retorna um {{jsxref("Iteration_protocols", "iterator")}} permitindo que se percorra todas as chaves dos pares chave/valor contidos nesse objeto.
{{domxref("DOMTokenList.values()")}}
Retorna um {{jsxref("Iteration_protocols", "iterator")}} permitindo que se percorra todos os valores dos pares chave/valor contidos nesse objeto.

Examples

In the following simple example we retrieve the list of classes set on a {{htmlelement("span")}} element as a DOMTokenList using {{domxref("Element.classList")}}, add a class using {{domxref("DOMTokenList.add()")}}, and then update the {{domxref("Node.textContent")}} of the <span> to equal the DOMTokenList.

First, the HTML:

<p class="a b c"></p>

Now the JavaScript:

var para = document.querySelector("p");
var classes = para.classList;
para.classList.add("d");
para.textContent = 'paragraph classList is "' + classes + '"';

Trimming of whitespace and removal of duplicates

Methods that modify the DOMTokenList (such as {{domxref("DOMTokenList.add()")}}) automatically trim any excess whitespace and remove duplicate values from the list. For example:

<span class="    d   d e f"></span>
var span = document.querySelector("span");
var classes = span.classList;
span.classList.add("x");
span.textContent = 'span classList is "' + classes + '"';
