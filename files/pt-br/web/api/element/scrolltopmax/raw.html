OElement.scrollTopMax com a propriedade read-only retorna {{jsxref("Number")}} representando o scroll máximo para o elemento.

Sintaxe

var pxl = elt.scrollTopMax;

Especificações

Essa propriedade não faz parte de nenhuma especificação.
