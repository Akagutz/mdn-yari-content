A propriedade read-only NavigatorLanguage.languages retorna um  array de {{domxref("DOMString")}}s representando as preferências de idioma do usuario. O idioma é representado usando tags de idioma BCP 47. No array retornado, são ordenados por prioridade os idomas de preferencia. 

The value of {{domxref("NavigatorLanguage.language","navigator.language")}} is the first element of the returned array [3]. (But it refers to the system language setting.)

When its value changes, as the user's preferred languages are changed a {{event("languagechange")}} event is fired on the {{domxref("Window")}} object.

The Accept-Language HTTP header in every HTTP request from the user's browser uses the same value for the navigator.languages property except for the extra qvalues (quality values) field (e.g. en-US;q=0.8).

Syntax

preferredLanguages = globalObj.navigator.languages

Examples

navigator.language   //"en-US"
navigator.languages  //["en-US", "zh-CN", "ja-JP"]
