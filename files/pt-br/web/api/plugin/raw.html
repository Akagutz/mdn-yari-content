A interface do Plugin dá informações sobre plugin ao navegador.

Propiedades

Nome Descrição Tipo de retorno Availability
{{domxref("Plugin.description")}} Descrição legível do plugin. Apenas leitura. {{domxref("DOMString")}} DOM 0
{{domxref("Plugin.filename")}} Nome do arquivo na pasta. Apenas leitura. {{domxref("DOMString")}} DOM 0
{{domxref("Plugin.name")}} Nome do plugin. Apenas leitura. {{domxref("DOMString")}} DOM 0
{{domxref("Plugin.version")}} Versão do plugin. Apenas Leitura. {{domxref("DOMString")}} Somente para Firefox 4 ou superior

Metodos

{{domxref("Plugin.item")}}
Returns the MIME type of a supported content type, given the index number into a list of supported types.
{{domxref("Plugin.namedItem")}}
Returns the MIME type of a supported item.
