Sintaxe

A propriedade background-color é especificada como um único valor de <color> .

Valores

{{cssxref("<color>")}}
É uma cor CSS {{cssxref("<color>")}} que descreve a cor uniforme do plano de fundo. Mesmo se um ou vários {{cssxref("background-image")}} são definidos, essa cor pode afetar a renderização, por trânsparência se as imagens não forem opacas. No CSS, transparent é uma cor.

  color: white;
