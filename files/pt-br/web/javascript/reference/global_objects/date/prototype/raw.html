A propriedade Date.prototype representa o protótipo para o construtor {{jsxref("Date")}}.

{{js_property_attributes(0, 0, 1)}}

Description

As instâncias JavaScript {{jsxref("Date")}} herdam de Date.prototype. Você pode modificar o objeto do protótipo do construtor para alterar as propriedades e métodos herdados pelas instâncias de JavaScript {{jsxref("Date")}}.

Para compatibilidade com cálculos do milênio (em outras palavras, para levar em conta o ano 2000), você sempre deveria especificar o ano de forma completa; por exemplo, usar 1998 e não apenas 98. Para ajudar a especificar o ano de forma completa, o JavaScript inclui os métodos {{jsxref("Date.prototype.getFullYear()", "getFullYear()")}}, {{jsxref("Date.prototype.setFullYear()", "setFullYear()")}}, {{jsxref("Date.prototype.getUTCFullYear()", "getUTCFullYear()")}} e {{jsxref("Date.prototype.setUTCFullYear()", "setUTCFullYear()")}}.

Começando no ECMAScript 6, o objeto Date.prototype é ele mesmo um objeto ordinário. Não é uma instância de {{jsxref("Date")}}.
