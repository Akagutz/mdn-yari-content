O método sub() cria um elemento HTML  {{HTMLElement("sub")}} que faz com que a string  seja exibida como subscript.

Sintáxe

str.sub()

Valor Retornado

Uma string contendo um elemento HTML {{HTMLElement("sub")}} .

Descrição

O método sub() embute uma string em uma tag <sub>: "<sub>str</sub>".

Exemplos

Usando os métodos sub() e sup()

Os exemplos seguintes usam o métodos sub() e {{jsxref("String.prototype.sup()", "sup()")}} para formatar uma string:

var superText = 'superscript'; 
var subText = 'subscript'; 

console.log('This is what a ' + superText.sup() + ' looks like.'); 
// This is what a <sup>superscript</sup> looks like 

console.log('This is what a ' + subText.sub() + ' looks like.'); 
// This is what a <sub>subscript</sub> looks like.
