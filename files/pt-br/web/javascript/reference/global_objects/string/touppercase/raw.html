O método toUpperCase() retorna o valor da string chamada convertida para maiúscula.

Sintaxe

str.toUpperCase()

Descrição

O método toUpperCase() retorna o valor da string convertida para maiúscula. toUpperCase() não afeta o valor da própria string.

Exemplos

Usando toUpperCase()

console.log('alfabeto'.toUpperCase()); // 'ALFABETO'
