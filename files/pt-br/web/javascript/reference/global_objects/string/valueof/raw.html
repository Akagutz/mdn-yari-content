O método valueOf() retorna o valor primitivo de um objeto {{jsxref("String")}}.

{{EmbedInteractiveExample("pages/js/string-valueof.html")}}

Sintaxe

str.valueOf()

Valor retornado

Uma string representando o valor primitivo de um objeto {{jsxref("String")}}.

Descrição

O método valueOf() de {{jsxref("String")}} retorna o valor primitivo de um objeto {{jsxref("String")}} como um dado do tipo string. Esse valor é equivalente a {{jsxref("String.prototype.toString()")}}.

Esse método é normalmente chamado internamente pelo JavaScript e não fica explicito no código.

Exemplos

Usando valueOf()

var x = new String('Hello world');
console.log(x.valueOf()); // Mostrará 'Hello world'
