Tecnologias da Web

Recomendamos que você já tenha começado sua jornada na web, mas isto não é absolutamente necessário.

HTML — Estruturando a web
A HyperText Markup Language é utilizada para definir e descrever semanticamente o conteúdo de uma página da web seguindo um formato bem-estruturado.
Com HTML é possivel criar documentos estruturados, constituídos por blocos chamados elementos que são delineados por tags, escritos usando os caracteres de menor que (<) e maior que (>). Alguns introduzem conteúdo diretamente na página, outros fornecem informações sobre o texto do documento e podem incluir outras tags como sub-elementos. Obviamente os navegadores não os exibem, pois são usados para interpretar o conteúdo da página.
