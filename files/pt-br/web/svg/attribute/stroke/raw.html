O atributo stroke define a cor do contorno em um determinado elemento gráfico. O valor padrão do atributo stroke é none.

Contexto de Uso

Categorias Atributo de presentação
Valor <paint>
Animável Sim
Documento Normativo SVG 1.1 (Segunda Edição)

​

Linha em SVG com stroke

Exemplo 1: Desenhar linha reta verde usando stroke.

<svg height="50" width="300">
    <path stroke="green" d="M5 20 1215 0" />
</svg>
