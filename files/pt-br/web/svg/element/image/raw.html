{{SVGRef}}

O elemento de imagem do SVG (<image>) permite que uma imagem rasteirizada seja incluída em um documento SVG.

Contexto de uso

{{svginfo}}

Exemplo

Renderização básica de uma imagem PNG em um objeto SVG:

<svg width="100%" height="100%" viewBox="0 0 100 100"
     xmlns="http://www.w3.org/2000/svg" 
     xmlns:xlink="http://www.w3.org/1999/xlink"> 	 		
  <image xlink:href="/files/2917/fxlogo.png" x="0" y="0" height="100" width="100" />  	
</svg>
