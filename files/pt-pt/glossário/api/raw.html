Uma API (Interface de Programação de Aplicação) é um conjunto de funcionalidades e regras que permitem a interação entre software, forncendo a API e outros componentes de software. No desenvolvimento da Web, API normalmente significa um conjunto de {{glossary("method","métodos")}}, {{Glossary("property","propriedades")}}, eventos, e {{Glossary("URL","URLs")}} padrões para interagirem com o conteúdo da Web.
