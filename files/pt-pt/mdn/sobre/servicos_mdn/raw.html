Serviços da MDN, são utilitários experimentais para os programadores da Web projetados para se enquadrarem nos fluxos de trabalho existentes e ajudar os programadores a melhorar o seu código da Web. Todos os serviços da MDN estão nos primeiros estágios alfa de desenvolvimento, portanto, nós não recomendamos que confiem nois mesmos para garantir a qualidade do código. Se gosta de experimentar com protótipos, os serviços abaixo estão prontos para serem testados, e nós gostaríamos de receber os seus comentários sobre eles.

Serviço de Compatibilidade da Web ("Discord")

Discord is a GitHub webhook for catching compatibility issues in your code. It currently scans CSS code with doiuse and adds comments to your code commits on GitHub.
