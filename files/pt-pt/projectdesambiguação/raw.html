Introdução

Desambiguação no wiki MDC é o processo de resolução daambiguidade, referindo-se a conflitos que podem ocorrer quando artigos de dois ou mais tópicos diferentes receberem o mesmo título "natural".

Toda e qualquer página dentro do Wiki deve ter um título único. Isso pode causar problemas quando há duas ou mais páginas que, teoricamente deveriam ter o mesmo título. Por exemplo, indexOf é um método javascript tanto para String quanto para Arrays em Javascript.

NOTA: As seguintes regras foram editadas desde sua primeira publicação.

Como a Wikipedia, nós diferenciamos as páginas colocando qualificadores entre parênteses depois do título principal (ex: "Mercúrio (planeta)"). Usuários procurando pela informação em "indexOf", no entanto, virão ao MDC e efetuarão a busca digitando "indexOf" na caixa de texto. A busca retornará resultados para todas as páginas com "indexOf", o que eventualmente pode ser tornar algo desorganizado, dado que isto será mostrado em títulos, artigos, tutoriais, exemplso de código, e referências pelo wiki. De forma a ajudar os leitores a encontrar mais facilmente o que eles estão procurando, nós começamos a adicionar páginas de desambiguação no wiki.

Quando criar uma página de desambiguação?

Páginas de desambiguação devem ser criadas apenas para resolver problemas de ambiguidade entre duas ou mais páginas.

Como criar uma página de desambiguação

Páginas de desambiguação são feitas a mão por nossos editores para linkar todas as páginas que o leitor pode encontrar quando estiver procurando por um termo em particular ambíguo. Elas permitem aos leitores visualizar a lista de páginas ambíguas e escolher aquela que mais lhes convém.

Para criar uma página de desambiguação, simplesmente crie a página no qual o título é ambíguo, por exemplo indexOf.

O que incluir em uma página de desambiguação

Quando criar uma página de desambiguação pela primeira vez, você precisa adicionar duas coisas:

Pequena linha introdutória

Adicione uma pequena linha introdutória no topo da página, algo como "indexOf é um método para vários objetos em javascript:", "indexOf é usado de vários modos:", ou, mais simples, "indexOf pode significar várias coisas:".

O template "desambíguo"

Adicione a predefinição Desambiguar no rodapé da página. Isto insere uma nota padrão e a categoria de desambiguação.

Lista de links para os títulos ambíguos

Então simplesmente adicione uma lista de links para as páginas com títulos ambíguos, com um pequeno resumo de cada. Aqui estamos usado o exemplo do indexOf como padrão das explicações.

Se esta lista de links tornar-se desorganizada, você é livre para organizá-la em seções lógicas ou grupos.

Boas Práticas

  1. Páginas de desambiguação devem conter links somente para páginas existentes no wiki.
  2. Não faça links para outras páginas nos resumos de desambiguação. Por exemplo, não façar isso: "indexOf - método para o objeto String no JavaScript". Deixe isto apenas como "indexOf - método para o objeto string no javascript". Links estranhos não servem para o propósito das páginas de desambiguação.
  3. Resuma o título da página, se necessário, para o termo que está sendo desambiguado. Ao invés de linkar para Core JavaScript 1.5 Reference:Objects:String:indexOf, use um texto alternativo como indexOf.
  4. Páginas de desambiguação não são feitas para jogos de "associação livro". Por favor use-as com cuidado e somente quando precisar.

Concertando links para tópicos desambiguados

Depois de você ter criado a página de desambiguação, clique no link "Quais links aqui" para ver se alguma página existente está linkando para sua nova página de desambiguação. Se há páginas linkando diretamente para a página de desambiguação, por favor mude os links destas páginas, onde possível, de forma que eles apontem para a página correta no wiki.

Por exemplo, se você tem uma página falando sobre o método indexOf do objeto String do Javascript linke-a para a página de desambiguação indexOf este link deve ser mudando, então, de forma que aponte diretamente para a página Core JavaScript 1.5 Reference:Objects:String:indexOf.

Se você estiver criando uma página de desambiguação que envolverá mover uma página existente clique em "Quais links aqui" para encontrar todas as páginas que linkam para esta que você está querendo mover. Tenha certeza que estas páginas serão corrigidas ou que elas não serão afetadas, antes de fazer a modificação.

O código de honra para criação de páginas de desambiguação é concertar todos os links de outras páginas envolvidas
