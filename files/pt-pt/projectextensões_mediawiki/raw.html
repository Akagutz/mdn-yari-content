Esta página lista as extensões do MediaWiki instaladas como parte do MDC wiki, e descreve como usá-las onde são requeridas.

Extensão de Navegação Breadcrumbs

Os wikis MDC possuem uma extensão personalizada que permite a geração automática de uma string de navegação em uma página. Para usá-la, adicione simplesmente o seguinte elemento (com as tags de abertura e fechamento) à sua página do wiki:

<breadcrumbs></breadcrumbs>

Notas

  1. A extensão breadcrumbs não funciona quando é adicionada como um único elemento vazio (ex: <breadcrumbs />). As tags de abertura e fechamento são necessárias.

Extensão Title Override

A extensão Title Override (cortesia de Shaver) foi instalada. Esta extensão nos permite indicar o título com underscores (o que não é possível na instalação padrão do MediaWiki).

Esta extensão muda somente o título da página exibido na parte de navegação do Breadcrumb, inserido com a extensão de navegação Breadcrumbs. Nós estaremos trabalhando para modificar o tema do wiki de modo a permitir underscores no título da página através dessa extensão.

Para usar a extensão title-override, simplesmente adicione o título como deve ser exibido na página do wiki dentro das tags "title-override", como a seguir:

<title-override>MAX_VALUE</title-override>

O valor da extensão title-override substitui o título normal indicado para a página, e não é processado para normalizar o caso ou remover caracteres especiais (como _).

Você pode ver esta extensão em uso na página MAX_VALUE, como um exemplo.

Notas

  • Devido às limitações técnicas com a atualização para o MediaWiki1.5, esta extensão foi alterada de <title> para <title-override>. Se você encontrar alguma página com a versão antiga desta extensão, sinta-se livre para atualizar para a versão atual.

Extensão RSS Feed

A extensão PHP MagpieRSS e a extensão MediaWiki Mafs/RSS estão instaladas.

Uso

charset=
O charset usado pelo feed, iconv é usado para convertê-lo.
short
Não exibe o texto das tags description/summary de cada item. ("long" é o padrão).
max=X
Exibe os X últimos itens.
highlight= termo1 termo2
Os termos separados por um espaço são destacados na exposição.
filter= termo1 termo2
Exibe apenas os itens que contêm ao menos um dos termos especificados.
reverse
Exibe os itens em ordem reversa.

Exemplos

<rss>http://www.w3.org/2000/08/w3c-synd/home.rss|charset=utf-8|short|max=5</rss>
