Este guia do escritor é nosso "estilo da casa" para o Mozilla Developer Center. Pretende-se com este documento ter um guia amigável ao invés de um conjunto restrito de regras. Entretanto não se surpreenda caso um colaborador venha a editar seu trabalho para deixá-lo conforme este guia.

Capitalização

Nomes de páginas devem seguir uma regra de capitalização de forma que apenas sua primeira palavra seja capitulada:

  • Correto: "Um novo método para criar rollovers Javascript"
  • Incorreto: "Um Novo Método Para Criar Rollovers Javascript"
Por enquanto estas regras de capitalização se aplicam somente à páginas novas. Há um grande número de páginas em nosso Wiki que não estão conformes com estas regras, mas por favor não corrija-as por enquanto. Nós iremos corrigir isto futuramente.

Acrônimos e abreviações

Capitalização e pontos

  • Use letras capitais e apague os pontos em todos os acrônimos e abreviações, incluindo países como "EUA" e organizações como "ONU".
    • Correto: XUL
    • Incorreto: X.U.L.; Xul

Expansões

  • Ao mencionar um termo pela primeira vez numa página, introduza a sua expansão entre parênteses.
    • Correto: "XUL (XML User Interface Language) é a linguagem da Mozilla baseada em XML..."
    • Incorreto: "XUL é a linguagem da Mozilla baseada em XML..."
Exceções
  • Prefira não expandir os acrônimos muito comuns aos utilizadores, salvo se estiver em dúvida desta informação.
    • Correto: A XML é uma linguagem de marcação de uso geral...
    • Incorreto: A XML (eXtensible Markup Language) é uma linguagem de marcação de uso geral...

Plurais de acrônimos e de abreviações

  • Para plurais de acrônimos ou abreviações, adicione s.
    • Correto: CD-ROMs
    • Incorreto: CD-ROM's

Números e numerais

Datas

  • Para datas (não incluindo datas em amostras de código) use o formato: 1º de Janeiro de 1980.
    • Correto: 24 de Fevereiro de 2006, 24/02/2006
    • Incorreto: Fevereiro 24, 2006; 02/24/2006
  • Como alternativa, pode usar o formato YYYY/MM/DD.
    • Correto: 2006/02/24
    • Incorreto: 02/24/2006; 02/24/06

Décadas

  • Para décadas, use o formato: 1990s.
    • Correto: 1990s
    • Incorreto: 1990's

Plurais de numerais

  • Para plurais de numerais adicione s.
    • Correto: 486s
    • Incorreto: 486's

Pontos e vírgulas

  • Em texto corrente use ponto separador em números com mais de três dígitos, agrupando-os de três em três, a contar pela direita. A vírgula deve ser usada somente em números que contenham casas decimais diferentes de zero.
    • Correto: 4.000 ; 54.157.140 ; 0,47
    • Incorreto: 4,000 ; 54157140 ; 0,4700

Pontuação

Separação com vírgula

  • Use vírgula para separar três itens ou mais em uma frase. Antes do último item use o separador "e" / "ou". Atente-se para o "eu" sempre como último dos itens, quando existir.
    • Correto: Participaram apenas três pessoas: João, Maria e eu.
    • Incorreto: Participaram apenas três pessoas: eu, João, e Maria.
