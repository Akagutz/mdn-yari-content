O wiki MDC tem a habilidade de redirecionamento para sites externos. Nós fizemos isso para que possamos ter páginas com links externos dentro do wiki que serão incluidas nas páginas de categorias

Criando um redirecionamento externo

Para criar um redirecionamento externo, faça o seguinte:

Primeiramente, crie a página que servirá com redirecionamento externo, colocando o sufixo "(externo)" em seu título. Por exemplo: Mozilla Project Wiki (externo). Este sufixo "(externo)" é usado com um indicador para o usuário de que o link que está clicando vai levá-lo para uma página fora do wiki MDC. Iso é para ser usado primariamente nas listas de categorias, onde a linkagem manual não é possível. Veja abaixo maiores informações sobre como criar páginas de redirecionamento externo.

Depois, edite aquela página e adicione o comando de redirecionamento, seguido pela URL encapsulada entre dois colchetes. Por exemplo:

 #RE‌DIRECT [[http://wiki.mozilla.org]]

Finalmente, adicione uma categoria para a página de redirecionamento. Note que a categoria deve estar na mesma linha que o comando de redirecionamento, ou o parser irá ignorá-lo. Por exemplo:

 #RE‌DIRECT [[http://wiki.mozilla.org]] [[Categoria:Projeto Mozilla]]

Quando terminar, simplesmente salve a página normalmente

Linkando manualmente para redirecionamentos externos

Você pode não criar links de redirecionamento externo. No entanto, apenas criar links externos regulares para o site. A razão para isso é porque linkar para um redirecionamento externo deixa a URL obscura e o usuário não saberá para onde ela aponta. Isso pode ser muito bom para usuários anônimos, mas usuários anônimos não são nada bons.

Migrando páginas referentes a redirecionamentos externos

Se você migrar um artigo no MDC cheque a página da categoria para ver se há algum redirecionamento externo postado.

Se houver, cheque "Artigos Relacionados" (link na coluna da direita, em Toolbox ) para ver se alguma outra página linka para aquela de redirecionamento externo. Se algum link existir, mude-os de forma a apontar para o novo artigo migrado.

Quando nenhuma página linka para um redirecionameno externo, redirecione-o para o novo artigo migrado e coloque-o na categoria "Sucata" (usando [[Category:Sucata]]) então ela pode ser deletada.
