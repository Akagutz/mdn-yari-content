Esta página descreve o 'Depurador de JavaScript' como este aparece no Firefox 52 e superior, no Firefox Nightly e Firefox Developer Edition.

Para ver o que é como em versões anteriores do Firefox ou no Firefox Beta e Release, consulte Depurador (antes de Firefox 52).

Se está a receber esta versão do depurador e precisa de voltar para a versão antiga, pode fazê-lo visitando about: config e definir a preferência "devtools.debugger.new-debugger-frontend" para false.

{{EmbedYouTube("QK4hKWmJVLo")}}

O 'Depurador de JavaScript permite-lhe percorrer o código JavaScript e examinar ou modificar o seu estado para ajudar a rastrear erros.

Pode utilizá-lo para depurar o código executado localmente no Firefox ou em execução remotamente, por exemplo, um dispositivo Firefox no Android ouFirefox OS. Consulte rdepuramento remoto para aprender em como ligar o depurador a um destino remoto.
