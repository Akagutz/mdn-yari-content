Pode utilizar as ferramentas de desenvolvimento do Firefox no seu pc para depurar código executado remotamente: num processo diferente no mesmo dispositivo ou num dispositivo completamente diferente. Para o fazer, utilize o Firefox para anexar a Caixa de Ferramentas ao processo remoto, e a caixa de ferramentas é depois executada na sua própria janela. De momento, as seguintes ferramentas suportam a depuração remota:

  • Firefox para PC
  • Firefox para Android
    • USB
    • Wi-Fi
  • Firefox OS
    • USB
    • Wi-Fi
  • Thunderbird

Other runtimes

The experimental Valence add-on enables you to debug Web sites in runtimes that are not based on Gecko, like Google Chrome Desktop, Chrome on Android, and Safari on iOS.

Valence está incluido por preefinição na Edição do Programador do Firefox.

  • Chrome para Android
  • Chrome PC
  • Safari no iOS
