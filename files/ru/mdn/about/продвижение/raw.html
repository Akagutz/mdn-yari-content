Любите MDN? Помогите поделиться крутостью Открытого Интернета с другими веб-разработчиками с помощью материалов продвижения Сети Разработчиков Mozilla! Обои, наклейки, плакаты, анимации, значки сайтов, вы найдёте их здесь.

Хотите узнать больше о MDN или как помочь сделать хороший вики-справочник для веб-технологий?

WordPress плагин

Плагин продвижения MDN для WordPress блогов доступен, чтобы помочь людям, ведущим блоги о веб-технологиях или о продуктах, связанных с Mozilla, как для продвижения собственных блогов, так и самой MDN. Он работает в двух направлениях:

  1. Когда вы публикуете пост в блоге, он автоматически связывает ключевые термины (такие как “HTML”, “JavaScript”, и “CSS”, среди других) с соответствующими страницами на MDN. Это позволяет вам писать о веб-технологиях и автоматически ссылаться на актуальную документацию без необходимости делать что-либо самостоятельно. Это невероятно полезно! Термины и их назначенные страницы на MDN сконфигурированы с помощью специальной страницы на MDN.
  2. Специальный “Notify Mozilla of this post” флажок добавлен для размещения редактированных страниц; это опция отправляет письмо PR и MDN команде Mozilla, так что мы можем, если необходимо, упомянуть ваш пост в блоге в продвигаемом материале, твитнуть об этом, и так далее. Установка данного флажка не даёт гарантии об упоминании, но это безусловно привлечёт к вам внимание нужных людей!

Установка этого плагина полезна как для вас, так и для MDN, так что если вы ещё не установили его, то попробуйте! Если вы хотите помочь написанию кода для этого плагина, перейдите по ссылке wp-promote-mdn GitHub repo.

JavaScript фрагмент

Рекомендуемый сбособ для автоматического ссылания ключевых слов к страницам MDN - это Promote MDN WordPress плагин приведённый выше, но если ваш сайт или блог не работает на WordPress, вы можете встроить JavaScript код приведённый внизу для того, чтобы сделать тоже самое. Данный скрипт может то же самое, что и плагин WordPress. Он преобразует известные ключевые слова в содержании страницы в ссылки на подходящую документацию MDN.

Вы можете сконфигурировать максимальное число ключевых слов, которые превращаются в ссылки изменив значение maxLinks в опциях объекта. Также вы можете указать элементы для поиска, класс для использования создаваемых ссылок, а также набор дополнительных ключевых слов и целевых URL адресов для сканирования.

Для использования этого отрывка кода, просто вставьте его прямо перед закрывающемся </body> тегом в конце страницы.

<script type="text/javascript">
    (function (d,f,a) {
        // Ваши настройки
        var o = {maxLinks: 4, searchElements: ['div', 'h'], linkClass: 'link-to-mdn', extraLinks: {'keywordx': 'http://example.com'}};

        var s=d.createElement("script");s.type="text/javascript";if(s.f)
        {s.a=function(){if(s.f=="loaded"||s.f=="complete")
        {s.a=null;PromoteMDN(o)}}}else{s.onload=function()
        {PromoteMDN(o)}}s.src="https://raw.githubusercontent.com/riverspirit/promote-mdn-script/master/promote-mdn.js";
        d.getElementsByTagName("head")[0].appendChild(s)
    })(document,'readyState','onreadystatechange');
</script>

Этот отрывок кода загружает основной код, который делает всю работу с Github. Если вы предпочитаете хостить этот код на вашем сервере самостоятельно (для безопасности, производительности, и т.д), просто скачайте код из Github, добавьте его на ваш сайт, и измените URL на строке 9 вышеуказанного кода.

Настольные обои

Впервые создал Sean Martell в 2006 на Firefox Developer's Summit, основываясь на идее Chris Beard, эта графика стала хитом на конференции SXSW в 2007. Сейчас доступна, как обои для рабочего стола в различных вариациях размеров:

  • 800x600
  • 1024x768
  • 1280x1024
  • 1440x900
  • 1680x1050

Эти обои доступны для использования в соответствии с условиями лицензии Creative Commons Attribution-NonCommercial. Эти обои также покрываются Mozilla Trademark Policy.
