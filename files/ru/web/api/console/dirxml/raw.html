{{APIRef("Console API")}}

Отображает дочерние елементы указаного XML/HTML в виде интерактивного дерева. Если переданный обьект не может быть передставлен как дерево елементов, обьек будет отобажен как обьект JavaScript. Вывод представлен иерархическим списоком расширяемых елементов, которые позволяют видеть содержимое дочерних елементов.

 

Синтакис

console.dirxml(object);

Параметры

object

Объект JavaScript свойства которого будут отображены.
