Выводит стэк трэйс в  Веб Консоль.

{{AvailableInWorkers}}

Посмотрите Стэк трэйсы в документации {{domxref("console")}} с деталями и примерами.

Синтаксис

console.trace();

Пример

function foo() {
  function bar() {
    console.trace();
  }
  bar();
}

foo();

В консоли следующий трэйс будет отображен так:

bar
foo
<anonymous>
