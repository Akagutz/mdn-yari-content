Событие scroll возникает когда вид документа или элемента прокручивается.

Общая информация

Спецификация
DOM L3
Интерфейс
UIEvent
Всплытие
События не всплывают на элементах, но всплывают к default view когда возникают на document'е
Отменяемое
Нет
Цель
defaultView, Document, Element
Действие по умолчанию
Никакого

Свойства

Property Type Description
target {{readonlyInline}} EventTarget Цель события (the topmost target in the DOM tree).
type {{readonlyInline}} DOMString Тип события.
bubbles {{readonlyInline}} Boolean Whether the event normally bubbles or not.
cancelable {{readonlyInline}} Boolean Whether the event is cancellable or not.
view {{readonlyInline}} WindowProxy document.defaultView (window of the document)
detail {{readonlyInline}} long (float) 0.
