{{APIRef("DOM Events")}}

The MouseEvent.altKey read-only property is a {{jsxref("Boolean")}} indicates if the alt key was pressed (true) or not (false) when the event occurred.

 Синтаксис

var altKeyPressed = instanceOfMouseEvent.altKey

Пример

<html>
<head>
<title>Пример с altKey</title>

<script type="text/javascript">
document.addEventListener('click', function(e) {
  console.log('Нажата клавиша Alt?', e.altKey);
})
</script>
</head>

<body>
<p>
Кликните на любой место на странице и посмотрите результат в консоли
</p>
</body>
</html>
