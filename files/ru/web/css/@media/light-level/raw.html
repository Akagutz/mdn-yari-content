CSS медиа функция light-level может использоваться для проверки уровня освещенности.

Синтаксис

dim
Устройство используется в затемненной обстановке, где чрезмерный контраст и яркость могут отвлекать или создавать неудобства для читателя. Например: ночное время или слабо освещенная внутренняя среда.
normal
Устройство используется в условиях с уровнем освещенности в идеальном диапазоне для экрана и не требует каких-либо особых настроек.
washed
Устройство используется в исключительно ярких условиях, из-за чего экран становится размытым и его трудно читать. Например: яркий дневной свет.

Примеры

Этот код, скорее всего, не будет работать на всех устройствах (совместимость с устройствами низкая).

@media (light-level: normal) {
  p {
    background: url("texture.jpg");
    color: #333;
  }
}

@media (light-level: dim) {
  p {
    background: #222;
    color: #ccc;
  }
}

@media (light-level: washed) {
  p {
    background: white;
    color: black;
    font-size: 2em;
  }
}
