Краткое описание

Свойство CSS transition - это сокращенное свойство для {{ cssxref("transition-property") }}, {{ cssxref("transition-duration") }}, {{ cssxref("transition-timing-function") }}, и {{ cssxref("transition-delay") }}. Оно позволяет определять переходное состояние между двумя состояниями элемента. Различные состояния могут быть определены с помощью псевдоклассов, таких как {{cssxref(":hover")}} или {{cssxref(":active")}} или установлены динамически с помощью JavaScript.

{{cssinfo}}

Синтаксис

/* Применить к 1 свойству */
/* имя свойства | длительность */
transition: margin-left 4s;

/* имя свойства | длительность | задержка */
transition: margin-left 4s 1s;

/* имя свойства | длительность | временная функция | задержка */
transition: margin-left 4s ease-in-out 1s;

/* Применить к 2 свойствам */
transition: margin-left 4s, color 1s;

/* Применить ко всем измененным свойствам */
transition: all 0.5s ease-out;

/* Глобальные значения */
transition: inherit;
transition: initial;
transition: unset;

Заметьте, что порядок значений в свойстве важен: первое значение, которое может быть интерпретировано как время, назначается в {{cssxref("transition-duration")}}, а второе значение, которое может быть интерпретировано как время, назначается в {{cssxref("transition-delay")}}.

Посмотрите, что предпринимается когда у списков значений свойств разные длины. Кратко, избыточные свойства, не подлежащие анимации, игнорируются.
