HTML элементы <h1>–<h6> представляют собой 6 уровней заголовков секций. <h1> это наибольший заголовок и<h6> - наименьший

Content categories Flow content, heading content, palpable content.
Permitted content Phrasing content.
Tag omission {{no_tag_omission}}
Permitted parents Any element that accepts flow content; don't use as a child of {{HTMLElement("hgroup")}} element, it is now deprecated
Permitted ARIA roles {{ARIARole("tab")}}, {{ARIARole("presentation")}}
DOM interface {{domxref("HTMLHeadingElement")}}

Атрибуты

Эти элементы включают глобальные атрибуты.

Атрибут align устаревший; не используйте его.

Примечания к использованию

  • Heading information may be used by user agents, for example, to construct a table of contents for a document automatically.
  • Не используйте низкие уровни чтобы снизить размер шрифта: используйте CSS {{cssxref("font-size")}} вместо.
  • Избегайте пропуск уровней заголовков: всегда начинайте с <h1>, потом используйте <h2>, и так далее.
  • Вам следует рассмотреть избегание использования <h1> более раза на страницу. Смотрите {{SectionOnPage("/en-US/docs/Web/Guide/HTML/Using_HTML_sections_and_outlines", "Defining sections")}}.
