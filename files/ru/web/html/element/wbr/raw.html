Элемент HTML <wbr> представляет возможность разрыва слова - позицию в тексте, где браузер может по желанию разбить строку, хотя в противном случае его правила разрыва строки не будут создавать разрыв в этом месте.

Заметки

 

На страницах в кодировке UTF-8 <wbr> ведет себя как кодовая точка Z + WIDTH SPACE U + 200B. В частности, он ведет себя как код BN Unicode Bidi, то есть он не влияет на порядок двунаправленности: отображаются <div dir = rtl> 123, <wbr> 456 </ div>, если они не разбиты на две строки, 123 456 и не 456,123.

По той же причине элемент <wbr> не вводит дефис в точке разрыва строки. Чтобы дефис отображался только в конце строки, используйте вместо него объект с символом мягкого дефиса (& shy;).

Этот элемент был впервые реализован в Internet Explorer 5.5 и был официально определен в HTML5.

 

Пример

Руководство по стилю Yahoo рекомендует разбивать URL перед пунктуацией, чтобы не оставлять знаки препинания в конце строки, которые читатель может принять за конец URL.

<p>http://this<wbr>.is<wbr>.a<wbr>.really<wbr>.long<wbr>.example<wbr>.com/With<wbr>/deeper<wbr>/level<wbr>/pages<wbr>/deeper<wbr>/level<wbr>/pages<wbr>/deeper<wbr>/level<wbr>/pages<wbr>/deeper<wbr>/level<wbr>/pages<wbr>/deeper<wbr>/level<wbr>/pages</p>
