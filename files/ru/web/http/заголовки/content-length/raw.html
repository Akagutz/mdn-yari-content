Заголовок Content-Length указывает размер тела объекта в десятичном числе октетов (байтов), отправленных получателю.

Тип заголовка Entity header
Можно не передавать да

Синтаксис

Content-Length: <длина>

Параметры

<длина>
Длина в десятичном числе октетов.
