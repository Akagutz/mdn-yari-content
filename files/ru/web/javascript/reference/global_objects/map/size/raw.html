Свойство size возвращает количество элементов в объекте {{jsxref("Map")}}.

Описание

size - целое число, соответствующее количеству элементов в объекте Map. Переопределить это своство нельзя.

Примеры

Используем size

var myMap = new Map();
myMap.set("a", "alpha");
myMap.set("b", "beta");
myMap.set("g", "gamma");

myMap.size // 3
