Сводка

Метод Number.isInteger() определяет, является ли переданное значение целым числом.

Синтаксис

Number.isInteger(value)

Параметры

value
Значение, проверяемое на целочисленность.

Описание

Если целевое значение является целым числом, возвращает true, в противном случае возвращает false. Если значение является {{jsxref("Global_Objects/NaN", "NaN")}} или бесконечностью, возвращает false.

Примеры

Number.isInteger(0.1);     // false
Number.isInteger(1);       // true
Number.isInteger(Math.PI); // false
Number.isInteger(-100000); // true
Number.isInteger(NaN);     // false
Number.isInteger(0);       // true
Number.isInteger('10');    // false

Полифилл

Number.isInteger = Number.isInteger || function(value) {
  return typeof value === 'number'
         && Number.isFinite(value)
         && !(value % 1);
};
