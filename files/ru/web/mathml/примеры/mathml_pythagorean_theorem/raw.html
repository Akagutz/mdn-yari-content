Сейчас мы докажем Теорему Пифагора:

Утверждение: В прямоугольном треугольнике квадрат гипотенузы равен сумме квадратов двух других сторон (катетов).

То есть если a и b - катеты, а c - гипотенуза, тогда a 2 + b 2 = c 2 .

Доказательство: Мы можем доказать теорему алгебраически, показав, что площадь большого квадрата равна площади внутреннего квадрата (квадрата гипотенузы) плюс площадь четырех треугольников: ( a + b ) 2 = c 2 + 4 ⋅ ( 1 2 a b ) a 2 + 2 a b + b 2 = c 2 + 2 a b a 2 + b 2 = c 2
