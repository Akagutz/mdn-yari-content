Аттрибут viewBox позволяет определить что данный набор графических растяжек соответствует конкретному элементу контейнера.

Значение атрибута viewBox - это список из четырех чисел min-x, min-y, width и height, разделённых пробелами и/или запятой, которые определяют прямоугольник в пользовательском пространстве, который должен отображаться на границах области просмотра установленный данным элементом, с учетом атрибута {{SVGAttr ("preserveAspectRatio")}}.

Отрицательные значения для ширины или высоты недопустимы, а значение нуля отключает рендеринг элемента.
