{{JSRef}}

URIError อ็อบเจกต์จะแสดง error เมื่อฟังก์ชั่นการจัดการ URI ถูกใช้ผิดรูปแบบ

Syntax

new URIError([message[,fileName[, lineNumber]]])

Parameters

message
Optional. คำอธิบาย error
fileName {{non-standard_inline}}
Optional. ชื่อไฟล์ที่เกิด exception
lineNumber {{non-standard_inline}}
Optional. เลขบรรทัดที่เกิด exception

Description

URIError จะถูก thrown เมื่อใส่พารามิเตอร์ที่ผิดรูปแบบในฟังก์ชั่นการจัดการ URI

Properties

{{jsxref("URIError.prototype")}}
อนุญาตให้เพิ่ม properties เข้ามาในอ็อบเจกต์ URIError

Methods

The global URIError ไม่มี methods ของตัวเอง แต่สามารถสืบทอด method ผ่าน prototype ได้

Examples

Catching an URIError

try {
  decodeURIComponent('%');
} catch (e) {
  console.log(e instanceof URIError); // true
  console.log(e.message);             // "malformed URI sequence"
  console.log(e.name);                // "URIError"
  console.log(e.fileName);            // "Scratchpad/1"
  console.log(e.lineNumber);          // 2
  console.log(e.columnNumber);        // 2
  console.log(e.stack);               // "@Scratchpad/2:2:3\n"
}

Creating an URIError

try {
  throw new URIError('Hello', 'someFile.js', 10);
} catch (e) {
  console.log(e instanceof URIError); // true
  console.log(e.message);             // "Hello"
  console.log(e.name);                // "URIError"
  console.log(e.fileName);            // "someFile.js"
  console.log(e.lineNumber);          // 10
  console.log(e.columnNumber);        // 0
  console.log(e.stack);               // "@Scratchpad/2:2:9\n"
}
