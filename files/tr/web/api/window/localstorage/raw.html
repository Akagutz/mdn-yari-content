localStorage özelliği, yerel {{domxref("Storage")}} objesine erişim imkanı verir. localStorage özelliği sessionStorage özelliğine benzemektedir. Aralarındaki tek fark, localStorage özelliğinde saklanan verinin sona erme tarihi yok iken, sessionStorage özelliğinde saklanan veri web tarayıcısı oturumu sona erdiğinde "web tarayıcısı kapatıldığında" silinmektedir.

localStorage veya sessionStorage özelliklerinde saklanan verilerin, sayfanın protokolüne özgü olduğuna dikkat edilmelidir.

Söz Dizimi

myStorage = localStorage;

Değer

{{domxref("Storage")}}.

Örnek

The following snippet accesses the current domain's local {{domxref("Storage")}} object and adds a data item to it using {{domxref("Storage.setItem()")}}.

localStorage.setItem('myCat', 'Tom');

Note: Please refer to the Using the Web Storage API article for a full example.
