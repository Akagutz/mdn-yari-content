Редактор WYSIWYG (what-you-see-is-what-you-get — що ти бачиш те ти отримаєш) для вікі MDN Web Docs дозволяє легко створювати новий вміст. У цьому посібнику показано, як використовувати редактор і підвищити вашу продуктивність. Будь ласка, прочитайте та дотримуйтесь умов Mozilla, перш ніж редагувати або створювати нові сторінки.

MDN style guide дає інформацію про форматування та стилістику вмісту, включно з нашими граматичними та правописними правилами.
