{{JSRef}}

The setTime() method sets the {{jsxref("Date")}} object to the time represented by a number of milliseconds since January 1, 1970, 00:00:00 UTC.

Сінтаксис

dateObj.setTime(timeValue)

Параметри

timeValue
Ціле число, що прелставляє кільксть мілісекунд з 1-ого січня 1970, 00:00:00 UTC.

Return value

Кількість мілісекунд між 1-им січня 1970 00:00:00 UTC та оновляємою датой (effectively, the value of the argument).

Опис

Використовуйте setTime() щоб призначити дату та час іншому {{jsxref("Date")}} об'єкту.

Приклади

Застосування setTime()

var theBigDay = new Date('July 1, 1999');
var sameAsBigDay = new Date();
sameAsBigDay.setTime(theBigDay.getTime());
