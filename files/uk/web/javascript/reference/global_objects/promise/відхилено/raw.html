Метод Promise.reject(reason) повертає об'єкт Promise, який відхилений з даною причиною.

Синтаксис

Promise.reject(reason);

Параметри

reason
Причина, чому даний Проміс (Promise) був відхилений.

Значення, яке повертається

{{jsxref("Promise")}}, який був відхилений з даною причиною.

Опис

Статичний метод Promise.reject повертає Promise, який було відхилено. З метою відлагодження а також вибіркового перехоплення помилок, зручно використовувати reason як instanceof {{jsxref("Error")}}.

Приклади

Використання статичного методу Promise.reject()

Promise.reject(new Error('fail')).then(function(error) {
  // не викликається
}, function(error) {
  console.log(error); // Stacktrace
});
