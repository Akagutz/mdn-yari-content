Опис

Стала MAX_SAFE_INTEGER має значення 9007199254740991 (9,007,199,254,740,991 або ~9 квадриліонів). The reasoning behind that number is that JavaScript uses double-precision floating-point format numbers as specified in IEEE 754 and can only safely represent numbers between -(253 - 1) and 253 - 1.

"Безпечний" в даному контексті означає можливість точно відтворювати цілі числа та правильно порівнювати їх. Наприклад, Number.MAX_SAFE_INTEGER + 1 === Number.MAX_SAFE_INTEGER + 2 дасть в результаті true, що, з математичної точки зору, неправильно. Дивіться більше на {{jsxref("Number.isSafeInteger()")}}.

У зв'язку з тим, що MAX_SAFE_INTEGER - це статичне поле {{jsxref("Number")}}, слід завжди викликати його як Number.MAX_SAFE_INTEGER, аніж як метод створеного вами об'єкту {{jsxref("Number")}}.

Приклади

Number.MAX_SAFE_INTEGER // 9007199254740991
Math.pow(2, 53) - 1     // 9007199254740991
