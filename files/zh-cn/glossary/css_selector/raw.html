css 选择器是css规则的一部分，使你能应用样式到指定元素。例如：

***HTML***
<div> I am inside of a div element. </div>
<p> I am inside of a paragraph element. </p>


***CSS***
div {
 color: green;
}

p {
 color: red;
}


/* data-*="" */

/* <p data-test*="test abc xyz" /> */

[data-test*="test"]
