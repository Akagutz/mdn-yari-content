{{APIRef("DOM")}}

Attr.localName为只读属性，返回the local part of the qualified name of an element。

在DOM4之前此API被定义在{{domxref("Node")}}接口里。

语法

name = attribute.localName

返回值

A {{domxref("DOMString")}} representing the local part of the attribute's qualified name.

例子

下列例子将在alert中显示"id"。

HTML部分

<button id="example">Click me</button>

JavaScript部分

const element = document.querySelector("#example");
element.addEventListener("click", function() {
  const attribute = element.attributes[0];
  alert(attribute.localName);
});

注意

The local name of an attribute is the part of the attribute's qualified name that comes after the colon. Qualified names are typically used in XML as part of the namespace(s) of the particular XML documents.

注意: 在{{Gecko("1.9.2")}}或更早的版本，这个属性返回大写的the local name for HTML attributes in HTML DOMs (as opposed to XHTML attributes in XML DOMs). In later versions, in compliance with HTML5, the property returns in the case of the internal DOM storage, which is lower case for both HTML attributes in HTML DOMs and XHTML attributes in XML DOMs.
