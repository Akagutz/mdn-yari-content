Clipboard API的Clipboard接口提供了一种读写操作系统剪贴板的方式。

方法

{{domxref("Clipboard.read()","read()")}}
从剪贴板读取数据（比如图片）。
{{domxref("Clipboard.readText()","readText()")}}
从操作系统读取文本。
{{domxref("Clipboard.write()","write()")}}
写入数据（比如图片）至操作系统剪贴板。
{{domxref("Clipboard.writeText()","writeText()")}}
写入文本至操作系统剪贴板。
