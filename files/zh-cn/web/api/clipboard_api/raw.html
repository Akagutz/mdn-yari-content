剪贴板 API 是一个提供异步功能和安全上下文支持的现代API。
该 API 被设计用来取代{{domxref("document.execCommand()")}}的剪贴板访问方式。

剪贴板接口

{{domxref("Clipboard")}} {{securecontext_inline}}
提供了一个用于读写系统剪贴板上文本和数据的接口. 规范上称为'异步 剪贴板 API.'
{{domxref("ClipboardEvent")}} {{securecontext_inline}}
表示提供的修改剪贴板有关信息的事件, {{event("剪切")}}、{{event("复制")}}、 {{event("粘贴")}}.规范上称为 '剪贴板 事件 API'.
