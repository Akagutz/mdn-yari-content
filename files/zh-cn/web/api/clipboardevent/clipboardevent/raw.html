ClipboardEvent() 构造器返回一个新建的 {{domxref("ClipboardEvent")}} 对象， 这个对象描述了与修改剪切板相关的事件 ，这些事件包括 {{event("cut")}} 、 {{event("copy")}}  和 {{event("paste")}} 事件。

语法

clipboardEvent = new ClipboardEvent(type, {dataType: aMimeType, 
                                           data    : someData});

参数

ClipboardEvent() 构造器同样从 {{domxref("Event.Event", "Event()")}} 继承参数。

type
一个 {{domxref("DOMString")}} 字符串，描述了 ClipboardEvent  事件的类型名。它是大小写敏感，且只能是以下内容之一：'copy'、 'cut'或者 'paste'。
dataType
一个 {{domxref("DOMString")}} 字符串，描述了 data 参数携带的数据的 MIME 类型。
data
一个 {{domxref("DOMString")}} 字符串，包含了于剪切板事件相关的数据。
