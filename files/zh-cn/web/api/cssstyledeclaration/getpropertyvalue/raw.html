此 CSSStyleDeclaration.getPropertyValue() 接口会返回一个 {{domxref('DOMString')}} ，这个返回值将会包含预请求的CSS属性信息。

语法

var value = style.getPropertyValue(property);

参数

  • property 应是一个 {{domxref('DOMString')}}，为所需要查询的CSS 属性名称

返回值

  • value 是 {{domxref('DOMString')}} ，包含有欲查找的CSS属性的值.  若原CSSStyleDeclaration没有设置对应属性，则返回空字符串

例子

下面的例子试图在一个特定的CSS 规则中查询对应的margin 属性:

var declaration = document.styleSheets[0].cssRules[0].style;
var value = declaration.getPropertyValue('margin'); // "1px 2px"
