CSSStyleSheet 接口代表一个单一的  CSS样式表。它从其父代{{domxref("StyleSheet")}}继承属性和方法。

 

一个CSS样式表包含了一组CSS规则(CSS rules)。每条CSS规则可以通过与之相关联的的对象进行操作，这个关联对象实现了{{ domxref("CSSStyleRule") }}接口，而CSSStyleRule反过来实现了{{ domxref("CSSRule") }}。CSSStyleSheet允许你检测与修改和它相关联的的样式表(style sheet)，包括样式表的规则列表。

实际上，CSSStyleSheet也实现了更为通用的{{ domxref("StyleSheet") }} 接口。实现一个document的样式表的CSSStyleSheet列表可以过document.styleSheet属性获取(这个document通过外联样式表或内嵌的style元素定义样式)。

属性

cssRules
返回样式表中CSS规则的{{ domxref("CSSRuleList") }}对象
ownerRule
如果一个样式表示通过{{ cssxref("@import") }} 规则引入document的，则ownerRule将返回那个{{ domxref("CSSImportRule") }}对象，否则返回null

方法

deleteRule
从样式表中删除一条规则
insertRule
向样式表中插入一条新规则

注意

     在一些浏览器中，如果stylesheet是从不同域加载进来的，调用cssRules将导致SecurityError
