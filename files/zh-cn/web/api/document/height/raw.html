从Gecko 6.0开始,不再支持document.height属性. 请使用document.body.clientHeight来替代. 查看 {{ domxref("element.clientHeight") }}.

概述

返回当前文档中的{{ HTMLElement("body") }}元素的高度.

语法

height_value = document.height

例子

// 弹出文档高度
alert(document.height);

用什么替代该属性

document.body.clientHeight
document.documentElement.clientHeight
