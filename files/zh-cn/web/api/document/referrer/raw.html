Document.referrer 返回 跳转或打开到当前页面 的页面的 URI。

语法

var referrer = document.referrer;

属性值

如果用户直接打开了这个页面（不是通过页面跳转，而是通过地址栏或者书签等打开的），则该属性为空字符串。由于该属性只是返回一个字符串，所以不能够通过该属性引用页面的 DOM。

Inside an {{HTMLElement("iframe")}}, the Document.referrer will initially be set to the same value as the {{domxref("HTMLHyperlinkElementUtils/href", "href")}} of the parent window's {{domxref("Window/location", "Window.location")}}.

规范

  • DOM Level 2: referrer
