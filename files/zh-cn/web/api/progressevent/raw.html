{{APIRef("DOM Events")}}

 ProgressEvent 是一个用来测量底层操作进度的接口，可以测量HTTP请求（例如：一个 XMLHttpRequest请求、或者一个底层资源如 {{HTMLElement("img")}}, {{HTMLElement("audio")}}, {{HTMLElement("video")}}, {{HTMLElement("style")}} or {{HTMLElement("link")}}).

Constructor

{{domxref("ProgressEvent.ProgressEvent", "ProgressEvent()")}}
用给定的参数创造一个ProgressEvent

Properties

同时继承它的父元素 {{domxref("Event")}} 的属性。

{{domxref("ProgressEvent.lengthComputable")}} {{readonlyInline}}
它是一个布尔值标识，表明总共需要完成的工作量和已经完成的工作是否可以被底层所计算到。换而言之，它表示的就是过程是否是可以测量的。
{{domxref("ProgressEvent.loaded")}} {{readonlyInline}}
是一个unsigned long long类型，表示底层进程已经执行的工作量。所做的工作比率可以用属性和ProgressEvent.total计算。当使用HTTP下载资源时，这只表示内容本身的一部分，而不是头和其他开销。
{{domxref("ProgressEvent.total")}} {{readonlyInline}}
是unsigned long long类型，表示底层进程正在执行的工作总量。当使用HTTP下载资源时，这只表示内容本身，而不是头和其他开销。

Methods

同时继承它的父元素 {{domxref("Event")}} 的方法。

{{domxref("ProgressEvent.initProgressEvent()")}} {{deprecated_inline}}{{non-Standard_inline}}
使用已经被弃用的 {{domxref("Document.createEvent()", "Document.createEvent(\"ProgressEvent\")")}} 方法，初始化一个已经创建好的ProgressEvent。

Examples

下面展示一个为 {{domxref("XMLHTTPRequest")}} 请求添加 ProgressEvent接口，并且用它来显示这个请求的状态的例子。

  var progressBar = document.getElementById("p"),
      client = new XMLHttpRequest()
  client.open("GET", "magical-unicorns")
  client.onprogress = function(pe) {
    if(pe.lengthComputable) {
      progressBar.max = pe.total
      progressBar.value = pe.loaded
    }
  }
  client.onloadend = function(pe) {
    progressBar.value = pe.loaded
  }
  client.send()
