{{ApiRef("DOM")}}

调用 Range.selectNode() 方法后， {{domxref("Range")}} 设置为包含整个 {{domxref("Node")}} 。此 {{domxref("Range")}} 的 startContainer 属性和 endContainer 属性保存的 {{domxref("Node")}} 将设置为 referenceNode 的父节点。

语法

range.selectNode(referenceNode);

参数

referenceNode
{{domxref("Range")}} 要包含的 {{domxref("Node")}}。

举例

var range = document.createRange();
var referenceNode = document.getElementsByTagName("div").item(0);

range.selectNode(referenceNode);
