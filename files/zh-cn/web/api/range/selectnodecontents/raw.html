Range.selectNodeContents() 方法设置 {{ domxref("Range") }} 使其包含一个 {{ domxref("Node") }} 的内容。

{{ domxref("Range") }} 的 startContainer 属性和 endContainer 属性保存的 {{ domxref("Node") }} 将被设置为 referenceNode。 startOffset 为 0,  endOffset 将被设置为 referenceNode 的字符数或子节点个数，具体参见Range.endOffset。

Syntax

range.selectNodeContents(referenceNode);

Parameters

referenceNode
Range 要包含的 Node。

Example

range = document.createRange();
referenceNode = document.getElementsByTagName("div")[0];
range.selectNodeContents(referenceNode);
