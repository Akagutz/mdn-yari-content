Range.startOffset 是一个用于返回代表 Range 起始位置的 startContainer 数字的只读属性。

如果 startContainer 是一个文本（{{domxref("Text")}}）、注释（{{domxref("Comment")}}）或者CDATA区块（{{domxref("CDATASection")}}）节点那么返回的偏移量是从 startContainer 开始到{{domxref("Range")}}的边界点的距离。对于其他的节点类型， startOffset 返回 startContainer 到边界点的子元素数量。

如果您想改变一个 Range 的 startOffset 位置，可以使用{{domxref("Range.setStart")}} 方法。

用法

startRangeOffset = range.startOffset;

范例

range = document.createRange();
range.setStart(startNode,startOffset);
range.setEnd(endNode,endOffset);
startRangeOffset = range.startOffset;
