{{ ApiRef("Range") }}

Range.surroundContents()方法将Range对象的内容移动到一个新的节点, 并将这个新节点放到指定区域.

这个方法与newNode.appendChild(range.extractContents()); range.insertNode(newNode)很相近. 选定后, range拥有了节点标签.

也有例外情况, 如果选定的{{ domxref("Range") }}区域包含仅有一个节点标签的{{ domxref("Text") }}. 那标签将不会自动成对生成，操作将失败.

语法

range.surroundContents(newNode);

参数

newNode
一个放置区域内容的目标{{ domxref("Node") }} .

例子

var range = document.createRange();
var newNode = document.createElement("p");

range.selectNode(document.getElementsByTagName("div").item(0));
range.surroundContents(newNode);
