{{APIRef("WebGL")}}
WebGL API 的 WebGLRenderingContext.getUniform() 方法返回指定位置的统一变量的值。

句法

any gl.getUniform(program, location);

参数

代码
{{domxref("WebGLProgram")}} 包含统一的属性
本地的
包含要获取的统一属性位置的 {{domxref("WebGLUniformLocation")}} 对象。

返回值

返回的类型取决于统一类型：

示例

var loc = gl.getUniformLocation(program, 'u_foobar');
gl.getUniform(program, loc);

//code in vertex-shader
//...
