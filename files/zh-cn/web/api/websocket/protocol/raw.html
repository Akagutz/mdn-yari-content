WebSocket.protocol 是个只读属性，用于返回服务器端选中的子协议的名字；这是一个在创建{{domxref("WebSocket")}} 对象时，在参数{{domxref("protocols")}}中指定的字符串。

语法

var protocol = aWebSocket.protocol;

返回值

DOMString.
