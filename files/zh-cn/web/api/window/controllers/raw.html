概要

返回chorme浏览器窗口下的XUL控制器。

语法

controllers = window.controllers
  • controllers 是 XULControllers (nsIControllers)类型的一个对象.

规范

XUL-专用， 不属于任何规范。

在默认情况下，窗口的控制器包含支持全局窗口命令的代码。

Chrome代码可以添加控制器（与globalOverlay.js中的goDoCommand和goUpdateCommand函数配合使用）。

然而，当浏览器窗口关闭时候，这些添加的控制器我们必须手动删除，因为浏览器并不会自动的完成这些操作。

每当有忘记删除的控制器都会引起错误：

bug 415775: 
ASSERTION: XPConnect is being called on a scope without a 'Components' property!
