W3C并没有定义XMLDocument接口。原因是document接口本来就是为xml定义的。而HTML只是xml的一种实现。因此需要定义HTMLDocument。但是在FireFox有一个专门的XMLDocument构造器。应该是对document对象的一些扩展。通常情况下，就当作document来使用。

此接口没有特定的属性，也不继承任何属性

此接口没有添加任何新的方法
