概述

::-webkit-progress-bar CSS 伪元素 表示 {{HTMLElement("progress")}} 元素的整个条形. 通常在条形值没有达到满值时可以见到，{{ cssxref("::-webkit-progress-value") }} 伪元素代表值的条形.::-webkit-progress-bar 是{{cssxref("::-webkit-progress-inner-element")}}伪元素的子元素，同时是{{cssxref("::-webkit-progress-value")}} 伪元素的父级元素。

Note: 为了能让::-webkit-progress-value起作用, {{cssxref("-webkit-appearance")}} 需要添加至 <progress> 元素.

Specifications

Not part of any specification. This is a proprietary pseudo-element specific to WebKit/Blink.

Example

CSS content

progress {
  -webkit-appearance: none;
}

::-webkit-progress-bar {
   background-color: orange;
}

HTML content

<progress value="10" max="50">
