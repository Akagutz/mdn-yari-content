CSS 并集选择器, 逗号隔开

/* 选择所有匹配的元素 */
span, div {
  border: red 2px solid;
}

为了使样式表更简洁, 可以在逗号分隔的列表中对选择器进行分组

语法

element, element, element { style properties }

例子

单行分组

在一行中使用逗号为选择器分组

h1, h2, h3, h4, h5, h6 { font-family: helvetica; }

多行分组

使用逗号对多行的选择器进行分组

#main,
.content,
article {
  font-size: 1.1em;
}

选择器列表无效

使用选择器列表的一个缺点是，以下内容不等价

h1 { font-family: sans-serif }
h2:maybe-unsupported { font-family: sans-serif }
h3 { font-family: sans-serif }
h1, h2:maybe-unsupported, h3 { font-family: sans-serif }

这是因为如何选择器列表(并集选择器)中如果有一个不被支持那么整个就会失效.

解决这个问题的一个方法是使用 {{CSSxRef(":is", ":is()")}} 选择器, 它会忽视它的参数列表中失效的选择器, 但是由于 {{CSSxRef(":is", ":is()")}}  计算特异性的方式, 这样做的代价是所有的选择器要有相同的特异性.

h1 { font-family: sans-serif }
h2:maybe-unsupported { font-family: sans-serif }
h3 { font-family: sans-serif }
:is(h1, h2:maybe-unsupported, h3) { font-family: sans-serif }
