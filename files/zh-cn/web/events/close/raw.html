close事件由几个API生成：

HTML DOM
发送到目标{{HTMLElement("dialog")}}，让它知道它已经被关闭。
IndexedDB API
当数据库意外关闭时，发送到表示数据库连接的 {{domxref("IDBDatabase")}} 对象。这发生在所有交易都被中止之后。你可以使用 {{domxref("IDBDatabase.onclose")}} 来建立事件处理程序。
WebRTC API
发送到{{domxref("RTCDataChannel")}}，让它知道数据连接已经终止。除使用{{domxref("EventTarget.addEventListener", "addEventListener()")}}外，你可以用{{domxref("RTCDataConnection.onclose")}} 属性创建事件处理程序。
WebSocket API
WebSocket连接关闭时发送。这里需要更多的信息。
Web Workers API {{Non-standard_inline}} {{obsolete_inline}}
一些浏览器使用关闭事件来指示给定的 {{domxref("Worker")}}已经关闭；这只是规范的一部分，但已从规范中删除，浏览器正在取消对Web Workers API中此事件的支持。
