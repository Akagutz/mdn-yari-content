fieldset 元素通常用来对表单中的控制元素进行分组(也包括 label 元素)

 

Content categories Flow content, sectioning root, listed, form-associated element, palpable content.
Permitted content An optional {{HTMLElement("legend")}} element, followed by flow content.
Tag omission {{no_tag_omission}}
Permitted parent elements Any element that accepts flow content.
DOM interface {{domxref("HTMLFieldSetElement")}}

注意: 与几乎任何其他元素不同，WHATWG HTML Rendering规范建议 {{cssxref("min-width")}}: min-content 作为{{HTMLElement("fieldset")}}默认样式的一部分, 并且许多浏览器都实现了这种样式 (或者与其类似的东西).

属性

 fieldset  中包含所有全局属性

{{htmlattrdef("disabled")}} {{HTMLVersionInline(5)}}
如果设置了这个 bool 值属性, 它的后代表单控制元素也会继承这个属性, 它的首个可选的 legend 元素除外, 例如, 禁止编辑. 该元素和它的子元素不会接受任何浏览器事件, 比如点击或者 focus 事件, 一般来说浏览器会将这样的元素展示位灰色.
{{htmlattrdef("form")}} {{HTMLVersionInline(5)}}
规定{{htmlattrdef("fieldset")}}所属的{{htmlattrdef("form")}}表单. 这个属性的值与其所属的{{htmlattrdef("form")}}的{{htmlattrdef("ID")}}相同. 默认值是最近的那个{{htmlattrdef("form")}}. 
      
{{htmlattrdef("name")}} {{HTMLVersionInline(5)}}
控制元素分组的名称
The label for the field set is given by the first {{HTMLElement("legend")}} element that is a child of this field set.

示例

Example #1: 一个拥有 fieldset, legend 和 label 的表单

<form action="test.php" method="post">
  <fieldset>
    <legend>Title</legend>
    <input type="radio" id="radio"> <label for="radio">Click me</label>
  </fieldset>
</form>
