状态码 405 Method Not Allowed 表明服务器禁止了使用当前 HTTP 方法的请求。需要注意的是，{{HTTPMethod("GET")}} 与 {{HTTPMethod("HEAD")}} 两个方法不得被禁止，当然也不得返回状态码 405。

状态码

405 Method Not Allowed
