{{JSRef}}

静态方法 Reflect.getPrototypeOf() 与 {{jsxref("Object.getPrototypeOf()")}} 方法是一样的。都是返回指定对象的原型（即，内部的 [[Prototype]] 属性的值）。

语法

Reflect.getPrototypeOf(target)

参数

target
获取原型的目标对象。

返回值

给定对象的原型。如果没有继承的属性，则返回 {{jsxref("null")}}。

异常

抛出一个 {{jsxref("TypeError")}} 异常，如果目标不是 {{jsxref("Object")}}。

描述

Reflect.getPrototypeOf 返回指定对象的原型 (例如：内部的 [[Prototype]] 属性的值) 。

示例

使用 Reflect.getPrototypeOf()

Reflect.getPrototypeOf({}); // Object.prototype
Reflect.getPrototypeOf(Object.prototype); // null
Reflect.getPrototypeOf(Object.create(null)); // null
