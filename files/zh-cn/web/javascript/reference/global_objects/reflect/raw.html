{{JSRef}}

Reflect 是一个内置的对象，它提供拦截 JavaScript 操作的方法。这些方法与proxy handlers的方法相同。Reflect不是一个函数对象，因此它是不可构造的。

描述

与大多数全局对象不同，Reflect不是一个构造函数。你不能将其与一个new运算符一起使用，或者将Reflect对象作为一个函数来调用。Reflect的所有属性和方法都是静态的（就像{{jsxref("Math")}}对象）。

方法

Reflect对象提供以下静态函数，它们具有与处理器对象方法相同的名称。这些方法中的一些与 {{jsxref("Object")}} 上的对应方法相同。

{{jsxref("Reflect.apply()")}}
对一个函数进行调用操作，同时可以传入一个数组作为调用参数。和 {{jsxref("Function.prototype.apply()")}} 功能类似。
{{jsxref("Reflect.construct()")}}
对构造函数进行 new 操作，相当于执行 new target(...args)。
{{jsxref("Reflect.defineProperty()")}}
和 {{jsxref("Object.defineProperty()")}} 类似。
{{jsxref("Reflect.deleteProperty()")}}
作为函数的delete操作符，相当于执行 delete target[name]。
{{jsxref("Reflect.enumerate()")}}
该方法会返回一个包含有目标对象身上所有可枚举的自身字符串属性以及继承字符串属性的迭代器，for...in 操作遍历到的正是这些属性。
{{jsxref("Reflect.get()")}}
获取对象身上某个属性的值，类似于 target[name]。
{{jsxref("Reflect.getOwnPropertyDescriptor()")}}
类似于 {{jsxref("Object.getOwnPropertyDescriptor()")}}。
{{jsxref("Reflect.getPrototypeOf()")}}
类似于 {{jsxref("Object.getPrototypeOf()")}}。
{{jsxref("Reflect.has()")}}
判断一个对象是否存在某个属性，和 in 运算符 的功能完全相同。
{{jsxref("Reflect.isExtensible()")}}
类似于 {{jsxref("Object.isExtensible()")}}.
{{jsxref("Reflect.ownKeys()")}}
返回一个包含所有自身属性（不包含继承属性）的数组。(类似于 {{jsxref("Object.keys()")}}, 但不会受enumerable影响).
{{jsxref("Reflect.preventExtensions()")}}
类似于 {{jsxref("Object.preventExtensions()")}}。返回一个{{jsxref("Boolean")}}。
{{jsxref("Reflect.set()")}}
将值分配给属性的函数。返回一个{{jsxref("Boolean")}}，如果更新成功，则返回true。
{{jsxref("Reflect.setPrototypeOf()")}}
类似于 {{jsxref("Object.setPrototypeOf()")}}。
