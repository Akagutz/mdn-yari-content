valueOf() 方法返回一个{{jsxref("String")}}对象的原始值（primitive value）。

语法

str.valueOf()

返回值

一个字符串代表着给定{{jsxref("String")}} 对象的初始值。

描述

{{jsxref("String")}} 对象的 valueOf 方法返回一个{{jsxref("String")}}对象的原始值。该值等同于{{jsxref("String.prototype.toString()")}}。

该方法通常在 JavaScript 内部被调用，而不是在代码里显式调用。

示例

例子：使用 valueOf

x = new String("Hello world");
console.log(x.valueOf())          // Displays "Hello world"
