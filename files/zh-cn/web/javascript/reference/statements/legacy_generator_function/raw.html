遗留的生成器函数是 SpiderMonkey 的特殊特性，会在未来的某个时刻被修改。为了以后的使用，考虑使用 {{jsxref("Statements/function*", "function*")}}。
{{jsSidebar("Statements")}}

遗留的生成器函数语句 使用特殊参数声明遗留的生成器函数。

你也可以使用带有functionBody的构造{{jsxref("函数")}} ，至少一个 {{jsxref("运算符/yield", "yield")}} 表达式，以及{{jsxref("运算符/Legacy_generator_function", "遗留的生成器函数表达式", "", 1)}}来定义函数。

语法

function name([param,[, param,[..., param]]]) {
   [statements]
}
name
函数名。
param
传入函数的参数名，一个函数最多有255个参数。
statements
构成函数体的语句。应该至少含有一个 {{jsxref("Operators/yield", "yield")}} 表达式。

描述

用法概述可在 迭代器和生成器页面上查看。
