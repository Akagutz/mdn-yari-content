網站、網際應用程式相關變更

已經支援 ICC 色彩調校。目前僅有標記圖片會採用這種調整方式，但是否應用到所有圖片上則還在討論。詳情請見這篇文章 。

CSS 相關變動

錯誤修補

  • 「ch」單位值現在已可像其他單位值般在任何地方正常使用，並依據標準規範、以字元「0」的寬度為基準。({{ Bug(363706) }})

CSS 2.1 支援

  • 依據 CSS 2.1 調整 ::before 及 ::after，詳情請見這篇文章。

新功能

  • Media queries

新特性

  • {{ Cssxref("text-shadow") }}
  • 以 {{ Cssxref("-moz-box-shadow") }} 的方式支援 box-shadow
  • 以 {{ Cssxref("-moz-border-image") }} 的方式支援 border-image
  • 以 {{ Cssxref("-moz-column-rule") }} 的方式支援 column-rule。詳情請見這篇文章。
  • {{ Cssxref("word-wrap") }}。詳情請見這篇文章 。
  • 將 {{ Cssxref("-moz-nativehyperlinktext") }} 視為色彩值常數
  • 將 {{ Cssxref("-moz-mac-unified-toolbar") }} 視為 -moz-appearance 的合法設定值。({{ Bug(439354) }})
  • 將 pre-line 視為 {{ Cssxref("white-space") }} 的合法設定值。({{ Bug(230555) }})。詳情請見這篇文章。
  • 將 {{ Cssxref("-moz-win-glass") }} 視為 -moz-appearance 的合法設定值。({{ Bug(418454) }}).
  • 以 {{ Cssxref("-moz-transform") }} 的方式支援 CSS transforms。詳情請見這篇文章。

HTML 相關變動

  • 支援 HTML 5 的拖放 (drag & drop) 事件。詳情請見這篇文章。

DOM 相關變動

新功能

  • 選取符 API (querySelector 及 querySelectorAll)
  • NodeIterator 物件 API
  • DOM worker threads (see this article for initial information, and the specification).
  • replaceWholeText method and wholeText property.  (DOM) (See this blog post for information).
  • Geolocation API ({{ Bug(437948) }})
  • Element Traversal API ({{ Bug(447917) }})
  • HTML 5 拖放 API ({{ Bug(356295) }})

Bug fixes

  • Dispatched events can be redispatched ({{ Bug(412567) }}).
  • DOM 2 Range improvements ({{ Bug(332148) }}).

新功能

  • <script defer> ({{ Bug(28293) }})

新功能

SVG

新功能

  • GetSVGDocument is added for <object> and <iframe> for compatibility.
  • SVG effects can now be applied to HTML content.  See this article for more information.

錯誤修補

  • id references are now live ({{ Bug(344258) }}).Expand Me
  • filters now works for foreignObject ({{ Bug(445079) }}).

For XUL and application developers

New features

  • Search textbox widget (<textbox type="search">) ({{ Bug(388811) }}).
  • Drag and Drop between windows ({{ Bug(113934) }}).
  • The {{ XULAttr("panel.level") }} attribute of {{ XULElem("panel") }} element. And auto-hide panel element is not topmost window on non-Linux platforms without level attribute ({{ Bug(451015) }}).
  • Progress events for XMLHttpRequest ({{ Bug(435425) }}).

Changes to interfaces

  • {{ Interface("mozIStorageStatement") }} had the initialize() method removed as per {{ Bug(434796) }}. Consumers should use the {{ Ifmethod("mozIStorageConnection", "createStatement") }} method instead to get a new statement object.
