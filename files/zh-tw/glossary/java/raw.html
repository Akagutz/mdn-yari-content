Java 是一個 {{glossary("Compile", "編譯")}}, {{glossary("OOP", "面向對象")}}, 高度便攜 {{Glossary("computer programming", "程式設計")}} 語言。

Java 是靜態類型化並且具有類 C 的語法。它帶有一個大型而易於使用的功能包，Java 軟體開發套件(SDK)。

程式只提前 {{glossary("Compile", "編譯")}}一次，編譯成專有的字節碼和包格式運行在 Java 虛擬機(JVM) 內部。  JVM 支持跨平台，這使 Java 程式幾乎可以在任何地方運行，不需要再次編譯或打包。這個特性使它成為很多大型企業的首選，但可能認為它過於“沉重”。
