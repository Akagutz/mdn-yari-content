routers 是指路由器。在 Web 有兩種意思：

  1. 針對網路層，路由器是一種決定{{Glossary('封包')}}去向的網路設備。路由器是由 retailer 分發，允許用戶與網際網路互動。
  2. 針對應用層的 {{Glossary('SPA')}}，路由器是一種透過給定的 {{Glossary('URL')}} 來決定頁面如何呈現的{{Glossary('函式庫')}}。 This middleware module is used for all URL functions, they are given a path to a file which is rendered to open the next page.
