在我們第一個 JavaScript 教學，在帶領各位開始體驗撰寫 JavaScript 前，我們首先回答一些基本的問題，「什麼是 JavaScript ？」、「它看起來像什麼？」，以及「它可以做什麼？」。之後我們會詳細討論一些建立這些區塊的重要關鍵，諸如變數、字串、數字及陣列。

事前準備

在開始這個教學之前，你不需要具備任何的 JavaScript 相關知識，但是你應該要對 HTML 及CSS 有所了解。我們建議您在開始學習 JavaScript 前，先下看過列內容 :

  • 開始使用 Web  (包括 JavaScript 基礎簡介).
  • HTML 簡介.
  • CSS 簡介.

注意: 假如你正在使用 電腦/平板/其他裝置，你不需要建立自己的檔案，你可以嘗試線上程式撰寫系統來撰寫範例程式，像是 JSBin 或 Thimble.

概觀

什麼是 JavaScript?
歡迎來到 MDN JavaScript 新手村教學！在第一篇文章中我們將從某個高度上來看 JavaScript，解答一些「它是什麼？」、「它到底有什麼卵用？」，並且確保你能明白 JavaScript 的中心思想。
和 JavaScript 的第一次接觸體驗
現在你已經學習了一些關於 JavaScript 的理論，以及你可以用它來幹些什麼，我們將會透過一個非常實用的教學，給你一個有關 JavaScript 基本特性的速成課程。這裡你將會一步一步建立一個簡單的「估數字」遊戲。
發生什麼事了？ JavaScript 的疑難排解
在前一篇建立「估數字」遊戲的過程之中，你可能會發現它有些問題，不要慌。這篇文章的目標是透過提供一些如何在 JavaScript 程式中找出和補救錯誤的方法，將你從使你抓破頭皮的這些問題中拯救出來。
存一些你想要的資訊 — 變數
在閱讀過最近的幾篇文章之後，你現在應該能理解 JavaScript 是什麼，它可以為你做些什麼，你如何用其他的 Web 技術來配合它，以及它的主要特性。 在本篇文章中，我們將探討真實的底層知識，看看如何使用這些 JavaScript 的基本模組 — 變數 (Variables)。
JavaScript 的基本計算— 數字 與 運算子
本章中我們會討論 JavaScript 的計算 — 我們如何成功地按照我們的要求將運算子與其它內容物放在一起。
處理文字 — 字串 in JavaScript
接下來我們將目光轉向字串 (strings) — 寫程式時片段的文字的稱呼。在這篇文章中我們會檢視你在學習 JavaScript 中的字串，應該要知道的各種情況，例如創建一個字串、字串中的逃脫引用 (escaping quotes)、以及把它們放在一塊兒。
好用的字串方法
我們已經看過字串 (strings) 的基本用法，現在讓我們拿起工具然後想想看 JavaScript 裡面有哪些好用的內建方法 (method) ，例如找字串的長度、字串的串接與切開、替換字串中的某個字元，等等。
陣列
本次教學的最後一個部分，我們來看看陣列 (Array) — 乾淨整潔地把一連串資料儲存在一個變數內。這裡我們會看到為什麼它這麼的好用，然後我們探索如何創建一個陣列、加入項目、刪除項目、獲得陣列項目資訊，等等。
