喜歡 MDN 嗎？那就透過推廣 MDN 向其他開發者分享 open-Web 多棒吧！我們這裡有一些你能辦到的方法，也提供了桌面、貼紙、海報、動畫、網頁徽章、以及其他你想得到的方法，從這裡連結出去。

想知道什麼是 MDN 或如何幫忙在 Wiki 上寫網路技術嗎？

在社群網路分享連結

覺得 MDN 很有用嗎？在 Twitter、Facebook、Hacker News、reddit、或其他你覺得很酷、很喜歡的地方分享吧。讓我們 @MozDevNet 被 cue 到當然是最好，但也不是必要的。

回答問題時給個 MDN 連結

不管你在 Stackoverflow 回答問題、或有人在 Twitterverse 喧嘩、抑或同事在 Slack 之類的聊天室發問，記得回答那些能在 MDN 找到答案的網路開發問題。透過分享對發問者的問題有所助益的 MDN 連結，以幫他們獲取最充足的網路開發資訊。

談到技術時引用 MDN

不管你在本地的 JavaScript 社團聚會聊天、還是在主要的技術會議演講，MDN 通常都會有夠你加強論點、或提供聽眾基本資訊背景的文章。如果你在線上演講，可以的話 cue 我們 @MozDevNet 然後給我們一個連結。但最重要的是：你給需要的人分享需要的信息。

WordPress 套件

Promote MDN plugin for WordPress 部落格推廣的 WordPress MDN 套件，能在 MDN 得到幫助的同時，也能幫助那些在自己的部落格，發表網路技術或與 Mozilla 產品相關的部落客們。以下為兩種運作方式：

  1. 當你在部落格發布文章的時候，它會自動鏈接一些如「HTML」、「JavaScript」、「CSS」或其他的相關術語到 MDN 的頁面。這可以讓你在寫技術博文的時候，不用做任何事就能自動添加 MDN 相關文件的鏈接。它非常的有用！ MDN 的相關術語及其描述，都透過 Github repo 的 def-list 分支（https://github.com/mdn/wp-promote-mdn/tree/def-list）設定好了。

安裝這個套件對你和 MDN 都有很大的幫助，所以若還沒安裝的話，快裝吧！若想改進這個套件，請前往 wp-promote-mdn 的 GitHub repo。

JavaScript 程式

上面提到的 WordPress 套件，是自動把關鍵詞連到 MDN 相關文件的其中一種方法，但如果你的部落格並不是在 WordPress 上運行，你可以嵌入這段 JavaScript 的程式碼以實現相同的功能。這個腳本做的事情與 WordPress 套件一樣：它會自動把文章的已知術語，連到 MDN 的文件。

你可以透過更改 o 物件裡面的 maxLinks 屬性值，來設定關鍵詞的最大數量、也可以指定要搜索的元素、建立連結的 class、以及額外要搜索的關鍵詞，和其對應的 URL。

要使用的話，只把程式碼嵌到 </body> 標籤關閉前的地方。

<script type="text/javascript" >
    (function (d,f,a) {
        // Your settings here
        var o = {maxLinks: 4, searchElements: ['div', 'h'], linkClass: 'link-to-mdn', extraLinks: {'keywordx': 'http://example.com'}};

        var s=d.createElement("script");s.type="text/javascript";if(s.f)
        {s.a=function(){if(s.f=="loaded"||s.f=="complete")
        {s.a=null;PromoteMDN(o)}}}else{s.onload=function()
        {PromoteMDN(o)}}s.src="https://raw.githubusercontent.com/riverspirit/promote-mdn-script/master/promote-mdn.js";
        d.getElementsByTagName("head")[0].appendChild(s)
    })(document,'readyState','onreadystatechange');
</script>

這個程式優先載入 GitHub 的程式碼。如果出於安全或性能考量，想把原始碼放在自己的伺服器，只要從 GitHub 下載原始碼、然後把它加到你自己的網站中、並修改第九行的 URL 即可。
