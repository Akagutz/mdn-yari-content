Server 標頭包含了來源伺服器裡面，處理網路請求的軟體相關資訊。

請避免 Server 值的資訊過度冗長與詳盡，因為它們可能會洩漏實做細節、讓攻擊者容易找到已知安全漏洞並利用之。

標頭類型 {{Glossary("Response header")}}
{{Glossary("Forbidden header name")}} 否

語法

Server: <product>

指令

<product>
處理請求的軟體（或組件）名。

示例

Server: Apache/2.4.1 (Unix)
